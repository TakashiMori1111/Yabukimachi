# C-level向け企画提案整理

## 矢吹町が生き残るための「持続可能な自治体経営」構想

## 0. 結論

矢吹町が今後も生き残るために必要なのは、単なる歳出削減ではなく、**「自主財源を増やす」「行政コストを再設計する」「若者・現役世代に選ばれる生活基盤をつくる」**の3つを同時に進める自治体経営への転換。

特に推すべき方向性は、次の統合案。

```markdown
推奨案：
「選ばれる生活基盤 × 自主財源創出 × 行政コスト再編」を一体化した
“矢吹町サステナブル経営モデル”
```

理由は、人口減少下では、**住民数だけを追う政策は限界**があり、今いる住民・職員・事業者の生産性と定着率を高めながら、町外から人・企業・資金を呼び込む必要があるため。

添付メモでも、現役世代の生活基盤は、医療・社会保障・自治体インフラ・若者定着を分けずに、1つの構造問題として扱う必要があると整理されている。これは、矢吹町の経営課題にもそのまま接続可能。

---

# 1. 背景

## 1-1. 矢吹町を取り巻く構造変化

矢吹町の論点は、短期的な単年度収支ではなく、**人口・財政・公共施設・人材・生活基盤が同時に細ること**。

令和6年度の矢吹町財政状況資料集では、令和2年国勢調査人口17,287人に対し、令和7年1月1日の住民基本台帳人口は16,807人。短期で急崩壊しているわけではないが、人口は微減傾向。財政力指数は0.60、地方交付税種地は2-2であり、完全な自主財源型の自治体ではない整理。([やぶき町公式サイト][1])

歳入面では、令和6年度の歳入総額は約89.5億円、地方交付税は約21.5億円で歳入の24.0%、国庫支出金は約13.5億円で15.1%。つまり、一定程度は国・県財源に支えられている構造。([やぶき町公式サイト][1])

さらに、公共施設面では、矢吹町は265施設、総延床面積93,751㎡を保有し、これらをそのまま維持した場合、40年間で1,068億円、年平均約26億円が必要との試算あり。これは、今後の町政において最大級の固定費リスク。([やぶき町公式サイト][2])

---

## 1-2. 全国構造としての「支え手不足」

日本全体では、2020年の総人口1億2,615万人から、2070年には8,700万人へ減少し、高齢化率は28.6%から38.7%へ上昇する推計。これは、税・社会保険料を負担する現役世代が相対的に細り、医療・介護・年金・インフラ維持費が重くなる構造を意味する。([厚生労働省][3])

添付メモでも、人口減少は「税収・担い手不足」につながり、その結果として「自治体サービスの維持困難化」「インフラ更新負担の上昇」が起こると整理されている。

---

## 1-3. 「選ばれる自治体」の条件変化

内閣府の2025年調査では、「若者や女性にも選ばれる地方」を実現する文脈で、生活満足度、住宅・住環境、子育て環境、生活の楽しさ、雇用環境、賃金などが重視されている。選ばれる自治体では、女性や現役世代を中心に生活全般の満足度が高い傾向。([内閣府ホームページ][4])

また、厚労省の令和7年版白書では、高校生3,000人を対象とした調査で、医療への関心63.6%、年金への関心58.3%、公衆衛生への関心47.5%とされている。若者は医療・社会保障・公衆衛生に無関心ではない。([厚生労働省][5])

つまり、矢吹町に必要なのは、単に「安い町」「住める町」ではなく、**将来も安心して働き、暮らし、子育てし、健康でいられる町**として見られること。

# 3. 戦略案

## 案A：財政防衛型

### 「歳出抑制・公共施設再編」を最優先する案

## 概要

公共施設、補助金、事務事業、受益者負担を見直し、将来負担を抑える案。

矢吹町の第7次行財政改革大綱でも、令和7年度以降は大規模事業を見込むことで歳出超過が見込まれ、事業の選択と集中、自主財源の確保、受益者負担の見直しが必要と整理されている。([やぶき町公式サイト][2])

## 主な施策

```markdown
- 公共施設の統廃合・複合化・長寿命化
- 低利用施設の用途転換・民間活用
- 受益者負担の見直し
- 補助金・委託費の棚卸し
- 行政事務のデジタル化・共同化
- 近隣自治体との広域連携
```

## メリット

```markdown
- 即効性がある
- 財政規律を示しやすい
- 住民説明で「無駄の削減」として理解されやすい
- 公共施設更新リスクに対応できる
```

## デメリット

```markdown
- 削減中心だと町の魅力が高まらない
- 若者・企業に選ばれる理由にはなりにくい
- 住民には「サービス低下」と受け止められる可能性
- 職員負荷が増えやすい
```

## 評価

```markdown
必要だが、これ単独では生き残れない。
守りの戦略。
```

# 案B：自主財源創出型

## 「稼ぐ町」をつくる案

## 概要

町外から資金を呼び込み、町内で所得と税源を生む案。
自主財源の柱を、住民税・固定資産税・法人関連税・ふるさと納税・使用料・財産活用・企業連携に広げる。

## 主な施策

```markdown
- 企業誘致・工業系土地活用
- 遊休町有地・未利用施設の民間活用
- ふるさと納税の戦略強化
- 地元事業者の商品開発・販路支援
- 健康・スポーツ・教育・農産品を絡めた地域ブランド化
- 町外企業との実証事業誘致
- サテライトオフィス・小規模事業者誘致
```

## メリット

```markdown
- 自主財源の増加に直結しやすい
- 町外資金を取り込める
- 企業・雇用・固定資産税につながる可能性
- 「攻めの自治体経営」として説明しやすい
```

## デメリット

```markdown
- 成果まで時間がかかる
- 競合自治体が多い
- 企業誘致だけでは人口定着に直結しない場合あり
- 専門人材・営業機能が必要
```

## 評価

```markdown
必須。
ただし、生活基盤や人材定着とセットでなければ、単なる誘致合戦になる。
```

# 案C：選ばれる生活基盤型

## 「若者・女性・現役世代に選ばれる町」をつくる案

## 概要

仕事・住宅・子育て・健康・移動・学び・ウェルビーイングを一体で整え、若者・女性・働き盛り世代が「ここで暮らす意味」を感じられる町にする案。

添付メモでも、若者定着では仕事・給与・生活利便性が主因であり、健康・医療・介護環境は「将来も安心して暮らせる地域か」を支える安心材料として機能すると整理されている。

## 主な施策

```markdown
- 職員・町民向け健康増進施策
- 働き盛り世代の肩こり・腰痛・睡眠・メンタル・女性特有の健康課題への支援
- 子育て世代向けの生活導線整備
- 若者・女性向けの仕事・学び・居場所づくり
- 町内企業の健康経営・人材定着支援
- 交通・医療・買い物アクセスの再設計
- 移住定住サイトではなく、生活基盤そのものの改善
```

## メリット

```markdown
- 人口定着に直結しやすい
- 町民満足度を高めやすい
- 健康・福祉・産業・教育を横断できる
- 他自治体との差別化になる
- 町内企業の採用・定着支援にもなる
```

## デメリット

```markdown
- 効果測定が難しい
- すぐに税収増として見えにくい
- 部署横断が必要
- 旗印が曖昧だと単発施策の寄せ集めになる
```

## 評価

```markdown
最も重要。
ただし、単独では財源を生まないため、案B・案Aと統合する必要あり。
```

# 5. 推奨案

## 推奨

```markdown
案Cを旗印にし、案Bを収益エンジン、案Aを財政規律として統合する。
```

## 推奨コンセプト

```markdown
矢吹町サステナブル経営モデル
— 健康で働き続けられ、若者に選ばれ、外部資金を呼び込む町 —
```

---

## なぜ案Cを軸にするか

財政再建や自主財源確保は重要だが、それだけでは「選ばれる町」にはならないため。
人口減少時代に本当に必要なのは、**現役世代・若者・女性・子育て世代が、ここで暮らし続ける合理性を持てること**。

さらに、健康・医療・生活インフラは、今後の地域選択における安心材料となる。若者は医療・年金・公衆衛生に一定の関心を持ち、選ばれる自治体では生活満足度や住環境、子育て環境、雇用環境などが相対的に高い傾向が示されている。([内閣府ホームページ][6])

したがって、矢吹町は、単なる移住促進ではなく、次のポジションを取るべき。

```markdown
「現役世代が健康に働き続けられ、子育てし、年を重ねても暮らし続けられる町」
```

これは、あなたが進めている職員向けストレッチ実証とも接続可能。
最初は小さな健康施策でも、長期的には「町全体の健康・生産性・定着率を高める政策実証」の入口になる。

# 8. 経営メッセージ

## 首長・幹部向けメッセージ

```markdown
矢吹町の課題は、単なる人口減少ではありません。

本質は、人口が減る中で、税収・職員数・地域の担い手が細り、
一方で公共施設、医療・福祉、インフラ、生活支援の維持コストが重くなることです。

この構造に対して、従来どおりのフルセット行政を続けるだけでは、
いずれサービス水準か財政のどちらかに無理が出ます。

必要なのは、削る行政ではなく、選び直す行政です。

何を維持し、何を集約し、何に投資し、どこから外部資金を呼び込むのか。

矢吹町は、若者・女性・現役世代に選ばれる生活基盤を整えながら、
自主財源を増やし、公共施設と行政運営を再設計する段階に入っています。
```

```markdown
矢吹町が生き残るには、人口を奪い合うだけでなく、
「健康に働き続けられる町」「若者・女性に選ばれる町」「外部資金を呼び込める町」へ転換し、
自主財源と生活基盤を同時に強くする自治体経営が必要である。
```

# 10. 次の一手

まずは、いきなり大規模構想にせず、下記の順番が現実的。

```markdown
1. 矢吹町サステナビリティ診断を作る
2. 自主財源・公共施設・人口動態・健康施策を1枚に統合する
3. 職員健康実証を小さく実施する
4. 町内企業・住民向けに横展開する
5. 若者・現役世代の生活基盤調査を行う
6. 重点投資領域を3つに絞る
7. 次年度予算・総合計画・行財政改革と接続する
```

# 補足

現時点での最重要ポイントは、**健康施策を単独の福利厚生として扱わないこと**。
矢吹町にとっては、健康、財政、人口、産業、公共施設をすべて別々に見るのではなく、**「現役世代が残り、働き、税を支え、地域サービスを維持するための生活基盤投資」**として統合することが必要。
