# 肩こりの正体 ── なぜ「首と肩の間」が凝るのか

---

## 結論

**肩こりは「筋力不足」でも「仕事のせい」でもない。人間の体の構造と、日常の使い方が生み出す"自然な結果"**

凝っている場所を揉んでも戻りやすいのは、**凝りの「原因」と「結果」が別の場所にある**から

---

## 第1章：凝っている場所の正体

### 「首と肩の間のあの凝り」＝ほぼこの2つの筋

| 筋肉 | 役割 | なぜ凝るか |
|---|---|---|
| **僧帽筋の上部**（首〜肩の表層） | 肩をすくめる・頭を支える | 頭の重さを支え続けて疲労する |
| **肩甲挙筋**（肩甲骨〜首の深層寄り） | 肩甲骨を引き上げる | 肩甲骨が「下がる・回る」をサボると、代わりに頑張る |

この2つが**常にONになっている状態**＝いわゆる「肩こり」

### ポイント

凝りの正体は「筋肉が硬くなった」のではなく、**「働きすぎて休めていない」**状態。
筋肉そのものの問題というより、**神経が「ずっと力を入れろ」と命令し続けている**ことが本質

---

## 第2章：なぜ「働きすぎ」が起きるのか ── 5つの原因

### 原因①：頭が重い（構造上の宿命）

- 人間の頭は約4〜6kg（ボウリング球1個分）
- しかも重心が**前寄り**にある
- 頭が少しでも前に出ると、後ろから引っ張って支える必要がある
- その「引っ張り役」が**僧帽筋上部と肩甲挙筋**

> 💡 イメージ：釣り竿の先に重りがついた状態。竿を立てていれば楽だが、前に傾けた途端、手元にものすごい負担がかかる。これが「頭が前に出た姿勢」で首〜肩に起きていること

### 原因②：目を使うと頭が固定される

- 人間は情報の大部分を目から得ている
- スマホ・パソコン・子どもの様子を見る ── すべて「目線の固定」
- 目線が固定されると、頭も固定される
- 頭が固定されると、首〜肩の筋肉は、**「動かす役」から「固める役」に切り替わる**
- 固める役として最も動員されやすいのが、僧帽筋上部

### 原因③：呼吸が浅くなる

- 忙しい・緊張・集中 → 呼吸が浅く速くなる
- 本来、呼吸は**肋骨を広げて行う**もの（横隔膜＋肋間筋）
- 浅い呼吸では肋骨が動かず、代わりに**首・肩の筋肉が呼吸の補助に動員される**
- 僧帽筋上部と斜角筋（首の横の筋肉）が「呼吸するたびに働く」状態 → 1日に約2万回分の余計な仕事

> 💡 「肩で息をしている」という表現は、比喩ではなく実際に起きている現象

### 原因④：肩甲骨を動かす場面が日常にほぼない

- 日常動作の大半は「肘から先」で完結する
  - タイピング、スマホ操作、料理、抱っこ、着替えの補助…
- **肩甲骨はほとんど動いていない**
- 脳は使わない動きを忘れていく
- 結果：**肩甲骨＝固定物、腕＝操作部位**、という神経パターンが定着する
- 肩甲骨が動かなければ、**動きの帳尻を合わせるのは僧帽筋上部**

### 原因⑤：体は「安定」を最優先する

- 神経系の原則：**不安定より安定、動かすより固める**
- 疲れているとき、慣れない動きのとき、緊張しているとき
- → 体は胸郭（肋骨まわり）と肩甲骨を**固める方向を選ぶ**
- 固める役＝僧帽筋上部・肩甲挙筋

---

## 第3章：「揉んでも戻る」理由

### 凝りは「結果」であって「原因」ではない

凝っている場所（僧帽筋上部）を揉む・針を打つ → 一時的に筋緊張は下がる

しかし、以下が変わっていなければ**数時間〜翌日には元に戻る**：

- 頭の位置（前に出た姿勢）
- 視線の使い方
- 呼吸パターン
- 肩甲骨の使われ方
- 胸郭（背骨＋肋骨）の動き

> 💡 例えるなら：雑草を刈っても、根を取らないとまた生える。凝りを揉むのは「刈った雑草がまた伸びるのを待つ」ようなもの

---

## 第4章：肩甲骨と胸郭の関係 ── 凝りの「上流」を理解する

### 肩甲骨は「浮いている骨」

- 肩甲骨は、骨同士のつながりが鎖骨との1か所しかない
- 残りはすべて**筋肉で胸郭（肋骨）の上に吊られている**
- つまり、**胸郭の状態が肩甲骨の動きを決める**

### 胸郭が動かないと肩甲骨も動けない

肩甲骨が腕を上げるとき必要な動き：
- **上方回旋**（上向きに回る）
- **後傾**（上端が後ろに倒れる）
- **外旋**（外向きに回る）

これらはすべて、**胸郭（背骨が反る＋肋骨が広がる）が前提条件**

胸郭が固まっている（猫背・肋骨が開かない）状態では：
- 肩甲骨の回旋・後傾が**物理的に出せない**
- 腕を上げる、胸を開くといった動作で肩甲骨が動けない
- 動けない分を**僧帽筋上部が「上に引き上げる」戦略で代償する**

> 💡 イメージ：テーブルの上に置いた皿（肩甲骨）を滑らせたいのに、テーブル自体が傾いて固まっている（胸郭が硬い）。皿は滑れないので、上から持ち上げるしかない（僧帽筋上部が頑張る）

## 第5章：ママさんに特に起きやすい理由

育児期は、第2章の5つの原因が**同時に揃いやすい**

| 原因 | 育児場面 |
|---|---|
| 頭が前に出る | 授乳、おむつ替え、添い寝 |
| 視線の固定 | 子どもの見守り、スマホ（片手抱っこ中） |
| 浅い呼吸 | 睡眠不足、常時の緊張・気疲れ |
| 肩甲骨が動かない | 抱っこ紐、前かがみ姿勢の持続 |
| 体が「固める」を選ぶ | 疲労、慣れない動作の連続 |

さらに産後特有の要因：
- **腹筋群（腹横筋・骨盤底筋）の機能低下** → 体幹の安定性が下がる → 代わりに肩まわりで安定を取ろうとする
- **抱っこ紐による肩甲帯の固定** → 肩甲骨が長時間「動けない」環境に置かれる

## 第6章：解決の方向性

### 根本方針

**「凝っている場所を緩める」のではなく、「凝らなくて済む条件を作る」**

具体的には：

| やること | 狙い |
|---|---|
| **胸郭を動かす**（背骨を反る・肋骨を広げる） | 肩甲骨が動ける「土台」を作る |
| **肩甲骨の支持筋を再教育する**（下部僧帽筋・前鋸筋・広背筋） | 僧帽筋上部の「代わり」を育てる |
| **呼吸を整える**（肋骨を使った呼吸） | 僧帽筋上部を呼吸補助から解放する |

### 経験則：運動介入で改善しやすいパターン

> 以下は個人的な経験に基づく観察であり、すべての人に当てはまるとは限らない

- 針やマッサージで一時的に楽になるが**すぐ戻る**パターン → 胸郭・肩甲骨の運動介入で改善しやすかった
- 胸郭回旋（ワイパー系のストレッチ）で**即時的に凝りが軽くなる**場合 → 胸郭の可動性制限が主因である可能性が高い
- ロウ（引く動作）やラットプルダウンを**肩をすくめずに**行えるようになると、日常の凝りが減りやすかった
- 自重だけでも効果は出る。**高負荷より「正しい場所が働いている感覚」が大事**

## 第7章：器具なしエクササイズ（優先度順）

すべて共通ルール：**「首と肩の間が力まない」ことが最優先**。力んだ時点で逆効果

### ① 四つ這いロウ（肩甲骨の支持再学習）

- 四つ這いから片手を軽く浮かせ、肘を後ろに引く意識
- 実際に大きく動かなくてもOK
- 合言葉：**「首は長く、肩はすくめない」**
- 左右 各5〜8回 × 2〜3セット（1回3〜5秒キープ）

### ② 壁プル（ラットプル動作の神経学習）

- 壁に向かって立ち、両手を壁につけて「上から引き下ろす」意識
- 壁は動かないので、実際には力を入れるだけ（等尺性収縮）
- 合言葉：**「肘で引く、肩で引かない」**
- 5秒 × 5〜8回

### ③ うつ伏せYホールド（下部僧帽筋の活性化）

- うつ伏せで親指を天井に向け、Y字に腕を軽く浮かせる
- 高く上げない。首が反らない
- 5〜10秒 × 5回

### ④ 壁スライド（肩甲骨を肋骨に沿わせる制御）

- 壁に前腕をつけ、肩をすくめずに上へスライド
- 合言葉：**「肩甲骨が肋骨に吸い付く感じ」**
- 6〜10回 × 2セット

### ⑤ 胸郭回旋＋呼吸（仕上げ）

- 仰向けまたは四つ這いで体をひねり、ひねった側の背中に息を入れる
- 肋骨の広がりを感じられればOK
- 左右 各5呼吸

### よくある失敗

- 「効かせよう」として力む → 僧帽筋上部がまた出る
- 肩を下げすぎる → 別の代償が出る
- 胸を張りすぎる → 腰が反って胸郭が動いていない
- 呼吸を止める → 呼吸補助筋がまた動員される

## 第8章：「悪循環」と「好循環」のモデル

### 悪循環（多くの人が陥っているパターン）

```
頭が前に出る → 視線固定 → 呼吸浅化 → 胸郭が動かなくなる
→ 肩甲骨が動けなくなる → 僧帽筋上部が常時ON → 凝り形成
→ 凝りを揉む → 一時的に楽 → 条件が変わっていないので再燃
```

### 好循環（介入の方向性）

```
胸郭を動かす → 肋骨が広がる → 呼吸が変わる
→ 肩甲骨が動ける → 支持筋が働く → 僧帽筋上部が休める
→ 凝りの「上流」が改善 → 再燃しにくい
```

## 第9章：受診すべきサイン

以下がある場合は、セルフケアの前に医療機関（整形外科）の受診を優先

- **しびれ**（腕、手、指）
- **夜間痛**（寝ていて痛みで目が覚める）
- **安静時痛**（何もしていなくても痛い）
- **明確な筋力低下**（物を落とす、握れない）
- **外傷後の急性痛**
- **産後の強い腹部・骨盤の違和感**（腹直筋離開・骨盤底機能の評価が必要）

## まとめ

- 肩こりの「凝っている場所」は**結果**。原因は別の場所にある
- **頭の重さ × 視線固定 × 浅い呼吸 × 肩甲骨の不使用 × 体の安全戦略**が組み合わさって起きる
- 解決は「揉む」ではなく「**凝らなくて済む条件を作る**」
- 具体的には：**胸郭を動かす → 肩甲骨が動ける → 僧帽筋上部が休める**
- 器具なし・自重だけで十分。大事なのは**正しい場所が働いている感覚**
