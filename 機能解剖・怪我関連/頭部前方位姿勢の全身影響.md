# 正常な頭部ポジション

## 解剖学的基準

**矢状面での理想アライメント**
- 外耳道（耳の穴） → 肩峰（肩の出っ張り）の垂直線上
- 頭部重心位置：第2頸椎（C2）椎体レベル
- 頸椎前弯角：20-40度（C2-C7のCobb角）

**頭部の重量と力学**
- 成人頭部重量：約4.5-5.5kg（体重の6-8%）
- 正常位置での頸椎負荷：5-6kg
- **前方偏位による負荷増加の公式**
  ```
  頸椎負荷 ≈ 頭部重量 × (1 + 前方偏位角/10)
  
  例：
  0度（正常）：5kg
  15度前方：約12kg（+140%）
  30度前方：約18kg（+260%）
  45度前方：約22kg（+340%）
  60度前方：約27kg（+440%）
  ```

# 頭部前方位の分類

## タイプ別の特徴

| タイプ | 頸椎配列 | 主要な問題部位 | 典型例 |
|--------|----------|--------------|--------|
| **Type 1：上位頸椎伸展型** | C1-C2過伸展 + C3-C7屈曲 | 後頭下筋群過緊張 | デスクワーク、スマホ使用 |
| **Type 2：全体屈曲型** | C1-C7全体が屈曲（ストレート化） | 頸椎椎間板前方圧迫 | 猫背との合併 |
| **Type 3：下位頸椎伸展型** | C1-C4屈曲 + C5-C7過伸展 | 頸胸移行部ストレス | 長時間立位作業 |

# 頸部レベルでの代償機構

## 筋群の活動変化

### 過緊張（短縮）する筋群

**1. 後頭下筋群（Suboccipital Muscles）**
```
構成：
- 大後頭直筋
- 小後頭直筋  
- 上頭斜筋
- 下頭斜筋

機能：
- C1-C2の伸展・回旋制御
- 頭部の微細な位置調整

【頭部前方位での変化】
正常活動：MVC の 10-20%
    ↓
前方位：MVC の 40-70%（+200-350%）

【影響】
- 緊張性頭痛：後頭部痛（有病率2.8倍）
- 椎骨動脈圧迫（理論上、臨床的意義は議論あり）
- 大後頭神経絞扼 → 後頭神経痛
```

**2. 上部僧帽筋（Upper Trapezius）**
```
正常活動：MVC の 5-15%（立位安静時）
    ↓
前方位：MVC の 25-45%（+300-400%）

【機序】
頭部前方移動（前方へ2.5cm毎）
    ↓
頭部重心の前方偏位
    ↓
後方への引き戻しトルク必要
    ↓
上部僧帽筋の過活動

【影響】
- 肩こり（最多愁訴）
- 筋筋膜性疼痛症候群（トリガーポイント形成）
- 上肢への関連痛
```

**3. 肩甲挙筋（Levator Scapulae）**
```
正常活動：MVC の 10-15%
    ↓
前方位：MVC の 30-50%

【影響】
- 頸部側面〜肩甲骨内側上角の痛み
- 頸部回旋制限（同側への回旋時痛）
- 肩甲骨挙上拘縮
```

**4. 胸鎖乳突筋（Sternocleidomastoid：SCM）**
```
正常：姿勢保持では最小活動
    ↓
前方位：持続的等尺性収縮（MVC の 15-30%）

【機序】
頭部を後方へ引き戻す代償
特に顎が上がるパターンで顕著

【影響】
- 側頸部の張り・痛み
- 頸部回旋時痛（対側へ）
- めまい・耳鳴り（前庭系への影響、推察）
```

---

### 弱化（伸長）する筋群

**1. 深層頸部屈筋群（Deep Neck Flexors：DNF）**
```
構成：
- 頭長筋
- 頸長筋
- 前頭直筋
- 外側頭直筋

正常機能：
- 頸椎の分節的安定化
- 頭部の前後位置制御
- 頸椎前弯の保持

【頭部前方位での変化】
活動レベル：正常の 40-60%に低下
筋断面積：-15-25%（6ヶ月の前方位で）

【評価】
Cranio-Cervical Flexion Test（CCFT）
- 正常：26-30mmHgまで保持可能
- 機能不全：22mmHg以下で代償出現
```

**2. 中部・下部僧帽筋**
```
正常：肩甲骨の後退・下制
    ↓
前方位：活動低下（-30-40%）

【影響】
肩甲骨の位置異常
- 外転（abduction）
- 前傾（anterior tilt）
- 上方回旋不足
```

## 頸椎構造への影響

### 椎間板ストレス

**圧迫分布の変化**
```
【正常】
前方：60%
後方：40%

【頭部前方位（30度）】
前方：75-80%（+25-33%）
後方：20-25%（-37-50%）
```

**バイオメカニクス**
- C5/C6・C6/C7：最大負荷レベル
- 前方線維輪への圧迫応力：**1.4-1.8倍**
- 髄核の後方偏位
- 椎間板変性の加速（MRI上の変化が10-15年早い、推察）

### 椎間関節（Facet Joint）への影響

**Type 1（上位頸椎伸展型）の場合**
```
C1-C2椎間関節
    ↓
過伸展による圧迫
    ↓
関節包の後方線維への伸張ストレス
    ↓
- 関節炎（cervical facet arthropathy）
- 後頭部への関連痛
- C2神経根の刺激
```

**下位頸椎（C4-C7）の影響**
- 椎間関節の開離（前屈位での）
- 関節包の弛緩
- 分節的不安定性（segmental instability）

# 胸椎レベルでの代償

## 胸椎後弯の増強

### メカニズム
```
頭部前方位
    ↓
重心の前方移動
    ↓
代償的な胸椎後弯増強（+5-15度）
    ↓
円背（猫背）の形成
```

**Cobb角の変化**
- 正常胸椎後弯（T1-T12）：20-40度
- 頭部前方位合併：35-55度
- 構造的円背への移行：45度以上で不可逆性リスク

### 筋活動の変化

**過緊張筋**
```
【大胸筋（Pectoralis Major/Minor）】
正常：姿勢保持で最小活動
    ↓
前方位：持続的短縮位
    ↓
適応性短縮（-15-25%の柔軟性低下）

【影響】
- 肩関節屈曲制限（-10-20度）
- 肩関節内旋拘縮
- 肩甲骨外転・前傾の助長
```

**弱化筋**
```
【菱形筋（Rhomboids）】
【僧帽筋中部・下部】
活動低下：-30-50%

【結果】
肩甲骨の位置異常
    ↓
肩峰下インピンジメント
    ↓
腱板損傷リスク（1.6-2.1倍）
```

---

## 胸郭の機能不全

### 呼吸パターンの変化

**正常呼吸**
- 横隔膜主導（腹式呼吸）
- 胸郭下部の拡張（lateral expansion）
- 1回換気量：450-550ml

**頭部前方位+円背での変化**
```
胸椎後弯増強
    ↓
胸郭前後径の減少（-10-15%）
    ↓
横隔膜の機能低下（ドーム形状の平坦化）
    ↓
代償的な上部胸式呼吸（apical breathing）
    ↓
呼吸補助筋の過使用
```

**筋活動の変化**

| 筋群 | 正常 | 前方位+円背 | 変化率 |
|------|------|-----------|--------|
| **横隔膜** | 70-80% | 40-50% | -40% |
| **胸鎖乳突筋** | 5-10% | 30-45% | +300% |
| **斜角筋群** | 10-15% | 35-50% | +250% |
| **上部僧帽筋** | 5-10% | 25-40% | +300% |

**臨床的影響**
- 1回換気量：-15-25%減少
- 呼吸数：+20-30%増加（代償）
- 呼吸効率の低下
- 慢性的な酸素飽和度低下（軽度、推察）

# 腰椎-骨盤レベルでの代償

## 脊柱全体のバランス戦略

### パターン1：腰椎過前弯型（最多）

**メカニズム**
```
頭部+胸椎の前方偏位
    ↓
全身重心の前方移動
    ↓
代償的な腰椎前弯増強（+5-10度）
    ↓
骨盤前傾増加（+3-8度）
```

**筋活動の変化**
```
【過緊張】
- 腸腰筋：+40-60%
- 脊柱起立筋（腰部）：+35-55%
- 大腿直筋：+25-40%

【弱化】
- 腹横筋：-30-45%
- 多裂筋：-25-35%
- 大臀筋：-30-50%
```

**バイオメカニクス**
- 腰椎椎間関節への圧迫：**1.5-2.0倍**（特にL4/L5、L5/S1）
- 椎間板後方への圧力：正常比-30%
- 椎間板前方への圧力：正常比+40%

---

### パターン2：腰椎平坦化型（Flat Back）

**メカニズム**
```
頭部+胸椎の前方偏位
    ↓
腹筋群の代償的過活動
    ↓
骨盤後傾
    ↓
腰椎前弯の減少（-5度〜消失）
```

**特徴**
- 若年者・腹筋優位者に多い
- 体幹前傾での代償（股関節屈曲）
- 椎間板後方への圧迫増大：**1.6-2.2倍**

## 仙腸関節への影響

**Force Closure（筋性安定化）の変化**
```
正常：
- 腹横筋：35%
- 多裂筋：30%
- 骨盤底筋：20%
- 横隔膜：15%

頭部前方位+姿勢代償：
- 腹横筋：20%（-43%）
- 多裂筋：18%（-40%）
- 脊柱起立筋（代償）：45%
- 大腿筋膜張筋（代償）：17%
```

**結果**
- 仙腸関節の剪断力：**1.4-1.7倍**
- 仙腸関節痛の有病率：**1.9倍**
- 骨盤帯痛（pregnancy-related pelvic girdle pain様）

# 神経系への影響

## 末梢神経の圧迫・伸張

### 頸部レベル

**1. 腕神経叢（Brachial Plexus）**
```
【胸郭出口症候群（TOS）様症状】
円背+肩甲骨位置異常
    ↓
鎖骨下間隙の狭小化
    ↓
C8-T1神経根の圧迫
    ↓
手指のしびれ（小指側）、筋力低下
```

**発生率**
- 頭部前方位者のTOS症状：正常姿勢の**2.3倍**（推察）
- 特に上肢挙上作業で症状増悪

**2. 頸神経根**
```
【椎間孔の狭小化】
C5/C6、C6/C7レベル：最多
    ↓
上肢への放散痛（dermatome分布）
- C5：肩外側
- C6：前腕橈側、母指
- C7：中指
- C8：小指側
```

---

### 腰部レベル

**3. 腰神経根**
```
【腰椎過前弯型】
椎間関節圧迫
    ↓
椎間孔狭窄（特にL4/L5、L5/S1）
    ↓
下肢への放散痛・しびれ
```

---

## 自律神経系への影響

### 交感神経の過緊張

**メカニズム（理論）**
```
頸部筋群の慢性過緊張
    ↓
頸部交感神経節（C2-C7）への機械的刺激
    ↓
交感神経系の過活動
```

**臨床症状（関連性は議論あり）**
- 慢性疲労感
- 睡眠障害（入眠困難）
- 消化機能低下
- 末梢循環不全（冷え性）

**科学的根拠**
- 確立されたエビデンスは限定的
- 頸部マニピュレーション後の自律神経指標変化は報告あり（HRV等）
- 因果関係は不明確

# 評価・測定法

## 頭部前方位の定量評価

**1. Craniovertebral Angle（CVA：頭頸角）**
```
【測定方法】
側面写真撮影
- C7棘突起マーク
- 耳垂マーク
- C7と耳垂を結ぶ線と水平線の角度

【判定】
- 正常：53度以上
- 軽度前方位：49-53度
- 中等度：45-49度
- 重度：45度未満

【意味】
角度1度減少 = 頭部約1cm前方移動
角度10度減少 = 頸椎負荷約+120%
```

**2. Forward Head Distance（FHD：前方偏位距離）**
```
【測定】
壁を背にして立つ
後頭部と壁の距離を測定

【判定】
- 正常：0-2cm
- 軽度：2-4cm
- 中等度：4-6cm
- 重度：6cm以上
```

---

## 機能的評価

**3. Deep Neck Flexor Endurance Test**
```
【方法】
仰臥位、頭部を床から2-3cm挙上
顎を引いた状態で保持

【判定】
- 正常：40秒以上
- 機能不全：30秒未満
- 重度機能不全：15秒未満
```

**4. 胸椎伸展可動域**
```
【方法】
座位、両手を頭の後ろで組む
胸椎のみを伸展（腰椎は固定）

【判定】
- 正常：25-30度
- 制限：20度未満
```

# 是正プログラム

## Phase 1：筋の再バランス（2-4週間）

### 過緊張筋のリリース

**1. 後頭下筋群**
```
【テニスボールリリース】
- 仰臥位、後頭部下にボール設置
- 後頭骨とC1の間を探る
- 圧迫+小さく頷く動作
- 各ポイント60秒、1日2回

【ストレッチ】
- 座位、顎を引く（chin tuck）
- 後頭部を両手で前方へ押す
- 30秒×3回、1日3セット
```

**2. 上部僧帽筋・肩甲挙筋**
```
【ストレッチ】
- 座位、頭を側屈+対側回旋
- 同側の手で頭を優しく牽引
- 30秒×3回、左右

【フォームローラー】
- 上部僧帽筋を横方向にローリング
- 各側2分、1日1回
```

**3. 大胸筋**
```
【ドアフレームストレッチ】
- ドア枠に前腕を当てる
- 体を前方へ移動
- 大胸筋の伸張感確認
- 30秒×3回、1日2セット
```

---

### 弱化筋の活性化

**1. 深層頸部屈筋（DNF）**
```
【Chin Tuck Exercise】
- 仰臥位、タオルを首の下に
- 顎を引く（二重顎を作る動作）
- 後頭部でタオルを押す
- 10秒×10回、1日3セット

【進化版：圧バイオフィードバック】
- 圧計測器を首の下に設置
- 20mmHg → 22 → 24 → 26 → 28 → 30mmHgへ段階的に
- 各レベル10秒保持×10回
```

**2. 中部・下部僧帽筋**
```
【Prone Y-T-W Exercise】
- 腹臥位、胸をベンチ端から出す
- Y字・T字・W字に腕を挙上
- 肩甲骨を内側に寄せる
- 各10回×3セット、週3回
```

**3. 菱形筋**
```
【Row Exercise（チューブ/軽ダンベル）】
- 座位、胸を張る
- 肘を後方へ引く
- 肩甲骨を内側に寄せる
- 12回×3セット、週3回
```

## Phase 2：姿勢制御の再教育（4-8週間）

### 頭部ポジション意識

**1. Wall Angel Exercise**
```
【目的】
頸椎・胸椎アライメントの学習

【方法】
- 壁に背を向けて立つ
- 後頭部・肩甲骨・仙骨を壁につける
- 両腕を壁に沿って上下（W→Y）
- 10回×3セット、1日2回

【ポイント】
- 後頭部が壁から離れない
- 腰椎の過度な反りを作らない
```

**2. Brugger's Relief Position**
```
【実施】
- 座位、骨盤を前傾
- 胸を張り、肩甲骨を寄せる
- 手のひらを外向き
- この姿勢で深呼吸5回
- 1時間ごとに実施
```

---

### 胸椎可動性の向上

**1. 胸椎伸展エクササイズ（フォームローラー）**
```
【方法】
- 仰臥位、ローラーを胸椎中部に
- 両手を頭の後ろ
- 胸椎のみ伸展（腰椎は固定）
- 5回×3セット、1日1回

【進化版：セグメント別】
- ローラー位置を3-5cmずつ移動
- T4-T5、T6-T7、T8-T9を個別に
```

**2. Quadruped Thoracic Rotation**
```
【方法】
- 四つ這い
- 片手を頭の後ろ
- 胸椎を回旋（天井を見る）
- 各側10回×3セット
```

## Phase 3：日常動作への統合（継続）

### デスクワーク環境の最適化

**モニター位置**
- 高さ：目線の高さ、またはやや下（視線-10〜-20度）
- 距離：腕を伸ばして指先が画面に触れる程度（50-70cm）
- 角度：やや後傾（5-10度）

**椅子の設定**
- 座面高：足裏全体が床接地、膝90度
- 背もたれ：腰椎サポートあり
- アームレスト：肘90度で肩リラックス

**作業習慣**
- 20-20-20ルール：20分ごとに、20秒間、20フィート（6m）先を見る
- 1時間ごとに姿勢リセット（Brugger's Position）

```
【NG】

【OK】
- スマホを目線の高さまで持ち上げる

- 頸椎中間位を保持
→ 頸椎負荷：5-6kg（正常）
```

# 科学的エビデンス

## 頭部前方位と症状の関連

**研究1：頸椎負荷の計算モデル**
- 頭部15度前方で頸椎負荷12kg（+140%）
- 30度で18kg（+260%）
- 60度で27kg（+440%）
（出典：Hansraj, 2014, Surgical Technology International）

**研究2：筋活動の変化**
- 頭部前方位者の上部僧帽筋活動：+325%
- DNF活動：-42%
（出典：Yip et al., 2008, Manual Therapy）

**研究3：頭痛との関連**
- CVA<50度で緊張性頭痛オッズ比：3.1
- DNF筋力と頭痛頻度の負の相関（r=-0.58）
（出典：Fernández-de-las-Peñas et al., 2006, Cephalalgia）

**研究4：介入効果**
- 8週間の是正プログラム（DNF強化+ストレッチ）
  - CVA：+6.3度改善
  - 頸部痛VAS：-3.8点
  - 上部僧帽筋活動：-38%
（出典：Kim et al., 2015, Journal of Physical Therapy Science）

### 介入の優先戦略

**Week 1-2：症状軽減**
- 過緊張筋のリリース（毎日）

**Week 3-4：筋再バランス**
- DNF活性化（毎日）
- 中下部僧帽筋強化（週3回）

**Week 5-8：姿勢再教育**
- Wall Angels（毎日）
- 胸椎可動性向上（週3-4回）

**Week 9以降：習慣化**
- 環境調整
- 姿勢チェック習慣
- 維持プログラム（週2回）

**期間**：明確な症状改善4-8週、姿勢変化3-6ヶ月、完全是正6-12ヶ月
