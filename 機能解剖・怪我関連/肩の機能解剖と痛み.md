# 肩の機能解剖と肩こり・肩痛・腰痛連動の統合整理

## 結論

肩こり・肩痛は、肩だけを揉んだり、肩甲骨だけを動かしたりしても再発しやすい。

理由は、肩の動きが次の要素に強く影響されるから。

* 頭の位置
* 頸椎
* 胸椎
* 肋骨
* 鎖骨
* 肩甲骨
* 上腕骨
* 呼吸
* 目線
* 体幹
* 骨盤
* 腰椎
* 日常姿勢
* 作業量

一言で言えば、

> 肩こりは「肩の筋肉が弱い」だけではなく、
> **首・胸郭・肩甲骨・呼吸・日常動作の協調が崩れ、首肩が休めなくなっている状態**。

---

## 扱う範囲

* 肩甲帯の構造
* 肩こりの仕組み
* 肩痛・腕痛の見立て
* 猫背・巻き肩と肩の関係
* 広背筋と回旋制限
* 肩甲骨運動の評価
* 腰痛との連動
* 運動指導での介入順序

## 扱わない範囲

* 医療診断
* 画像診断
* 注射・投薬・手術判断
* 強い痛みの治療
* 神経症状の個別対応

---

## 注意事項

以下がある場合は、運動指導より医療機関評価を優先。

* しびれ
* 感覚低下
* 明らかな筋力低下
* 物を落とす
* 腕が急に上がらない
* 夜間痛で眠れない
* 安静時痛
* 外傷後の急性痛
* 変形
* 発熱
* 原因不明の体重減少
* がん既往
* 感染リスク
* 急に肩甲骨が浮いてきた
* 胸痛、息切れ、冷汗、左肩〜腕への放散痛

# 1. 肩こりの基本分類

## 肩こりは3分類で見る

| 分類     | 内容                     | 対応           |
| ------ | ---------------------- | ------------ |
| 本態性肩こり | 一般的な肩こり。明確な疾患背景がない     | 姿勢・運動・生活調整   |
| 心因性肩こり | ストレスや緊張による肩こり          | 呼吸・睡眠・ストレス管理 |
| 症候性肩こり | 整形外科・内科・眼科・歯科などの背景疾患あり | 医療機関評価       |

---

## 本態性肩こり

一般的な肩こり。

関与しやすい要素。

* 長時間デスクワーク
* スマホ
* 運動不足
* 睡眠不足
* 寒冷環境
* 過労
* 同じ姿勢の継続
* 肩甲骨を動かさない生活
* 呼吸の浅さ

主な部位。

* 僧帽筋上部
* 肩甲挙筋
* 後頭下筋群
* 斜角筋群
* 胸鎖乳突筋
* 小胸筋

---

## 心因性肩こり

ストレスや緊張により、交感神経が優位になり、筋緊張や血流低下が起こる肩こり。

起こりやすい場面。

* プレゼン前
* 締切前
* 対人緊張
* 会議
* 育児ストレス
* 睡眠不足
* 不安が強い時期

対応。

* 呼吸
* 低強度運動
* 睡眠
* 入浴
* 散歩
* 作業分割
* 緊張を下げる環境づくり

---

## 症候性肩こり

病気や外傷が背景にある肩こり。

関与しうるもの。

* 頸椎椎間板ヘルニア
* 頸椎症性神経根症
* 腱板損傷
* 凍結肩
* 肩関節周囲炎
* 胸郭出口症候群
* 狭心症などの内科疾患
* 眼精疲労
* 歯科疾患
* 呼吸器・消化器疾患

このタイプは、運動だけで解決しようとしない。

# 2. 肩甲帯の構造

## 肩甲帯とは

肩甲帯は、肩まわりの骨と関節のユニット。

構成。

* 肩甲骨
* 鎖骨
* 胸郭
* 上腕骨
* 胸骨
* 肋骨

肩は、肩関節だけでなく、複数の関節が連動して動く。

---

## 肩の関節セット

| 関節     | 内容                 |
| ------ | ------------------ |
| 肩甲上腕関節 | 上腕骨と肩甲骨の関節。一般的な肩関節 |
| 肩鎖関節   | 鎖骨と肩甲骨の関節          |
| 胸鎖関節   | 胸骨と鎖骨の関節           |
| 肩甲胸郭関節 | 肩甲骨が肋骨上を滑る機能的関節    |

---

## 要点

肩の不調は、肩甲上腕関節だけでなく、
**胸郭・鎖骨・肩甲骨・上腕骨の協調不全**として見る。

「肩だけが悪い」と決めつけない。

---

# 3. 肩甲骨と胸郭の関係

## 肩甲骨は浮いている骨

肩甲骨は、骨同士の直接的なつながりが鎖骨との1か所だけ。
残りは筋肉によって胸郭の上に吊られている。

つまり、
> 胸郭の形と動きが、肩甲骨の動きを決める。

---

## 胸郭が動かないと肩甲骨も動きにくい

腕を上げるには、肩甲骨に次の動きが必要。

| 肩甲骨の動き | 意味            |
| ------ | ------------- |
| 上方回旋   | 肩甲骨が上向きに回る    |
| 後傾     | 肩甲骨の上端が後ろに倒れる |
| 外旋     | 肩甲骨が外向きに回る    |

これらは、胸椎の伸展、肋骨の広がり、呼吸と関係する。

---

## 胸郭が固いと起こりやすい代償

* 肩甲骨が上方回旋しにくい
* 肩甲骨が後傾しにくい
* 腕を上げる時に肩がすくむ
* 僧帽筋上部が過剰に働く
* 首で腕を上げようとする
* 腰を反って腕を上げる
* 肩前面が詰まる

---

## 一般向け説明

肩甲骨は、肋骨の上を滑る骨。
肋骨や背中が固まっていると、肩甲骨は滑れない。
すると、本来動くべき肩甲骨の代わりに、首や肩の上の筋肉が頑張りすぎる。

---

# 4. 肩甲骨の重要3運動

## 上方回旋

腕を上げる時に、肩甲骨が上向きに回る動き。

不足すると、

* 腕が上がりにくい
* 肩がすくむ
* 肩の上が詰まる
* 首肩が疲れやすい

---

## 後傾

肩甲骨の上端が後ろへ倒れる動き。

不足すると、

* 肩前面が詰まりやすい
* 肩峰下スペースが狭くなりやすい
* 腕を上げると痛みが出やすい
* 巻き肩が強調されやすい

---

## 外旋

肩甲骨が外向きに回る動き。

不足すると、

* 肩甲骨の内側が浮きやすい
* 翼状肩甲が出やすい
* 肩甲骨が肋骨に沿いにくい
* プッシュ動作で肩が不安定になりやすい

---

# 5. 肩甲上腕リズム

## 肩甲上腕リズムとは

腕を上げる時、上腕骨と肩甲骨が連動して動く仕組み。

一般的には、腕を180°上げる時に、

* 上腕骨：約120°
* 肩甲骨：約60°

という配分で動くと説明される。

ただし、実際には全角度で一定ではなく、動作区間や個人差で変わる。

---

## なぜ重要か

肩甲骨が土台として適切に動かないと、上腕骨だけで腕を上げようとする。

その結果、

* 肩がすくむ
* 肩前面が詰まる
* 肩の外側が痛む
* 腱板や滑液包に負担がかかる
* 首肩が過剰に働く

---

## 一般向け説明

腕は、腕だけで上がっているわけではない。
肩甲骨という土台が一緒に動くから、腕が高く上がる。
土台が動かないまま腕だけ上げると、肩の中が詰まりやすくなる。

---

# 6. 腱板の機能解剖

## 腱板とは

腱板は、肩関節を安定させる筋腱群。

構成。

| 筋    | 主な役割           |
| ---- | -------------- |
| 棘上筋  | 腕を横に上げる初動、肩の安定 |
| 棘下筋  | 外旋             |
| 小円筋  | 外旋補助           |
| 肩甲下筋 | 内旋             |

---

## 腱板の重要な役割

腱板の主な役割は、上腕骨頭を肩甲骨の受け皿に引き寄せること。

つまり、

> 腕を動かす筋であると同時に、肩を中心に保つ筋。

腱板がうまく働かないと、上腕骨頭が上方や前方へずれやすく、肩の詰まりや痛みにつながる場合がある。

---

# 7. 肩甲骨安定筋

## 主要筋

| 筋     | 役割               | うまく働かない時      |
| ----- | ---------------- | ------------- |
| 前鋸筋   | 肩甲骨を肋骨に密着、上方回旋   | 翼状肩甲、プッシュ不安定  |
| 僧帽筋下部 | 肩甲骨下制、上方回旋、後傾    | 肩すくみ、挙上時の詰まり  |
| 僧帽筋中部 | 肩甲骨内転            | 姿勢保持低下        |
| 菱形筋   | 肩甲骨内転、下方回旋       | 固めすぎると上方回旋を邪魔 |
| 小胸筋   | 肩甲骨前傾・下制         | 短縮で巻き肩、後傾制限   |
| 広背筋   | 上腕骨内旋・伸展、体幹と腕の連結 | 硬いと挙上・回旋制限    |

---

# 8. 広背筋の役割

## 広背筋とは

広背筋は、背中の大きな筋肉で、体幹と腕をつなぐ橋渡し。

起始。

* 胸腰筋膜
* 下位胸椎
* 腰椎
* 腸骨稜
* 下位肋骨

停止。

* 上腕骨小結節稜

---

## 主な作用

| 作用       | 動き          |
| -------- | ----------- |
| 肩関節伸展    | 腕を後ろへ引く     |
| 肩関節内転    | 腕を体側へ引く     |
| 肩関節内旋    | 腕を内側へひねる    |
| 体幹と上肢の連結 | 体幹の力を腕へ伝える  |
| 呼吸補助     | 胸郭後下部の動きに関与 |

---

## 広背筋が硬いと起こりやすいこと

* 腕が上がりにくい
* 腰を反って腕を上げる
* 肩甲骨が下制方向に固定される
* 体幹回旋が出にくい
* 振り向きづらい
* 歩行時の腕振りが硬い
* 肩前面が詰まりやすい
* 腰背部が張りやすい

---

# 9. 肩こりの正体

## 凝りやすい主な筋

| 筋     | 位置      | 役割           | 凝りやすい理由    |
| ----- | ------- | ------------ | ---------- |
| 僧帽筋上部 | 首〜肩の表層  | 肩をすくめる、頭を支える | 頭部前方位、肩すくみ |
| 肩甲挙筋  | 首〜肩甲骨上角 | 肩甲骨を引き上げる    | 肩甲骨の動き不足   |
| 後頭下筋群 | 頭の付け根   | 頭部の微調整       | 目線固定、スマホ   |
| 斜角筋群  | 首横      | 呼吸補助、頸部安定    | 浅い呼吸、緊張    |
| 胸鎖乳突筋 | 首前外側    | 頭部制御         | 頭部前方位      |

---

## 肩こりの本質

肩こりは、単に筋肉が硬い状態ではなく、
**首肩の筋肉が休めず、常に低レベルで働き続けている状態**。

つまり、

```txt id="9f4d77"
胸郭が動かない
  ↓
肩甲骨が動かない
  ↓
腕を上げる・支える時に首肩が代償
  ↓
僧帽筋上部・肩甲挙筋が常時ON
  ↓
肩こり
```

# 10. 肩こりが起こる5つの要因

## 1. 頭が前に出る

頭は約4〜6kgある。

頭が前に出るほど、首の後ろや肩の筋肉が頭を支え続ける必要がある。

起こりやすい場面。

* スマホ
* PC
* 読書
* 子どもの見守り
* 運転
* 長時間座位

---

## 2. 目線が固定される

目線が固定されると、頭と首も固定されやすい。

首肩の筋肉は、動かす役から固める役に変わる。

例。

* PC画面を見続ける
* スマホを見続ける
* 細かい作業
* 緊張した見守り
* 運転

---

## 3. 呼吸が浅くなる

緊張や集中で呼吸が浅くなると、首や肩の筋肉が呼吸補助に使われやすい。

特に関与しやすい筋。

* 斜角筋
* 胸鎖乳突筋
* 僧帽筋上部
* 小胸筋
* 広背筋

---

## 4. 肩甲骨を動かす場面が少ない

日常動作の多くは、肘から先だけで完結する。

例。

* タイピング
* スマホ
* 料理
* 書く
* 抱っこ
* マウス操作

肩甲骨を動かさない生活が続くと、肩甲骨は固定されやすい。

---

## 5. 身体は安定を優先する

疲労、不安、痛み、慣れない動作では、身体は動かすより固める戦略を選びやすい。

固める役として使われやすいのが、

* 僧帽筋上部
* 肩甲挙筋
* 斜角筋群
* 広背筋
* 脊柱起立筋

---

# 11. 揉んでも戻る理由

## 凝りは結果

揉む、押す、鍼をすることで一時的に筋緊張が下がることはある。

ただし、次が変わっていなければ戻りやすい。

* 頭の位置
* 目線固定
* 呼吸の浅さ
* 胸郭の硬さ
* 肩甲骨の動き不足
* 作業量
* 睡眠不足
* ストレス
* 筋持久力不足

---

## 一般向け説明

肩こりを揉むのは、火災報知器の音を一時的に止めるようなもの。
楽にはなるが、火元が残っていればまた鳴る。
火元は、胸郭・肩甲骨・呼吸・作業姿勢・休息不足にあることが多い。

---

# 12. 姿勢と肩こり・肩痛の関係

## 基本整理

姿勢が悪いから必ず痛くなる、とは言えない。

ただし、猫背や巻き肩は、肩甲骨や胸郭の動きを不利にしやすい。

安全な表現。

> 姿勢は痛みの直接原因とは限らないが、
> 肩が動きにくくなる条件にはなりやすい。

---

## 猫背

猫背は、胸椎後弯が強くなり、背中が丸くなる姿勢。

起こりやすいこと。

* 胸郭が動きにくい
* 肩甲骨が前傾しやすい
* 腕を上げると肩がすくみやすい
* 首が前に出やすい
* 呼吸が浅くなりやすい
* 腰を反って腕を上げやすい

---

## 巻き肩

巻き肩は、肩甲骨が外転・前傾・内旋方向に偏り、肩が前に出る状態。

起こりやすいこと。

* 肩前面が詰まりやすい
* 小胸筋が硬くなりやすい
* 上腕骨が内旋しやすい
* 肩峰下スペースが不利になりやすい
* 腕を上げる時に違和感が出やすい

---

## 巻き肩＋広背筋過緊張

巻き肩と広背筋の硬さが重なると、肩甲骨が前傾・外転・下制方向に固定されやすい。

その結果。

* 肩甲骨の上方回旋が出にくい
* 肩甲骨の後傾が出にくい
* 腕を上げると肩前面が詰まる
* 首肩がすくむ
* 腰を反って代償する
* 体幹回旋が硬くなる

---

# 13. 肩痛・腕痛の見立て

## まず確認すること

肩や腕が痛い場合、肩こりと決めつけない。

確認すべきこと。

* 痛みの場所
* 痛みの出る角度
* 夜間痛
* しびれ
* 筋力低下
* 可動域制限
* 外傷歴
* 首の動きで変わるか
* 寝方で変わるか
* 手指まで症状があるか

---

## 痛みの場所別の見立て

| 訴え           | 推察される候補         | 見るポイント        |
| ------------ | --------------- | ------------- |
| 肩の前〜力こぶの上が痛い | 上腕二頭筋長頭腱、肩前方組織  | 結節間溝、肘屈曲、回外   |
| 肩の横〜上腕外側が痛い  | 腱板関連痛、肩峰下滑液包    | 外転、挙上、夜間痛     |
| 肩全体が痛く硬い     | 凍結肩、肩関節拘縮       | 他動可動域も制限      |
| 腕の後ろ側が痛い     | 腱板後方、後方関節包、頸椎由来 | 首の動き、しびれ      |
| 首から腕に響く      | 頸椎神経根症          | しびれ、感覚低下、筋力低下 |
| つり革でしびれる     | 胸郭出口症候群など       | 腕挙上時のしびれ      |

---

# 14. 夜間痛の見立て

## 肩関節由来で起こりやすい理由

寝ている時は、肩甲骨が動きにくく、肩関節に圧や牽引がかかりやすい。

起こりやすい条件。

* 痛い側を下にする
* 仰向けで肩が後ろに落ちる
* 寝返りで肩が伸展・外旋される
* 肩峰下組織が圧迫される
* 腱板・滑液包が刺激される
* 上腕二頭筋長頭腱が引っ張られる

---

## 頸椎由来で起こりやすい理由

枕や寝姿勢によって、首の神経の出口が狭くなり、腕に痛みやしびれが出ることがある。

疑うサイン。

* 首を反らすと悪化
* 首を横に倒すと悪化
* 指先までしびれる
* 握力低下
* 腕のだるさ
* 手を頭に乗せると楽

---

## 安全な初期対応

* 痛い側を下にして寝ない
* 仰向けでは肘〜上腕の下にクッション
* 肩が後ろへ落ちすぎない位置にする
* 痛い角度でのストレッチを避ける
* 強い肩回しを避ける
* 夜間痛が強い場合は受診

---

# 15. 主な肩の病態整理

## 1. 腱板関連痛

特徴。

* 肩の外側〜上腕外側に痛み
* 腕を上げると痛い
* 横向きで痛い
* 夜間痛が出る場合あり
* 中高年で多い
* 腱板断裂があっても無症状の人もいる

対応の考え方。

* 痛みの出る角度を一時的に減らす
* 肩甲骨の上方回旋・後傾を改善
* 腱板の低負荷トレーニング
* 急な高負荷プレスを避ける

---

## 2. 上腕二頭筋長頭腱関連痛

特徴。

* 肩の前側が痛い
* 力こぶの上が痛いと表現されやすい
* 物を持ち上げると痛い
* 肘を曲げる、手のひらを上に向ける動作で痛い
* 寝返りで痛い

対応の考え方。

* 強い肩関節伸展ストレッチを避ける
* 肘屈曲・回外の負荷を調整
* 肩甲骨と腱板の安定性を戻す
* 痛み0〜3/10程度の等尺性から開始

---

## 3. 肩峰下痛・インピンジメント関連

特徴。

* 腕を横から上げる途中で痛い
* 60〜120°付近で痛みが出やすい
* 肩の前外側が痛い
* 肩をすくめると悪化しやすい
* 親指を下にして挙上すると痛みが出る場合あり

対応の考え方。

* 痛い角度帯を避ける
* 肩甲骨の上方回旋・後傾を作る
* 前鋸筋・僧帽筋下部を使う
* 胸椎伸展を出す
* 肩をすくめない挙上を再学習

---

## 4. 凍結肩・肩関節拘縮

特徴。

* 肩全体が痛い
* 夜間痛がある
* 自分で動かしても、他人が動かしても硬い
* 外旋が特に制限されやすい
* 手を背中に回しにくい
* 数か月〜年単位で経過することがある

対応の考え方。

* 強引に動かさない
* 痛みが強い時期は負荷調整優先
* 可動域維持を痛みの範囲内で行う
* 医療機関と連携

---

## 5. 頸椎神経根症

特徴。

* 首から肩、腕、手指へ痛みやしびれ
* 首を反らすと悪化することがある
* 感覚低下
* 筋力低下
* 握力低下
* 肩甲骨周囲の強い痛み

対応の考え方。

* 肩こりとして扱わない
* しびれや筋力低下がある場合は受診
* 首の強いストレッチや牽引を自己判断で行わない

---

## 6. 胸郭出口症候群

特徴。

* 腕を上げるとしびれる
* つり革姿勢で症状
* 鎖骨周辺、首、胸前、腕に違和感
* 手のだるさ
* 冷感や色調変化が出る場合あり

対応の考え方。

* 神経・血管症状を確認
* 強い症状は医療機関へ
* 小胸筋、斜角筋、鎖骨周囲の負荷を整理
* 呼吸と胸郭の動きを改善

---

# 16. 肩甲骨運動異常

## 肩甲骨運動異常とは

肩甲骨の動きや位置が、腕の動きに対してうまく協調していない状態。

これは診断名ではなく、**動作上の所見**。

---

## よく見るパターン

| パターン  | 見え方        | 関与しやすい筋      |
| ----- | ---------- | ------------ |
| すくみ優位 | 肩が耳に近づく    | 僧帽筋上部、肩甲挙筋   |
| 前巻き優位 | 肩が前へ落ちる    | 小胸筋、大胸筋、広背筋  |
| 翼状肩甲  | 肩甲骨内側が浮く   | 前鋸筋          |
| 下制固定  | 肩甲骨が下に引かれる | 広背筋、小胸筋      |
| 上げにくさ | 肩より上で詰まる   | 前鋸筋、僧帽筋下部、腱板 |

---

## 注意点

肩甲骨の動きが悪いから痛い、と断定しない。

肩甲骨運動異常は、原因にも結果にもなり得る。

見るべきことは、

* 痛みと一致するか
* 疲労で悪化するか
* 修正すると痛みが減るか
* 反復で崩れるか
* 日常動作で再現するか

---

# 18. 肩こりへの介入フレーム

肩こりは、凝っている場所を直接ほぐすだけでなく、
**凝らなくて済む条件を作る**。

```txt id="28okne"

  ↓
```

* どの姿勢で出るか
* どの高さで痛いか
* どの寝方で痛いか
* 翌日に残るか
* ストレスや睡眠と関係するか
* スマホ・PC・抱っこ時間
* 片側作業の量

* 作業を分割
* 肘を支える
* 画面を上げる
* 肩より上の作業を一時的に減らす

* 寝る姿勢を調整

# 20. 肩こり・肩痛で避けたいこと

## 避けたい行為

* 痛みを我慢した強い肩回し
* 反動を使うストレッチ
* 強い二頭筋ストレッチ
* 夜間痛が強い時の高負荷プレス
* 肩を下げすぎる意識
* 肩甲骨を寄せ続ける意識
* 首を強く揉み続ける
* しびれを無視する
* 痛みを筋肉痛扱いする

---

## 理由

肩は可動域が広く、安定性を筋に依存する関節。

痛みがある時に無理なストレッチや反動運動を行うと、腱板、関節包、滑液包、神経症状を悪化させる場合がある。

# 22. 肩と腰痛の連動

## 肩こりと腰痛はつながる

肩こりと腰痛は別々に見えるが、胸郭・広背筋・胸腰筋膜・骨盤を介してつながる。

特に重要な構造。

* 広背筋
* 胸腰筋膜
* 大殿筋
* 腹斜筋群
* 脊柱起立筋
* 腰方形筋
* 胸椎
* 骨盤
* 股関節

---

## 広背筋と腰痛

広背筋は、腕と骨盤をつなぐ大きな筋。

硬くなると、

* 腕が上がりにくい
* 胸郭が回旋しにくい
* 腰を反って腕を上げる
* 背中から腰に張りが出る
* 歩行時の腕振りが硬くなる
* 体幹回旋が腰に集中する

---

## 胸郭が硬いと腰が代償する

胸椎が伸展・回旋できないと、腰椎が代わりに反ったり捻ったりする。

起こりやすい代償。

* 腕を上げると腰が反る
* 振り向く時に腰で捻る
* スクワットで肋骨が開く
* デッドリフトで腰背部が固まる
* 呼吸が浅くなる

---

## 一般向け説明

肩と腰は、背中の筋膜でつながっている。
背中や肋骨が動かないと、肩も腰も同じように固まりやすい。
肩こりが強い人に腰の張りが出ることがあるのは、別々の問題ではなく、体幹の動き不足が共通していることがある。

# 26. 一般向け説明テンプレート

## 肩こり

> 肩こりは、肩の筋肉が弱いというより、首肩が休めなくなっている状態です。
> 背中や肋骨が動かないと、肩甲骨が動けず、首肩が代わりに頑張ります。
> だから揉むだけでなく、呼吸・胸郭・肩甲骨を一緒に動かすことが大切です。

---

## 肩甲骨

> 肩甲骨は腕を動かす土台です。
> 土台が動かないと、腕だけで無理に上げることになり、肩が詰まりやすくなります。
> 肩甲骨を寄せるだけではなく、上に回る・後ろに倒れる動きが必要です。

---

## 巻き肩

> 巻き肩は、見た目だけの問題ではありません。
> 肩甲骨が前に倒れたまま固まると、腕を上げる時の通り道が狭くなります。
> 胸を張るだけではなく、肋骨と肩甲骨が動くことが大切です。

---

## 夜間痛

> 寝ている時に肩が痛い場合、ただの肩こりではなく、腱や関節包、首の神経が関係することがあります。
> 睡眠が妨げられるほど痛い、しびれがある、腕が上がらない場合は医療機関で確認した方が安全です。

---

## 腰痛との関係

> 肩と腰は別々に見えて、背中の筋膜や肋骨の動きでつながっています。

# 片手ダンベルローイングの狙いと一般向け代替種目

## 前提

対象は、**急性外傷・しびれ・麻痺・夜間痛・強い放散痛がない一般成人**を想定。

目的は、片手ダンベルローイングそのものの筋力向上ではなく、以下。

* 肩甲骨の動き改善
* 胸椎伸展（背中上部を軽く伸ばす動き）の再獲得
* 首・肩まわりの過緊張軽減
* 肩こり感の軽減が期待できる身体操作づくり

---

片手ダンベルローイングは、肩こり対策として見るなら、**背中を鍛える種目**ではなく、
**肩甲骨を肋骨の上で前後に動かし、胸椎を軽く伸ばし、首に頼らず腕を使うための種目**。

肩こり感が減った可能性は、片手DBロー単独ではなく、以下の組み合わせによるものと考えるのが妥当。

| 要素           | 役割                        |
| ------------ | ------------------------- |
| 片手DBロー       | 肩甲骨内転、肩関節伸展、体幹抗回旋         |
| 胸椎伸展         | 肩甲骨が動く土台づくり               |
| 肩甲骨外転プッシュアップ | 前鋸筋による肩甲骨の安定              |
| ラットプル未実施     | 過剰な肩甲骨下制・広背筋固めを避けられている可能性 |

ただし、**肩こり改善を断定する表現は避けるべき**。
正確には、以下。

> 肩甲骨・胸椎・体幹の動きが改善し、首肩まわりへの負担分散が起きた結果、肩こり感が軽減した可能性。

# 2. 片手DBローで本当に狙っている機能

## ① 肩甲骨内転

肩甲骨内転は、肩甲骨を背骨側へ寄せる動き。

主に関わる筋。

| 筋            | 役割           |
| ------------ | ------------ |
| 僧帽筋中部        | 肩甲骨を背骨側へ寄せる  |
| 菱形筋          | 肩甲骨内転・軽い下方回旋 |
| 僧帽筋下部        | 肩甲骨の後傾・下制補助  |
| 三角筋後部        | 肩関節水平伸展      |
| ローテーターカフ後方線維 | 上腕骨頭の安定      |

肩こり対策で重要なのは、肩甲骨を強く寄せることではなく、
**肩甲骨が固まらずに、肋骨の上を滑ること**

---

## ② 肩関節伸展

肘を後ろに引く動きでは、肩関節伸展が入る。

主に関わる筋。

| 筋       | 役割          |
| ------- | ----------- |
| 広背筋     | 肩関節伸展・内転・内旋 |
| 大円筋     | 肩関節伸展・内転・内旋 |
| 三角筋後部   | 肩関節伸展・水平伸展  |
| 上腕三頭筋長頭 | 肩関節伸展補助     |

ここで注意点。

広背筋を効かせすぎると、

* 肋骨が開く
* 腰を反る
* 肩甲骨を下げすぎる
* 首まわりが固まる

という代償が出ることがある。

そのため、肩こり対策目的では、
**広背筋で強く引くより、肩甲骨と胸椎を連動させることが優先**。

---

## ③ 胸椎伸展

胸椎伸展は、背中上部を軽く伸ばす動き。

片手DBローに胸椎伸展を入れると、

* 肩甲骨が後傾しやすい
* 鎖骨が横に広がりやすい
* 肩前面の詰まりが減りやすい
* 頸部伸展代償が減りやすい
* 僧帽筋上部・肩甲挙筋の過緊張が減りやすい

ただし、一般向けではここが一番ズレやすい。

NGはこれ。

* 胸を張る
* 腰を反る
* 顎を上げる
* 肩を後ろに固める
* 肋骨を前に突き出す

正しくは、
**腰ではなく、みぞおちの裏〜肩甲骨の間あたりを軽く起こす感覚**

---

## ④ 体幹抗回旋

片手DBローは片側負荷。
そのため、身体には回旋方向の力がかかる。

必要な機能。

| 機能     | 内容              |
| ------ | --------------- |
| 抗回旋    | 体がねじれすぎないように止める |
| 抗側屈    | 体が横に倒れないように止める  |
| 股関節ヒンジ | 骨盤と背骨を安定させる     |
| 腹圧保持   | 肋骨と骨盤の距離を保つ     |

肩こり対策としても、ここは重要。
肩だけで引くのではなく、**体幹で土台を作って腕を動かす練習**になるため。

# 3. 肩こりに関係する筋の整理

肩こり感を減らす文脈では、「どの筋を鍛えるか」より、

# 6. 一般向けレッスンでの置き換え方

## 基本方針

片手DBローをそのまま一般向けに出すと、以下が起きやすい。

* 首がすくむ
* 肘だけ曲げる
* 腰を反る
* 肩甲骨を寄せすぎる
* 肩甲骨を下げすぎる
* 背中ではなく腕だけ疲れる

そのため、一般向けには、いきなりローイングではなく、
**肩甲骨が動く順番を作る**方がよいです。

順番はこれ。

1. 胸椎を動かす
2. 肩甲骨を外へ動かす
3. 肩甲骨を背骨側へ戻す
4. 首をすくめずに腕を動かす
5. 体幹を保ちながら片手動作へ進む

肩こりは、首が悪いというより、背中と肩甲骨が動かなくなって首が頑張りすぎている状態に近いです。
片手ダンベルローイングは、肩甲骨を背中側へ戻して、首ではなく背中で腕を支える練習です。

# 最終判断

片手DBローの一般向け置き換えは、以下の4種目が最も実用的。

1. **壁プッシュアッププラス**
   → 前鋸筋、肩甲骨外転

2. **壁ローイング**
   → 僧帽筋中部、菱形筋、三角筋後部

3. **壁スライド**
   → 前鋸筋、僧帽筋下部、肩甲骨上方回旋

4. **四つ這い片手リーチ＆ロー**
   → 肩甲骨前後運動、体幹抗回旋

この4つで、
**肩甲骨を広げる・戻す・上に動かす・片手で使う**までカバー可能。

個別の状態評価に基づく最終判断は、担当の医師・理学療法士・本人の評価による。

## 肩甲骨が肋骨に沿う状態

同じ「肩甲骨が肋骨に沿う」でも、意味がまったく違います。

* **前鋸筋で肩甲骨を肋骨に這わせる**
  → 肩甲骨が肋骨の上を**密着しながら滑れる状態**

* **筋肉がべったり張り付いて巻き肩っぽい状態**
  → 肩甲骨が前に固まり、肋骨の上を**滑れない状態**

一言で言うと、
**前鋸筋の良い働きは「吸盤つきのスライド」**。
悪い張り付きは、**「接着剤で固定」**。

---

# 【専門層】

## 1. 前鋸筋で肩甲骨を肋骨に這わせる状態

［確立］
前鋸筋は、第1〜第8/9肋骨あたりから起こり、肩甲骨内側縁の前面に付く筋。主な役割は以下。

| 機能       | 内容                       |
| -------- | ------------------------ |
| 肩甲骨外転    | 肩甲骨を肋骨上で外側・前方へ滑らせる       |
| 肩甲骨上方回旋  | 腕を上げる時に肩甲骨を上向きに回す        |
| 肩甲骨後傾    | 肩甲骨の上部を後ろへ倒し、肩前面の詰まりを減らす |
| 肩甲骨の胸郭固定 | 肩甲骨内側縁が浮かないように肋骨へ沿わせる    |

ここで大事なのは、前鋸筋は肩甲骨を**固める筋**ではなく、
**肋骨に沿わせながら動かす筋**ということ。

良い前鋸筋の働きでは、

* 肩甲骨が浮かない
* 肩がすくまない
* 首が力まない
* 肋骨が開きすぎない
* 腕を上げる時に肩甲骨が自然に上方回旋する
* 肩甲骨が前にも後ろにも滑れる

という状態になります。

つまり、**安定しているけど、動ける**。

---

## 2. 「べったり張り付いて巻き肩っぽい」状態

これは多くの場合、前鋸筋がきれいに働いている状態ではありません。

むしろ以下の組み合わせが多いです。

| 要素          | 起きていること                |
| ----------- | ---------------------- |
| 胸椎後弯増大      | 背中が丸まり、肩甲骨が前に流れる       |
| 肩甲骨外転位固定    | 肩甲骨が外へ開いたまま戻りにくい       |
| 肩甲骨前傾       | 肩甲骨の上部が前に倒れ、肩前面が詰まりやすい |
| 肩甲骨内旋       | 肩甲骨の内側縁が後ろへ浮きやすい       |
| 小胸筋短縮       | 肩甲骨を前傾・下制方向へ引きやすい      |
| 広背筋・大胸筋優位   | 肋骨を開く、肩を内旋方向へ固める       |
| 僧帽筋中部・下部低活動 | 肩甲骨を背骨側へ戻せない           |
| 前鋸筋の機能不全    | 肩甲骨を肋骨に沿わせて動かせない       |

見た目としては「肩甲骨が背中に張り付いている」ように見えても、実際には、
**肩甲骨が良い位置に密着しているのではなく、前方・外側・前傾位で固まっている**ことが多いです。

---

# 3. 違いの本質

## 良い密着と悪い張り付きの違い

| 観点     | 前鋸筋で這わせる   | 巻き肩っぽい張り付き      |
| ------ | ---------- | --------------- |
| 状態     | 動的安定       | 固定・過緊張          |
| 肩甲骨    | 肋骨上を滑れる    | 前に流れたまま戻りにくい    |
| 胸椎     | 軽く伸展できる    | 丸まりやすい          |
| 肩甲骨の角度 | 後傾・上方回旋が出る | 前傾・下方回旋傾向       |
| 首      | 力みにくい      | すくみやすい          |
| 肩前面    | 詰まりにくい     | 詰まりやすい          |
| 腕上げ    | スムーズ       | 引っかかる・重い        |
| 筋の使い方  | 前鋸筋＋僧帽筋下部  | 小胸筋・広背筋・僧帽筋上部優位 |
| 感覚     | 脇の下で支える    | 胸・首・肩前が固い       |

ポイントは、
**肩甲骨が肋骨に近いかどうかではなく、肋骨の上を滑れるかどうか**。

---

# 4. 「前鋸筋が効く」と「巻き肩」はなぜ見た目が似るのか

理由は、どちらも一見すると肩甲骨が外転、つまり外へ開くからです。

ただし、質が違います。

## 前鋸筋が効いている外転

* 肩甲骨が肋骨に沿って外へ滑る
* 内側縁が浮かない
* 肩がすくまない
* 胸椎が丸まりすぎない
* 腕を上げる動きにつながる
* 戻す動きもできる

## 巻き肩の外転

* 肩甲骨が外へ開いたまま固定
* 肩甲骨前傾が強い
* 小胸筋・大胸筋で前に引かれる
* 胸椎が丸い
* 首が前に出る
* 肩甲骨を背骨側へ戻しにくい

同じ「肩甲骨が外にある」でも、
**前鋸筋の外転は動作の一部**、
**巻き肩の外転は固定姿勢**。

---

# 5. 見分け方

## 良い前鋸筋パターン

壁プッシュアッププラスをした時に、

* 肩がすくまない
* 首が長い
* 肩甲骨の内側が浮かない
* 脇の下〜肋骨横に効く
* 胸を潰さずに背中が少し広がる
* 戻す時に肩甲骨も自然に戻る

この場合は良い方向。

---

## 悪い巻き肩パターン

同じ動きで、

* 肩が耳に近づく
* 首が前に出る
* 胸が潰れる
* 背中を丸めてごまかす
* 肩前面が詰まる
* 肩甲骨を戻せない
* 小胸筋あたりが硬く感じる
* 脇ではなく首・胸・腕だけ疲れる

この場合は、前鋸筋というより、
**胸郭ごと丸めて肩を前に出しているだけ**の可能性。

---

# 6. 一般向けに説明するなら

「肩甲骨を肋骨に這わせる」は、単に背中を丸めることではありません。

正しい狙いはこれ。

> 肩甲骨を肋骨から浮かせず、でも固めず、肋骨のカーブに沿って前後に滑らせること。

巻き肩の人は、肩甲骨が前にあること自体より、
**前に行った肩甲骨が戻れないこと**が問題。

だから必要なのは、

1. 前鋸筋で肩甲骨を外へ滑らせる
2. 僧帽筋中部・菱形筋で肩甲骨を背骨側へ戻す
3. 僧帽筋下部で肩甲骨の後傾・上方回旋を助ける
4. 胸椎伸展で肩甲骨が動く土台を作る

この4つ。

* 顎が前に出る

## 肩甲骨を戻す種目

### 壁ローイング

狙い：僧帽筋中部、菱形筋、三角筋後部。

キュー。

* 「肘で壁を後ろに押す」
* 「背中の真ん中で受ける」
* 「肩甲骨を軽く背骨側へ戻す」
* 「寄せすぎない」

NG。

* 胸を張りすぎる
* 腰を反る
* 肩を下げすぎる
* 肩甲骨を強く寄せる

## 上げる動きにつなげる種目

### 壁スライド＋プラス

狙い：前鋸筋、僧帽筋下部、肩甲骨上方回旋。

キュー。

* 「壁を軽く押したまま腕を上に滑らせる」
* 「肩ではなく脇の下で支える」
* 「肋骨を前に出さない」
* 「上げる高さより、首が楽なこと優先」

# 8. 指導上の最重要ポイント

肩甲骨が動かない人に、いきなり、

* 肩甲骨を寄せて
* 胸を張って
* 肩を下げて

と言うと、巻き肩の逆方向に固めるだけになりやすいです。

必要なのは、
**前に行ける、後ろに戻れる、上に回れる、浮かない**という4条件。

つまり、

| 必要な動き  | 種目          |
| ------ | ----------- |
| 前に滑る   | 壁プッシュアッププラス |
| 後ろに戻る  | 壁ローイング      |
| 上に回る   | 壁スライド       |
| 浮かずに動く | 四つ這いリーチ＆ロー  |

このセットが安全。

---

# 【伝え方】

肩甲骨を肋骨に這わせるのは、肩甲骨を「接着剤で貼る」ことではありません。
正しくは、肩甲骨が肋骨のカーブに沿って、浮かずにスーッと滑れる状態です。
巻き肩っぽい人は、肩甲骨が前にあるだけでなく、前に行ったまま戻れない状態。
だから、肩甲骨を広げる練習と、背中側へ戻す練習の両方が必要です。
