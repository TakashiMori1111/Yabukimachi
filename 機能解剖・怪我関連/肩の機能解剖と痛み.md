# 肩の機能解剖と肩こり・肩痛の統合整理

## 前提・範囲

- **対象**：一般成人（外傷直後・発熱・麻痺/しびれ・急激な筋力低下なし想定）
- **位置づけ**：一般的情報のみ（診断・治療の個別指示なし）
- **用語**：専門用語の直後に平易な補足（かっこ内）付記

---

## 全体像

- **肩こりの正体** ＝ 僧帽筋上部と肩甲挙筋が「働きすぎて休めていない」状態。筋力不足ではなく、**神経が「ずっと力を入れろ」と命令し続けている**ことが本質
- **猫背（胸椎後弯：背中の丸まり）・巻き肩（肩甲骨の外転/内旋：肩が前に出て内側に巻く）** は、肩甲骨運動を不利にする条件
- ただし**「姿勢＝痛みの直接原因」は定説ではない**（相関はあるが因果は弱い/限定的）
- 俗称「四十肩・五十肩」は医学的には**複数病態の混在**（凍結肩、腱板損傷、石灰沈着性腱炎等）
- **解決の方向** ＝ 「凝っている場所を緩める」のではなく「**凝らなくて済む条件を作る**」
- **実務フレーム** ＝ ①負荷と症状の交通整理 → ②動きの下地（胸郭・肩） → ③肩甲骨協調の再学習 → ④日常動作へ統合

---

# 【A】肩甲帯の構造と機能解剖

## 1. 肩甲帯の構造と関節

### 肩甲帯（肩まわりの骨のユニット）

- 肩甲骨＋鎖骨＋胸郭（肋骨）

### 肩の関節セット

- **肩甲上腕関節**（腕の骨と肩甲骨の関節 ＝ 一般に"肩関節"）
- **肩鎖関節**（鎖骨と肩甲骨の関節）
- **胸鎖関節**（鎖骨と胸骨の関節）
- **肩甲胸郭関節**（肩甲骨が肋骨上を滑る"機能的関節"）

### 要点

- 「肩関節だけを見ない」前提
- 肩の不調の多く ＝ 肩甲骨単体の問題ではなく、胸郭・鎖骨・上腕を含む**協調不全**

---

## 2. 肩甲骨と胸郭の関係

### 肩甲骨は「浮いている骨」

- 肩甲骨は骨同士のつながりが**鎖骨との1か所しかない**
- 残りはすべて**筋肉で胸郭（肋骨）の上に吊られている**
- つまり**胸郭の状態が肩甲骨の動きを決める**

### 胸郭が動かないと肩甲骨も動けない

腕を上げるとき肩甲骨に必要な動き：
- **上方回旋**（上向きに回る）
- **後傾**（上端が後ろに倒れる）
- **外旋**（外向きに回る）

これらはすべて**胸郭（背骨が反る＋肋骨が広がる）が前提条件**

胸郭が固まっている（猫背・肋骨が開かない）状態では：
- 肩甲骨の回旋・後傾が**物理的に出せない**
- 動けない分を**僧帽筋上部が「上に引き上げる」戦略で代償**

> イメージ：テーブルの上に置いた皿（肩甲骨）を滑らせたいのに、テーブル自体が傾いて固まっている（胸郭が硬い）。皿は滑れないので、上から持ち上げるしかない（僧帽筋上部が頑張る）。

---

## 3. 肩甲骨の重要3運動

| 運動 | 読み | 動きのイメージ |
|------|------|--------------|
| **上方回旋** | じょうほうかいせん | 腕を上げる時に肩甲骨が上向きに回る |
| **後傾** | こうけい | 肩甲骨の上端が後ろに倒れ、肩の屋根の下のスペースを確保 |
| **外旋** | がいせん | 肩甲骨が外向きに回り、内側縁の浮きを抑える |

- 肩の挙上での詰まり感・痛みに関与し得る所見

---

## 4. 肩甲上腕リズム（Scapulohumeral Rhythm）

### 正常比率

- 腕を180°挙上する際、**上腕骨120°：肩甲骨60°** の比率（2:1）で連動
- ただし全域で一定ではなく、区間により大きく変動する点に注意

### なぜ2:1が崩れるとマズいのか

肩甲骨が適切なタイミングで動いてくれないと、腕の骨の頭が屋根（肩峰）にぶつかる

> たとえ：クレーン車
> - **腕の骨** ＝ クレーンのアーム
> - **肩甲骨** ＝ クレーンの土台（台座）
> 
> アームだけを高く上げようとしても限界がある。土台（肩甲骨）が絶妙なタイミングで傾いてくれるから、アームは屋根にぶつからずに高く上がれる。

### 屈曲と外転での違い

**外転（横から上げる）**
- 肩の隙間（肩峰下）が最も狭くなりやすい動き
- 60°付近で肩が耳に近づくようにすくむ → インナーマッスル（棘上筋等）の機能不全＋僧帽筋上部の過活動の典型

**屈曲（前から上げる）**
- 骨の構造上、隙間には少し余裕がある
- 前から上げて肩がすくむ場合 → 肩甲骨を下に安定させる筋（前鋸筋・僧帽筋下部）の弱化で**肩甲骨が全体的に上に浮き上がっている**ことが多い

### セルフチェック

鏡の前で「横から上げる」「前から上げる」を交互に試す：
- **横から上げる時だけ極端にすくむ** → 肩峰下インピンジメントの可能性
- **どちらでもすくむ** → 肩甲骨周囲筋のバランスが全体的に崩れている可能性（猫背・巻き肩の影響が強い）

### 注意点

- 「2:1が崩れた＝異常」の断定は回避（区間で比率変動するため）

---

## 5. 腱板（ローテーターカフ）の機能解剖

### 構成筋と役割

| 筋肉 | 主な作用 |
|------|---------|
| **棘上筋** | 肩の外転（腕を横に上げる）の初動 |
| **棘下筋** | 外旋（腕を外にひねる） |
| **小円筋** | 外旋の補助 |
| **肩甲下筋** | 内旋（腕を内にひねる） |

- 共通の役割：上腕骨頭を関節窩に引き寄せる（**求心化**）＋各方向の回旋
- **弱化パターン**：棘上筋が弱いと挙上開始時に上腕骨頭が上方偏位 → インピンジメント誘発

---

## 6. 肩甲骨安定筋群

| 筋肉 | 役割 | 弱化/過緊張時 |
|------|------|-------------|
| **前鋸筋** | 肩甲骨を胸郭に密着、上方回旋 | 弱化 → 翼状肩甲 |
| **僧帽筋下部** | 肩甲骨の下制・上方回旋 | 弱化 → 肩甲骨挙上優位（すくみ） |
| **菱形筋** | 肩甲骨の内転・下方回旋 | 過緊張 → 肩甲骨の外転制限 |

---

## 7. 広背筋の解剖学的構造と機能

> 広背筋（こうはいきん）は背中の大きな筋肉で、体幹と腕をつなぐ"橋渡し"。ここが硬くなると、肩甲骨の動き・体幹の回旋・呼吸すべてに制限がかかる。

### 解剖学的構造

- **起始** ＝ 胸腰筋膜（背骨〜骨盤後面を覆う強い膜）、下位胸椎・腰椎棘突起、腸骨稜
- **停止** ＝ 上腕骨小結節稜（腕の骨の前内側）
- **特徴** ＝ 胸腰筋膜を介して体幹・骨盤と強固に連結

### 機能（作用）

| 作用 | 動きのイメージ |
|------|--------------|
| 肩関節伸展 | 腕を後ろに引く |
| 肩関節内転 | 腕を体側に引き寄せる |
| 肩関節内旋 | 腕を内側にひねる |
| 体幹―上肢の力伝達 | 体幹と腕の橋渡し |

---

## 8. 関節包の硬さと可動域制限

| 硬い部位 | 制限される動き | 臨床的意義 |
|---------|--------------|-----------|
| **前方関節包** | 伸展・外旋 | — |
| **後方関節包** | 屈曲・内旋 | GIRD（投球障害肩で重要） |
| **下方関節包** | 挙上 | 凍結肩で顕著 |

# 【B】肩こりのメカニズム

## 9. 凝っている場所の正体

### 「首と肩の間のあの凝り」＝ ほぼこの2つの筋

| 筋肉 | 位置 | 役割 | なぜ凝るか |
|------|------|------|-----------|
| **僧帽筋上部** | 首〜肩の表層 | 肩をすくめる・頭を支える | 頭の重さを支え続けて疲労 |
| **肩甲挙筋** | 肩甲骨〜首の深層寄り | 肩甲骨を引き上げる | 肩甲骨の動き不足で過剰に代償 |

- この2つが**常にONの状態** ＝ いわゆる「肩こり」
- 凝りの正体は「筋肉が硬くなった」ではなく**「働きすぎて休めていない」**
- 筋肉そのものの問題というより**脳からの指令（神経制御）の問題**

---

## 10. なぜ「働きすぎ」が起きるのか（5つの原因）

### ① 頭が重い（構造上の宿命）

- 人間の頭は約4〜6kg（ボウリング球1個分）、重心が**前寄り**
- 頭が前に出ると後ろから引っ張って支える必要がある → その役が僧帽筋上部と肩甲挙筋

> イメージ：釣り竿の先に重りがついた状態。竿を立てていれば楽だが、前に傾けた途端、手元にものすごい負担がかかる。

### ② 目を使うと頭が固定される

- スマホ・PC・子どもの見守り → すべて「目線の固定」
- 目線固定 → 頭固定 → 首〜肩の筋肉が「動かす役」から**「固める役」に切り替わる**
- 固める役として最も動員されやすいのが僧帽筋上部

### ③ 呼吸が浅くなる

- 忙しい・緊張・集中 → 呼吸が浅く速くなる
- 本来は**肋骨を広げて行う**呼吸（横隔膜＋肋間筋）が、浅い呼吸では首・肩の筋が**呼吸補助に動員**
- 1日に約2万回分の余計な仕事
- 広背筋が付着する**胸郭後下部も動かなくなる** → 広背筋も呼吸補助筋として過緊張に加担

> 「肩で息をしている」は比喩ではなく実際の現象。

### ④ 肩甲骨を動かす場面が日常にほぼない

- 日常動作の大半は「肘から先」で完結（タイピング、スマホ、料理、抱っこ等）
- 脳は使わない動きを忘れていく → **肩甲骨＝固定物**という神経パターンが定着
- 肩甲骨が動かなければ、帳尻を合わせるのは僧帽筋上部

### ⑤ 体は「安定」を最優先する

- 神経系の原則：**不安定より安定、動かすより固める**
- 疲れ・慣れない動き・緊張 → 胸郭と肩甲骨を**固める方向を選ぶ**
- 固める役 ＝ 僧帽筋上部・肩甲挙筋

---

## 11. 揉んでも戻る理由

### 凝りは「結果」であって「原因」ではない

凝っている場所を揉む・針を打つ → 一時的に筋緊張は下がる

しかし以下が変わっていなければ**数時間〜翌日に元通り**：
- 頭の位置（前に出た姿勢）
- 視線の使い方
- 呼吸パターン
- 肩甲骨の使われ方
- 胸郭（背骨＋肋骨）の動き

> 例え：雑草を刈っても根を取らないとまた生える。凝りを揉むのは「刈った雑草がまた伸びるのを待つ」ようなもの。

# 【C】姿勢パターンと肩への影響

## 12. 猫背単体（胸椎後弯）

### 骨・関節の起こりやすい配置

- 胸椎後弯増大
- 肩甲骨が前傾・外転位になりやすい
- 前鋸筋の長さ-張力関係が不利 → 上方回旋・後傾が出にくい

### 起こり得る影響（傾向）

- 肩の最大可動域の低下傾向（特に"だらんと丸めた姿勢"）
- ただし**胸椎後弯そのものと肩痛の差は「有意差なし」**の中等度エビデンス報告あり

### 指導時の伝え方

- 「背中が丸い＝肩が必ず痛くなる」ではない
- **「腕を上げるための"土台"が動きにくくなるので、肩だけで無理をしやすい条件」**という理解

---

## 13. 巻き肩単体（肩甲骨外転・内旋優位）

### 骨・関節の起こりやすい配置

- 肩甲骨外転＋内旋＋前傾
- 上腕骨内旋傾向、上腕骨頭前方偏位になりやすい条件
- 小胸筋・大胸筋の短縮

### 起こり得る影響（傾向）

- **肩峰下スペースが不利** → 腱板や滑液包にストレス増加
- 肩甲骨の運動異常（scapular dyskinesis）は肩障害の重要論点（ただし因果の単純化は不可）

### 補足：肩峰下で何が起きるか

- **肩峰下スペース**（肩の屋根＝肩峰と上腕骨の間の隙間）が狭くなる
- **挟まるもの**：
  - 腱板（棘上筋等の腱の束：肩を安定させる役割）
  - 滑液包（関節の間の"クッション袋"：組織同士の摩擦を軽減する、滑液が入った袋状の組織。肩・肘・膝など動く部位に多い）
- **結果**：腕を上げる時に腱や滑液包が骨に挟まれる（インピンジメント）→ 痛みや炎症が起きやすい条件

---

## 14. 巻き肩と広背筋過緊張の併発

### それぞれの肩甲骨への牽引方向

| 状態 | 肩甲骨への影響 |
|------|--------------|
| **巻き肩** | 前傾＋外転。上腕骨：内旋位 |
| **広背筋過緊張** | 上腕骨を内旋・伸展方向へ牽引。肩甲骨を下制方向へ牽引 |

### 併発時の肩甲骨位置

- **前傾＋外転＋下制** ＝ 肩甲骨が肋骨前面に貼りつくような固定状態
- → 肩甲骨の後傾・上方回旋が出にくくなる

### 構造的問題

- 肩甲上腕リズムの破綻
- 肩前面の詰まり感
- 首・僧帽筋上部の過活動（"いつもの代償者"がまた働く）

---

## 15. 一般生活で広背筋が張りやすい要因

| 要因 | メカニズム |
|------|----------|
| **長時間座位・猫背** | 広背筋が短縮位で固定 → 血流低下 |
| **片側優位動作の反復**（マウス、片手荷物、片側抱っこ） | 左右差のある緊張 → 体幹回旋のブレーキ役として常時緊張 |
| **体幹回旋・側屈の欠如** | 斜走線維が使われず滑走性低下 |
| **浅い呼吸** | 胸郭後下部が動かず呼吸補助筋として過緊張 |

---

## 16. 姿勢と痛みの関係（定説寄りの整理）

- 姿勢（猫背等）と肩痛の単純な因果は**弱い/限定的**という整理が妥当
- 一方で姿勢を起こした方が**肩可動域が増えやすい傾向は強い**
- → 「姿勢が悪い＝痛くなる」ではなく「**動きの選択肢が減る条件**」として指導

# 【D】肩の主要病態の整理

## 17. 四十肩・五十肩の正体

- **俗称**：四十肩・五十肩（年代で呼ぶ呼称）
- **医学的に混在しやすい病態**：
  - 凍結肩（癒着性肩関節包炎）
  - 腱板関連痛（腱板腱障害〜断裂）
  - 石灰沈着性腱炎
  - 上腕二頭筋長頭腱関連痛
  - 関節唇損傷（SLAP損傷等）

---

## 18. 凍結肩（癒着性肩関節包炎）

### 定義・特徴

- 関節包が肥厚・拘縮（硬く縮む）→ 痛みと可動域制限が進行
- 典型経過：**痛い時期 → 固い時期 → 戻る時期**（数か月〜年単位）

### なりやすい人（傾向）

- 40歳以上、とくに女性傾向
- 肩の長期不動（術後・骨折後等）
- 代謝・内分泌要因（糖尿病、甲状腺疾患等）

### 代表症状

- 夜間痛（夜に痛む）
- 全方向での可動域制限（特に外旋：肘を体側で外に回す動き）

### 症状緩和（一般論）

- 急性期は痛みを増悪させない範囲での可動維持、負荷調整
- ステロイド注射は短期で症状軽減の可能性報告あり

### 予防（一般論）

- 不動期間の短縮、関節の"固まり"を作らない生活管理
- 既往や糖代謝リスクがある場合の早期相談

---

## 19. 腱板（ローテーターカフ）関連痛

### 定義・特徴

- 腱板の使い痛み〜部分損傷〜断裂
- **腱板断裂は無症状でも一定割合で存在し、加齢で増える傾向**
- 画像所見（切れている/変性）＝痛みの強さ、ではないケースが一定数存在

### なりやすい人（傾向）

- 加齢（年代で断裂保有率が上昇）
- 高BMI、喫煙、利き腕などとの関連のメタ解析報告あり

### 代表症状

- 挙上時痛（腕を上げると痛む）
- 筋力低下（とくに外旋・挙上）
- 夜間痛（出る場合）

### 症状緩和（一般論）

- 過負荷局面（反復オーバーヘッド、痛みが強い角度帯）回避
- リハビリ（運動療法）中心の保存的アプローチが主要論点

### 予防（一般論）

- 肩甲帯の協調、腱板の持久力・筋力維持
- 反復負荷の量・頻度・回復（睡眠/休息）管理

---

## 20. 肩峰下痛／インピンジメント関連

### 定義・特徴

- 腕を上げる動作で、肩峰下の組織に負担が集中しやすい状態
- "骨を削る手術（肩峰下除圧）"は推奨が弱いガイドライン提示あり

### なりやすい人（傾向）

- 反復オーバーヘッド作業/競技
- 肩甲骨運動の乱れが併存するケース

### 代表症状

- 特定角度で痛む（例：横から上げる途中、60〜120°の Painful Arc）
- 動作開始痛、動作後の残痛

### 症状緩和・予防（一般論）

- 痛み誘発角度帯の負荷調整
- 肩甲骨の上方回旋・後傾を出す運動学習

---

## 21. 石灰沈着性腱炎

### 定義・特徴

- 腱板内の石灰沈着により痛み、とくに急性期は強い痛み
- 無症状の場合も一定割合あり

### 代表症状

- 突然の強い痛み、夜間痛、動かせない

### 症状緩和・予防（一般論）

- 急性期は炎症鎮静優先、無理な強制運動回避
- 予防の確立手段は**不明**（沈着機序が多因子）

---

## 22. 上腕二頭筋長頭腱関連痛

### 定義・特徴

- 肩の前側痛の原因となり得る腱
- 腱板損傷やSLAPと併存しやすい

### なりやすい人（傾向）

- 反復の引く/持ち上げる/オーバーヘッド動作
- 腱板病変の併存例

### 症状・緩和・予防（一般論）

- 肩前面痛、押すと痛い、抵抗運動で痛い
- 負荷調整＋肩甲帯/腱板の協調改善

---

## 23. 関節唇損傷（SLAP損傷）

### 定義・特徴

- 関節唇（受け皿の縁のクッション）の上方部が損傷
- オーバーヘッド競技でリスクが高い

### なりやすい人（傾向）

- 投球・サーブなど反復オーバーヘッド
- 後方の硬さ（GIRD：内旋可動域の不足）を伴う投球者でリスク上昇

### 代表症状

- 肩の奥の痛み、引っかかり、クリック感（クリック単独で診断不可）

### 症状緩和・予防（一般論）

- まず保存療法（3〜6か月）推奨
- 後方タイトネスの是正、肩甲骨運動の再教育が柱（投球者文脈）

---

# 【E】肩甲骨運動異常と評価

## 24. 一般成人・高齢者で頻出のエラー

### エラー分類（見え方ベース）

- **すくみ優位** ＝ 肩が耳に近づく・首肩が緊張
- **前巻き優位** ＝ 巻き肩：肩が前へ落ちる・胸が閉じる
- **翼状肩甲** ＝ 肩甲骨の内側が浮く
- **上げにくさ優位** ＝ 肩より上で詰まる・重い・引っかかる

### 理由カタログ（原因単独化の回避）

- **負荷要因**（使いすぎ・急増・休み不足）
- **可動性要因**（胸郭の硬さ、肩関節の動く余地の不足）
- **協調要因**（筋のタイミング不全、力みで別の筋が代償）
- **痛み要因**（痛みによる動作回避・運動制御変化）

### 重要な前提

- 肩甲骨の"異常な動き" ＝ 診断名ではなく**「機能障害（impairment）」**として扱う
- 原因にも結果にもなり得る

---

## 25. 評価時の注意点

### 代償動作の見極め

- 肩甲骨の早期挙上（僧帽筋上部優位）
- 体幹の側屈（反対側に倒れて挙上角度を稼ぐ）
- 腰椎の過伸展（後ろに反って挙上）

### 疼痛誘発角度（Painful Arc）

- 60〜120°で痛む → 肩峰下インピンジメント疑い
- 終末域で痛む → 関節包・関節唇の問題疑い

---

## 26. 評価（スクリーニング）最小セット

### 観察タスク（機材不要）

- 腕上げ（前・横）での左右差、すくみ、内側縁/下角の浮き
- 反復（10回程度）で崩れるか（疲労耐性の指標）
- 痛みの再現条件の言語化（高さ・回数・時間・翌日残り）

### 解釈ルール（誤診回避）

- 肩甲骨運動異常 ＝ "見えた＝原因"ではない
- ただし肩障害と併存し、評価・介入設計の手がかりになる

---

# 【F】回旋制限と連鎖

## 27. 広背筋過緊張が回旋に与える影響

### 体幹回旋の制動

- 広背筋は回旋に対して**ブレーキ側**として働く
- **過緊張時** → 胸郭と骨盤の相対回旋が制限
- **緊張低下時** → 胸郭─骨盤の分離運動が可能 → ひねる動きが軽くなる

### 胸腰筋膜の滑走性

- 広背筋は胸腰筋膜を強く牽引
- **硬化時** → 筋膜の層間滑走が低下
- **緩和時** → 体幹回旋・歩行時の腕振り・振り向く動作が少ない力で成立

### 肩甲帯と体幹の協調

- **過緊張時** → 肩甲骨が下制・内転方向に固定 → 代償動作増加
- **緩和時** → 肩甲骨の可動余地が拡大 → **体幹主導 → 上肢追従**の運動連鎖が成立

### 日常動作レベルの影響

| 状態 | 動作への影響 |
|------|-------------|
| 広背筋が硬い | 体がひねりにくい、振り向きづらい、腕が重い |
| 広背筋が緩む | 歩行が軽い、振り向きが速い、腕が自然に出る |

---

## 28. 回旋動作を制限するその他の構造

> 広背筋だけが原因ではない。

| 構造 | 役割 | 過緊張/硬化時の弊害 |
|------|------|-------------------|
| **腹斜筋群** | 体幹回旋の主動作筋 | 出力筋が制動筋化 → ひねり初動が重い、腰椎単独回旋の増加 |
| **脊柱起立筋群** | 体幹伸展・姿勢保持 | 胸椎回旋低下 → 回旋が腰椎に集中 → 腰部ストレス増大 |
| **腸腰筋** | 股関節屈曲・体幹安定 | 骨盤前傾固定 → 骨盤と胸郭の相対回旋減少 |
| **胸郭自体** | 回旋と呼吸の土台 | 肋骨が動かない → 呼吸と回旋が分離、動作が硬く疲れやすい |

---

# 【G】対策・介入フレーム

## 29. 解決の方向性

### 根本方針

**「凝っている場所を緩める」のではなく「凝らなくて済む条件を作る」**

| やること | 狙い |
|---------|------|
| **胸郭を動かす**（背骨を反る・肋骨を広げる） | 肩甲骨が動ける「土台」を作る |
| **肩甲骨の支持筋を再教育**（下部僧帽筋・前鋸筋・広背筋） | 僧帽筋上部の「代わり」を育てる |
| **呼吸を整える**（肋骨を使った呼吸） | 僧帽筋上部・広背筋を呼吸補助から解放 |
| **広背筋の緊張を下げる** | 肩甲骨の可動余地を広げ、回旋制限を解除 |

### 経験則：運動介入で改善しやすいパターン

> 以下は個人的な経験に基づく観察であり、すべての人に当てはまるとは限らない。

- 針やマッサージで一時的に楽になるが**すぐ戻る** → 胸郭・肩甲骨の運動介入で改善しやすかった
- 胸郭回旋（ワイパー系ストレッチ）で**即時的に凝りが軽くなる** → 胸郭の可動性制限が主因の可能性
- ロウやラットプルを**肩をすくめずに**行えるようになると、日常の凝りが減りやすかった
- 自重だけでも効果あり。**高負荷より「正しい場所が働いている感覚」が大事**

---

## 30. 介入の優先順位（機能解剖的視点）

1. **疼痛管理**：炎症期は負荷調整優先
2. **可動域改善**：関節包・筋の柔軟性確保（ただし強制は禁忌）
3. **肩甲骨運動の再教育**：前鋸筋・僧帽筋下部の活性化
4. **腱板の筋力強化**：求心化能力の向上
5. **動作パターンの修正**：代償動作の除去

---

## 31. 介入フレーム①〜④（実務ステップ）

### ① 負荷と症状の交通整理（最優先）

**目的**：痛みの上積み停止、悪化ループ遮断

**やること**
- 痛みを作る条件の特定（高さ・回数・連続時間・翌日残り）
- 量・頻度・角度の調整（連続作業の分割、肩より上の作業の一時削減等）
- てこの工夫（物を身体に近づける、片手偏重の回避等）

**目安**：痛み0–10で3以下、翌日に増悪持ち越しなし

### ② 動きの下地づくり（胸郭・肩の通り道）

**目的**：肩甲骨が滑る土台（胸郭）の確保

**やること**
- 呼吸で胸郭が動く感覚の回復（息止め回避）
- 猫背固定の解除（胸が起きる方向づけ）

### ③ 肩甲骨の協調再学習（タイミングと方向）

**目的**：すくみ代償の減少、上方回旋・後傾・外旋方向の協調

**言葉の指示**
- 「首を長く」（肩すくみ抑制）
- 「寄せすぎ注意」（胸張り固定の回避）
- "内側が浮くなら量を減らし質優先"ルール

### ④ 日常動作へ統合

**目的**：生活で再現できる「楽にできる肩」への移行

**やること**
- 低い位置 → 肩の高さ → 頭上の順の段階復帰
- 片手 → 両手、短時間 → 分割、軽い → 少し重いの順序設計

**合図**：「物は身体に近く」「息を止めない」「連続を分ける」

---

## 32. 器具なしエクササイズ（優先度順）

**共通ルール：「首と肩の間が力まない」ことが最優先。力んだ時点で逆効果。**

### ① 四つ這いロウ（肩甲骨の支持再学習）

- 四つ這いから片手を軽く浮かせ、肘を後ろに引く意識
- 実際に大きく動かなくてもOK
- 合言葉：**「首は長く、肩はすくめない」**
- 左右 各5〜8回 × 2〜3セット（1回3〜5秒キープ）

### ② 壁プル（ラットプル動作の神経学習）

- 壁に向かって立ち、両手を壁につけて「上から引き下ろす」意識
- 壁は動かないので等尺性収縮
- 合言葉：**「肘で引く、肩で引かない」**
- 5秒 × 5〜8回

### ③ うつ伏せYホールド（下部僧帽筋の活性化）

- うつ伏せで親指を天井に向け、Y字に腕を軽く浮かせる
- 高く上げない。首が反らない
- 5〜10秒 × 5回

### ④ 壁スライド（肩甲骨を肋骨に沿わせる制御）

- 壁に前腕をつけ、肩をすくめずに上へスライド
- 合言葉：**「肩甲骨が肋骨に吸い付く感じ」**
- 6〜10回 × 2セット

### ⑤ 胸郭回旋＋呼吸（仕上げ）

- 仰向けまたは四つ這いで体をひねり、ひねった側の背中に息を入れる
- 肋骨の広がりを感じられればOK
- 左右 各5呼吸

### よくある失敗

| 失敗パターン | なぜダメか |
|------------|-----------|
| 「効かせよう」として力む | 僧帽筋上部がまた出てしまう |
| 肩を下げすぎる | 別の代償が出る |
| 胸を張りすぎる | 腰が反って胸郭が動いていない |
| 呼吸を止める | 呼吸補助筋がまた動員される |

---

## 33. 悪循環と好循環のモデル

### 悪循環（多くの人が陥っているパターン）

```
頭が前に出る → 視線固定 → 呼吸浅化 → 胸郭が動かなくなる
→ 広背筋も短縮・過緊張 → 肩甲骨が動けなくなる
→ 僧帽筋上部が常時ON → 凝り形成
→ 凝りを揉む → 一時的に楽 → 条件が変わっていないので再燃
```

### 好循環（介入の方向性）

```
胸郭を動かす → 肋骨が広がる → 呼吸が変わる
→ 広背筋の緊張低下 → 肩甲骨が動ける → 支持筋が働く
→ 僧帽筋上部が休める → 凝りの「上流」が改善 → 再燃しにくい
```

| テーマ | 整理 |
|--------|------|
| **姿勢と痛み** | 単純な因果は弱い/限定的。姿勢を起こすと可動域は出やすい |
| **腱板断裂** | 無症状でも存在し加齢で増える。画像所見≠痛みの強さ |
| **介入の基本線** | 多くの肩痛はまず保存療法（負荷調整＋運動療法）。肩峰下除圧術は推奨が弱い |

## 35. ママさん・産後に特に起きやすい理由

育児期は肩こりの5つの原因（セクション10）が**同時に揃いやすい**

| 原因 | 育児場面 |
|------|---------|
| 頭が前に出る | 授乳、おむつ替え、添い寝 |
| 視線の固定 | 子どもの見守り、スマホ（片手抱っこ中） |
| 浅い呼吸 | 睡眠不足、常時の緊張・気疲れ |
| 肩甲骨が動かない | 抱っこ紐、前かがみ姿勢の持続 |
| 体が「固める」を選ぶ | 疲労、慣れない動作の連続 |

### 産後特有の要因

- **腹筋群（腹横筋・骨盤底筋）の機能低下** → 体幹安定性低下 → 肩まわりで安定を取ろうとする
- **抱っこ紐による肩甲帯の固定** → 肩甲骨が長時間「動けない」環境
- **片側抱っこの反復** → 広背筋の左右差・体幹回旋制限を助長

---

## 36. 受診すべきサイン（レッドフラッグ）

以下がある場合はセルフケアの前に医療機関（整形外科）受診を優先：

- **しびれ**（腕、手、指）
- **夜間痛**（寝ていて痛みで目が覚める）
- **安静時痛**（何もしていなくても痛い）
- **明確な筋力低下**（物を落とす、握れない）
- **外傷後の急性痛・変形**
- **急な翼状肩甲の出現・進行**（神経要因の除外必要）
- **産後の強い腹部・骨盤の違和感**（腹直筋離開・骨盤底機能の評価が必要）

---

## 37. 一般の人に伝える説明テンプレ

### 30秒版

- 肩甲骨 ＝ 腕を動かす土台
- 土台が遅れると、肩の関節が無理して詰まりやすい
- 形を固定するのでなく、痛みを増やさず動きと耐久力を戻す方針

### 誤解の修正（断定回避）

- 「肩甲骨を寄せれば正しい」一辺倒の回避（固めて悪化の可能性）
- 「痛み＝壊れている」断定の回避（許容量超過の可能性、赤旗は別扱い）
- 「2:1が崩れた＝異常」断定の回避（区間で比率変動）

## 38. 用語集

| 用語 | 説明 |
|------|------|
| **肩甲骨** | 背中側の平たい骨、腕の土台 |
| **上方回旋** | 腕を上げる時に肩甲骨が上向きに回る動き |
| **後傾** | 肩甲骨が後ろに倒れて"肩の屋根の下"を確保する動き |
| **腱板** | 肩の関節を安定させる腱の束（棘上筋・棘下筋・小円筋・肩甲下筋） |
| **関節包** | 関節を包む袋。硬くなると動かしにくい |
| **滑液包** | 関節の間のクッション袋。摩擦を軽減 |
| **肩峰下スペース** | 肩の屋根（肩峰）と腕の骨の間の隙間。狭くなると腱が挟まれやすい |
| **インピンジメント** | 挟み込み。腕を上げる時に腱や袋が骨に挟まれて痛む状態 |
| **SLAP** | 肩の受け皿の縁（関節唇）上方部の損傷タイプ |
| **GIRD** | 投球側の肩の内旋可動域が減少した状態 |
| **広背筋** | 背中の大きな筋肉。体幹と腕をつなぐ橋渡し |
| **胸腰筋膜** | 背骨〜骨盤を覆う強い膜。広背筋が付着 |
| **前鋸筋** | 肩甲骨を肋骨に密着させ上方回旋を担う筋 |
| **僧帽筋上部** | 首〜肩の表層。肩をすくめる・頭を支える |
| **僧帽筋下部** | 肩甲骨の下制・上方回旋を担う。弱化すると「すくみ」の原因 |
