# 呼吸と機能解剖

## ポイント
- とにかく吐かせる、ゆっくり吐き切る（副交感神経）
  - **リラックスが主目的**「細く長く〜」
- 膝を曲げる理由
  - 腰の反りを防ぐ
- 呼吸時に見るポイント
  - 肋骨の動き（内旋・外旋）
  - 顎の向き
    - 上がってる人、苦しそうな人は、枕とかを入れてあげる

## 肋骨の動きと横隔膜の関係
### 前提の定義（用語）

- **肋骨の外旋**：肋骨が背骨との関節で回り、肋骨弓が外へ開く方向（胸郭が横に広がりやすい方向）
- **肋骨の内旋**：肋骨弓が内へ閉じる方向（胸郭がしぼみやすい方向）

### 吸気・呼気の「内旋/外旋」＋「見た目（広がる/収縮）」整理

| 呼吸相 | 肋骨の回旋 | 肋骨の上下 | 胸郭の見た目 | 横隔膜（主役筋） | 典型キーワード |
|:------|:----------|:----------|:------------|:---------------|:--------------|
| **吸気（息を吸う）** | 外旋（外へ開く） | 挙上（上がる） | 広がる（前後・左右に拡張）<br>＝「胸が起きる/肋骨が開く」<br>・前後径↑：ポンプハンドル<br>・横径↑：バケツハンドル | 収縮→ドーム下降（下がる）<br>＝胸腔容積↑ | 「外へ開く」<br>「上・前・外へ」 |
| **呼気（息を吐く）<br>（安静）** | 内旋（内へ閉じる） | 下制（下がる） | しぼむ/収縮（胸郭が戻る）<br>＝「肋骨が閉じる」 | 弛緩→ドーム上昇（戻る）<br>＝胸腔容積↓ | 「内へ閉じる」<br>「下・後・内へ」 |
| **呼気（強く吐く）<br>（努力呼気）** | 内旋＋下制の強調 | 下制の強調 | 収縮の強調（腹部の締め出し） | 横隔膜は基本弛緩側 | 内肋間筋（内側の肋間筋群）などの呼気活動 |

### 見た目チェック（現場用・短縮）

- **吸気で「外旋＋広がる」**：下部肋骨が外へ広がる、胸郭が前後・左右に拡張
- **呼気で「内旋＋しぼむ」**：下部肋骨が内へ戻る、胸郭が収縮

### 補足（誤解の予防）

- **内旋/外旋の対象**：肩・股関節ではなく、ここでは肋骨（胸郭）の回旋の意味
- **「胸が上がる＝良い吸気」ではない**：外旋・拡張が出ても、首肩の補助呼吸（斜角筋など）優位の可能性は残存（評価は課題依存）


---

- 腹式呼吸と横隔膜
  - 腹式呼吸は、この横隔膜の上下運動を意識的に行う呼吸法。お腹が膨らむのは、横隔膜が下に押し下げられ、内臓が腹部へ移動するため。
  - 腹式呼吸は、ストレス軽減、内臓機能の活性化、姿勢改善、冷え性改善など、多くのメリットがあるとされている
- 日常の呼吸と横隔膜
  - 私たちは1日に約2万～2万5千回も呼吸をしており、そのほとんどを横隔膜が担っている。
  - 現代人は胸の上部だけを使う浅い呼吸（胸式呼吸）になりがちだが、意識的な横隔膜呼吸（腹式呼吸）を取り入れることで、より深い呼吸とリラックス効果が得られる。
- トレーニング・ほぐし方（例）
  - リラックスして行う:仰向けに寝て膝を立て、お腹に手を当て、お腹と胸が一緒に膨らむようにゆっくり呼吸する。
  - 横隔膜のマッサージ:肋骨の下あたりを指で軽く押しながら、深呼吸を繰り返す。
  - 息を吸い込む時間よりも、ゆっくり長く吐き切ることを意識

### 1. メカニズムの整理（専門用語・動き）

腹式呼吸は、主に**横隔膜の上下運動**によって行われます。肋骨（胸郭）の動きを最小限にし、横隔膜を大きく動かすのが特徴です。

| 呼吸のフェーズ | 横隔膜（主役） | 肋骨（脇役） | お腹の状態 |
| :--- | :--- | :--- | :--- |
| **吸う時** (吸気) | **収縮**して**下がる** | あまり動かさない<br>（上がらないようにする） | 臓器が押されて**膨らむ** |
| **吐く時** (呼気) | **弛緩**して**上がる** | 内旋して自然に**下がる**<br>（閉じる） | 元に戻る（**凹む**） |

#### 動きのポイント
*   **吸気（吸う）：** 横隔膜という筋肉が**収縮（緊張）**すると、平らになって**下がります**。これにより胸腔（肺が入っている部屋）が縦に広がり、空気が入ってきます。下がった横隔膜が内臓を下に押すため、お腹が膨らみます。
*   **呼気（吐く）：** 横隔膜が**弛緩（リラックス）**すると、ドーム状に戻りながら**上がります**。これにより肺が下から押されて空気が外に出ます。

---

### 2. 一般向けに口頭で伝えるポイント

一般の方に「収縮・弛緩」と言ってもイメージしづらいため、**「風船」や「注射器」**などの例え話や、**「肩・お腹」**といった触れやすい部位を使って伝えます。

#### ① 「横隔膜」の動きを伝える（注射器のイメージ）
横隔膜が下がることで空気が入る仕組みを説明します。

> **伝え方の例：**
> 「肺の下には『横隔膜』という膜があります。これが**注射器のピストンのように下がる**と、空気がスゥーッと吸い込まれます。
> 逆に、力が抜けて**膜がフワッと上がる**と、空気が押し出されて息が吐けます。」

#### ② 「お腹」と「肋骨」の関係を伝える（風船のイメージ）
なぜお腹が膨らむのか、肋骨（胸）はどうすべきかを説明します。

> **伝え方の例：**
> 「息を吸う時、**胸や肩は上げないように**リラックスしてください。
> その代わり、下がってきた横隔膜に押されて、**お腹が風船のように膨らむ**イメージです。
> 吐く時は、その風船がしぼんでいくのを感じましょう。」

#### ③ 指導時の具体的なキューイング（声かけ）

*   **吸う時：**
    *   「肩の力は抜いて、おへその下が膨らむように」
*   **吐く時：**
    *   「お腹を背中にくっつけるように凹ませて」
    *   「おへそが床にくっつくイメージ」

### まとめ
*   **専門的理解：** 吸う時は横隔膜が**収縮して下がり**、吐く時は**弛緩して上がる**。
*   **一般への伝達：** 「お腹の風船が膨らむ」とイメージさせ、「胸や肩（肋骨）は動かさない」ことを強調

## 呼吸における肋骨と横隔膜の動き

### 横隔膜

- 胸腔と腹腔を隔てるドーム状の筋肉
- 安静時は上に凸のドーム形
- 中心部は腱（中心腱）、周囲が筋肉
- 付着部：下位肋骨の内側、胸骨、腰椎

### 肋骨

- 12対、脊椎と胸骨をつなぐ
- 上位肋骨（1〜6番）：胸骨に直接付着
- 中位肋骨（7〜10番）：軟骨を介して胸骨に付着
- 下位肋骨（11〜12番）：浮遊肋骨（前方は遊離）

---

## 吸気（息を吸う）時の動き

### 横隔膜

| 項目 | 状態 |
|------|------|
| **筋肉の状態** | 収縮 |
| **形状の変化** | ドームが平坦化（下に下がる） |
| **胸腔への影響** | 胸腔が縦方向に拡大 |
| **腹腔への影響** | 腹部臓器が押し下げられ、お腹が膨らむ |

```
安静時          吸気時
   ∩              ―
  / \     →     /   \
 胸腔           胸腔拡大
  腹腔           腹腔圧迫
```

---

### 肋骨

**関与する筋肉：** 外肋間筋（主）、斜角筋、胸鎖乳突筋（補助）

| 肋骨の部位 | 動きのパターン | イメージ |
|------------|----------------|----------|
| **上位肋骨** | ポンプハンドル運動 | 胸骨が前上方に持ち上がる |
| **下位肋骨** | バケツハンドル運動 | 肋骨が左右に広がる |

**ポンプハンドル運動（上位肋骨）：**
```
      ↑ 胸骨が上がる
     ／
    ／
   ○ 肋椎関節を軸に回転
```

**バケツハンドル運動（下位肋骨）：**
```
        広がる
    ←  ∩  →
       │
    肋椎関節を軸に
```

**結果：**
- 胸郭が前後・左右に拡大
- 胸腔内の陰圧が高まる
- 空気が肺に流入

---

## 呼気（息を吐く）時の動き

### 横隔膜

| 項目 | 状態 |
|------|------|
| **筋肉の状態** | 弛緩（リラックス） |
| **形状の変化** | ドーム状に戻る（上に上がる） |
| **胸腔への影響** | 胸腔が縮小 |

**重要：** 安静時の呼気は横隔膜の弛緩と肺・胸郭の弾性反跳によって受動的に起こる（筋力をほとんど使わない）

---

### 肋骨

| 項目 | 状態 |
|------|------|
| **動き** | 下降・内側に戻る |
| **安静呼気** | 受動的（弾性反跳） |
| **努力呼気** | 内肋間筋、腹筋群が収縮 |

**努力呼気時に働く筋肉：**
- 内肋間筋：肋骨を引き下げる
- 腹直筋・腹斜筋・腹横筋：腹腔内圧を高め、横隔膜を押し上げる

---

## 吸気と呼気の比較まとめ

| 項目 | 吸気 | 呼気（安静時） | 呼気（努力時） |
|------|------|----------------|----------------|
| **横隔膜** | 収縮→下降 | 弛緩→上昇 | 弛緩＋腹筋で押し上げ |
| **肋骨** | 挙上・拡大 | 下降・縮小（受動的） | 下降・縮小（能動的） |
| **外肋間筋** | 収縮 | 弛緩 | 弛緩 |
| **内肋間筋** | 弛緩 | 弛緩 | 収縮 |
| **腹筋群** | 弛緩 | 弛緩 | 収縮 |
| **胸腔容積** | 増大 | 減少 | 減少（より強力） |

## 呼吸パターンと筋肉の使い方

### 腹式呼吸（横隔膜呼吸）

- **主役：** 横隔膜
- **特徴：** 吸気時にお腹が膨らむ、肋骨の動きは少なめ
- **利点：** 効率的、リラックス効果、深い呼吸が可能
- **適した場面：** 安静時、瞑想、ストレス軽減

### 胸式呼吸

- **主役：** 肋間筋、補助呼吸筋
- **特徴：** 吸気時に胸が広がる、肩が上がりやすい
- **利点：** 素早い換気が可能。交感神経優位。
- **適した場面：** 運動時、緊急時

### 理想的な呼吸

両方がバランスよく協調して働く状態。横隔膜が主導しつつ、肋骨も適度に動く。

使い分けのイメージはこんな感じです。

| 呼吸法 | 主な効果 | おすすめの場面 |
| --- | --- | --- |
| **胸式呼吸** | **活性化・集中・安定** | 運動中、仕事中、朝起きた時 |
| **腹式呼吸** | **リラックス・回復** | 寝る前、休憩中、ストレス解消 |

## 呼吸の問題と姿勢の関連

### 姿勢が崩れると起こること

| 姿勢の問題 | 呼吸への影響 |
|------------|--------------|
| **猫背** | 肋骨の動きが制限される、胸式優位になりやすい |
| **反り腰** | 横隔膜が下がりにくい、浅い呼吸になりやすい |
| **肩の挙上** | 補助筋（僧帽筋、斜角筋）に頼った非効率な呼吸 |
| **肋骨の開き** | 横隔膜が適切に機能しない |

---

### 呼吸機能チェック

**横隔膜が使えているか：**
1. 仰向けに寝る
2. 片手をお腹、片手を胸に置く
3. 吸気時にお腹の手が先に・大きく持ち上がれば横隔膜優位

**肋骨が動いているか：**
1. 両手を肋骨の横に当てる
2. 吸気時に手が左右に押し広げられる感覚があるか確認

## 補足：運動時の呼吸

### ランニング中の呼吸

- 横隔膜と肋間筋の両方がフル稼働
- 換気量を増やすため、肋骨の動きも大きくなる
- 呼吸筋も疲労する（長距離後半のパフォーマンス低下の一因）

### 呼吸筋のトレーニング

- 横隔膜の意識的なトレーニング（腹式呼吸の練習）
- 肋骨の可動性を保つストレッチ
- 体幹トレーニングで呼吸筋の持久力向上

## 横隔膜と骨盤底筋の解剖学的関係

### 基本構造：体幹の「圧力容器」

横隔膜と骨盤底筋は、腹腔を上下から挟む「蓋」と「底」の関係にあります。

```
      ────────────
     （  横隔膜  ）  ← 上の蓋
      ────────────
           │
         腹腔
      （内臓・腹圧）
           │
      ────────────
     （ 骨盤底筋 ）  ← 下の底
      ────────────
```

**構成要素：**

| 部位 | 役割 |
|------|------|
| **横隔膜** | 上蓋、呼吸の主動筋 |
| **骨盤底筋群** | 下底、内臓の支持・排泄制御 |
| **腹横筋** | 前後の壁（コルセット） |
| **多裂筋** | 後壁、脊柱の安定 |

この4つが協調して**腹腔内圧（IAP: Intra-Abdominal Pressure）**を調整しています。

## 姿勢による変化

### 脚を伸ばした仰臥位

**骨盤・腰椎の状態：**
- 腸腰筋が引っ張られ、骨盤が前傾しやすい
- 腰椎の前弯（反り）が増強
- 腰が床から浮く

**横隔膜への影響：**

```
        横隔膜
       ／    ＼
      ／      ＼  ← 前方が引き上げられる
     ／        ＼
    腰椎の過伸展で後方付着部が引っ張られる
```

- 横隔膜の後方付着部（腰椎部）が過伸展した腰椎に引っ張られる
- 横隔膜のドームが浅くなる（平坦化した状態に近づく）
- 収縮しても下降幅が小さくなる

**骨盤底筋への影響：**

- 骨盤前傾により骨盤底筋が前下方に傾く
- 横隔膜との対向関係がずれる
- 腹圧のベクトルが骨盤底筋に直接かからない

**両者の距離・位置関係：**

| 項目 | 状態 |
|------|------|
| **垂直距離** | 見かけ上は変わらないが、軸がずれる |
| **対向関係** | 平行でなくなる（ハの字に近づく） |
| **圧力伝達** | 非効率、前方に逃げやすい |

```
脚伸展時（横から見た図）：

    横隔膜 ──＼
              ＼
               │  ← 腹圧のベクトルがずれる
              ／
    骨盤底筋 ──／
           ↙
        圧力が前下方へ逃げる
```

---

### 膝を曲げた仰臥位

**骨盤・腰椎の状態：**
- 腸腰筋が緩み、骨盤がニュートラルに近づく
- 腰椎の過前弯が軽減
- 腰が床に近づく

**横隔膜への影響：**

- 腰椎部の付着部への過度な牽引がなくなる
- 横隔膜が自然なドーム形状を保てる
- 収縮時の下降幅が確保される

**骨盤底筋への影響：**

- 骨盤底筋が水平に近い位置になる
- 横隔膜と平行に対向する
- 腹圧を効率よく受け止められる

**両者の距離・位置関係：**

| 項目 | 状態 |
|------|------|
| **垂直距離** | 最適化される |
| **対向関係** | ほぼ平行（ピストンの上下関係） |
| **圧力伝達** | 効率的、軸がそろう |

```
膝屈曲時（横から見た図）：

    横隔膜 ────────
              │
              │  ← 腹圧がまっすぐ伝わる
              │
    骨盤底筋 ────────
              ↓
        圧力が骨盤底筋全体で受け止められる
```

## 呼吸効率への影響

### 横隔膜の可動域

| 姿勢 | 横隔膜の下降幅 | 呼吸効率 |
|------|----------------|----------|
| **脚伸展** | 制限される（1〜2cm程度減少の可能性） | 低下 |
| **膝屈曲** | 正常範囲（安静時3〜5cm、深呼吸時7〜10cm） | 最適化 |

**なぜ可動域が変わるか：**

1. **付着部の問題**：腰椎過伸展で横隔膜の脚部（腰椎付着部）が引っ張られ、収縮の起点が不安定になる
2. **腹腔の形状**：反り腰では腹腔が前方に開いた形になり、横隔膜が下降しても腹圧が効率的に高まらない
3. **拮抗筋の問題**：腹筋群が伸張されて機能しにくく、呼気の補助ができない

---

### 腹腔内圧の生成効率

**効率的な腹圧生成の条件：**

```
    横隔膜（下降）
        ↓
    ┌───────────┐
    │           │ ← 腹横筋（締める）
    │   腹腔    │
    │           │
    └───────────┘
        ↑
    骨盤底筋（支える）
```

この「圧力容器」が機能するには、横隔膜と骨盤底筋が**同軸上で対向**している必要があります。

| 姿勢 | 圧力容器の状態 | 腹圧生成 |
|------|----------------|----------|
| **脚伸展** | 傾いた容器、蓋と底がずれている | 非効率、圧が逃げる |
| **膝屈曲** | まっすぐな容器、蓋と底が対向 | 効率的、圧が保持される |

---

### 横隔膜と骨盤底筋の連動

**正常な呼吸での連動パターン：**

| 呼吸相 | 横隔膜 | 骨盤底筋 | 腹圧 |
|--------|--------|----------|------|
| **吸気** | 収縮・下降 | 伸張・下降（受動的） | 上昇 |
| **呼気** | 弛緩・上昇 | 収縮・上昇 | 低下 |

この連動は「ピストン運動」と呼ばれます。

```
吸気時：
横隔膜  ↓↓↓（収縮して下がる）
腹腔内圧 ↑↑↑（高まる）
骨盤底筋 ↓（圧を受けて下がる）

呼気時：
横隔膜  ↑↑↑（弛緩して上がる）
腹腔内圧 ↓↓↓（低下する）
骨盤底筋 ↑（反動＋収縮で上がる）
```

**姿勢による連動への影響：**

| 姿勢 | 連動の質 |
|------|----------|
| **脚伸展・骨盤前傾** | 連動が乱れやすい、骨盤底筋が圧を受け止めにくい |
| **膝屈曲・骨盤ニュートラル** | 連動がスムーズ、ピストン運動が効率的 |

---

## 具体的な数値・距離の目安

### 横隔膜と骨盤底筋の距離

直接的な距離の測定データは限られていますが、解剖学的な目安として：

| 項目 | おおよその値 |
|------|--------------|
| **成人の腹腔の縦幅** | 約15〜20cm |
| **横隔膜のドーム頂点の高さ** | 第4〜5肋間（安静時） |
| **骨盤底筋の位置** | 恥骨結合〜尾骨のライン |

**姿勢による変化：**
- 骨盤前傾では骨盤底筋が前下方に傾き、実質的な「機能的距離」が変化
- 距離そのものより、**軸のアライメント**が重要

---

### 横隔膜の移動距離

| 呼吸の深さ | 横隔膜の移動距離 |
|------------|------------------|
| **安静呼吸** | 約1.5〜3cm |
| **深呼吸** | 約7〜10cm |
| **最大努力呼吸** | 最大12cm程度 |

姿勢不良（脚伸展・骨盤前傾）では、この移動距離が**10〜30%程度減少**する可能性があります。

## 呼吸効率の観点からのまとめ

### 脚伸展時の問題

```
腸腰筋の牽引
    ↓
骨盤前傾・腰椎過伸展
    ↓
┌─────────────────────────────────────┐
│ 横隔膜：後方付着部が引っ張られる    │
│ 　　　　ドームが浅くなる            │
│ 　　　　下降幅が減少                │
├─────────────────────────────────────┤
│ 骨盤底筋：前下方に傾く              │
│ 　　　　　横隔膜との軸がずれる      │
│ 　　　　　腹圧を受け止めにくい      │
├─────────────────────────────────────┤
│ 腹横筋：伸張されて機能低下          │
│ 　　　　腹圧を保持しにくい          │
└─────────────────────────────────────┘
    ↓
呼吸効率の低下
・換気量の減少
・横隔膜と骨盤底筋の連動不全
・代償的に胸式呼吸優位になりやすい
```

---

### 膝屈曲時の利点

```
腸腰筋の弛緩
    ↓
骨盤ニュートラル・腰椎適度な前弯
    ↓
┌─────────────────────────────────────┐
│ 横隔膜：自然なドーム形状            │
│ 　　　　最大限の下降幅を確保        │
├─────────────────────────────────────┤
│ 骨盤底筋：水平に近い位置            │
│ 　　　　　横隔膜と同軸上に対向      │
├─────────────────────────────────────┤
│ 腹横筋：適切な長さで機能            │
│ 　　　　腹圧をしっかり保持          │
└─────────────────────────────────────┘
    ↓
呼吸効率の最適化
・横隔膜の可動域最大化
・腹圧の効率的な生成と伝達
・横隔膜と骨盤底筋のピストン連動
・腹式呼吸がしやすい
```

### 臨床・実践への応用

| 対象者 | 膝屈曲位が特に重要な理由 |
|--------|--------------------------|
| **腰痛のある方** | 腰椎への負担軽減＋呼吸による体幹安定化 |
| **骨盤底機能障害** | 横隔膜との連動を再学習しやすい |
| **産後の方** | 骨盤底筋と腹筋の回復エクササイズの基本姿勢 |
| **高齢者** | 安全に深い呼吸を練習できる |
| **アスリート** | 体幹の圧力制御を正しく学ぶ土台 |
