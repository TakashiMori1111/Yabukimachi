# 腰を守る回旋戦略

## 回旋・骨盤前後傾・脊柱バイオメカニクスの知識整理

---

### 1. 回旋動作の基本構造

- 体幹回旋は全身の連動動作  
  - 股関節
  - 骨盤
  - 胸椎
  - 頸椎  
  それぞれ役割分担
- 腰椎の回旋は必要最小限
  - 過剰なねじれによる障害リスク

---

### 2. 回旋力伝達の構造

```txt
足部
↓
股関節
↓
骨盤
↓
腰椎
↓
胸椎
↓
頸椎・視線
```

| 部位    | 主な役割            | 回旋での意味           |
| ------ | ---------------- | ------------------- |
| 足部    | 地面を捉える        | 回旋力の起点            |
| 股関節  | 下肢-骨盤をつなぐ    | 回旋を作る・受ける・伝える |
| 骨盤    | 力の中継            | 股関節-体幹をつなぐ        |
| 腰椎    | 安定・微調整         | 過剰なねじれ回避         |
| 胸椎    | 体幹回旋軸           | 回旋の主役              |
| 頸椎    | 視線の調整           | 視線方向制御             |

---

### 3. 腰椎の役割

- 腰椎は回旋の主動部ではない
- 構造的に屈曲・伸展得意
- 回旋可動域は胸椎より小さい
- 過剰なねじれ集中による障害リスク
  - 特に下記組み合わせに注意
    - 前屈＋強回旋
    - 重い物保持回旋
    - 疲労状態での反復
    - 股関節・胸椎不動時の腰代償
    - 痛み我慢しながらの無理な可動域拡大

---

### 4. 回旋の主役：胸椎と股関節

#### 胸椎

- 体幹回旋の主軸
- 肋骨・胸郭・肩甲骨・頸椎とも連動
- 胸椎硬化による代償
  - 腰過剰回旋
  - 首のみ回旋
  - 肩代償
  - スイング・投球時負担集中
  - デスクワークでの緊張

#### 股関節

- 股関節回旋と骨盤・体幹回旋の連動
- 内旋・外旋の可動域と個人差
- 股関節不動による代償
  - 腰のみ回旋
  - 膝逸脱
  - 骨盤片側流れ
  - 歩幅減少、切り返し動作での腰膝負担

---

### 5. 骨盤前傾・後傾の機能的意義

| 骨盤状態   | 起こりやすい見た目  | 評価ポイント                   |
| ---------- | ---------------- | --------------------------- |
| 前傾傾向   | 反り腰傾向         | 後傾へ戻れるか、腰過伸展の有無     |
| 後傾傾向   | 丸腰傾向           | 前傾へ動けるか、股関節可動の有無    |
| 中間域     | 前後に動ける姿勢     | 呼吸・股関節可動性            |

- 骨盤前傾・後傾は善悪ではなく、動きの切り替え能力に注目
- 固定より状況に応じた可動が重要

---

### 6. 骨盤前後傾と筋肉・要因

#### 前傾関連筋

| 筋         | 主な作用                   |
| ---------- | ------------------------ |
| 腸腰筋      | 股関節屈曲、腰椎制御           |
| 大腿直筋    | 股関節屈曲、膝伸展             |
| 脊柱起立筋  | 脊柱伸展                   |

#### 後傾関連筋

| 筋            | 主な作用                                   |
| ------------- | -------------------------------------- |
| 腹直筋・腹斜筋 | 体幹屈曲・骨盤後傾制御                         |
| 大殿筋         | 股関節伸展・骨盤制御                            |
| ハムストリングス | 膝屈曲・股関節伸展・骨盤後傾張力                  |

- 骨格形状、可動域、筋力、呼吸、競技歴、心理的要因など多因子で決定

---

### 7. 骨盤位置・呼吸・腹圧

| 方向    | 主な構造              |
| ------- | ------------------- |
| 上      | 横隔膜                  |
| 下      | 骨盤底筋                |
| 前・横   | 腹横筋、腹斜筋            |
| 後      | 多裂筋、胸腰筋膜           |

- 骨盤の極端な位置固定による呼吸・腹圧の調整困難リスク
- 呼吸と骨盤底筋の連動性
- 固定より、出力・緩み・戻しの切り替え能力重視

---

### 8. 脊柱カーブ・痛み判断

- 頸椎前弯、胸椎後弯、腰椎前弯の生理的カーブ意義
- カーブ形状のみで痛み判断不可
- 戻れる柔軟性、痛み・しびれ出現有無、呼吸停止の有無、股関節胸椎可動、生活上の困難、負荷耐性を重視

---

### 9. 椎間板内圧・腰負荷

- 負荷上昇の主な場面
  - 前屈
  - 重量物保持
  - 前屈＋回旋
  - 筋緊張維持
  - 長時間同一姿勢
  - 疲労状態で反復
- 問題は準備・筋力・可動性・疲労・痛み・負荷量の過不足
- 動作禁止より分担・負荷調整重視

---

### 10. 画像所見・痛みの非一致

- 画像異常のみで原因断定不可
- 無症状でも画像変化存在
- 判定は画像＋症状・神経所見・動作・負荷歴・生活背景の多面的評価
- 指導：現在困っている動作・悪化しやすい負荷・安全領域確認重視

---

### 11. 多裂筋・分節制御

- 多裂筋：背骨深層の支持筋
- 腰部分節制御の中心
- 慢性腰痛での筋質・脂肪浸潤例多いが因果非単純
  - 痛みによる不動
  - 機能低下による影響
  - 加齢・活動量ほか複合影響
- 優先は「画像を変えること」でなく、「機能的動作範囲拡大と局所過負荷回避」

---

### 12. 評価フロー

#### 1. 赤信号確認事項

- 強い痛み
- しびれ
- 筋力低下
- 排尿・排便異常
- 会陰部感覚異常
- 夜間痛
- 安静時痛増悪
- 発熱
- 原因不明体重減少
- 外傷後悪化
- 進行性麻痺

#### 2. 胸椎回旋評価

- 座位・骨盤固定で胸回旋  
  - 左右差・痛み・呼吸ストップ・腰同時動作を確認

#### 3. 股関節内外旋評価

- 座位or仰臥位での股関節内外旋  
  - 左右差・詰まり感・痛み・膝/腰代償確認

#### 4. 骨盤前後傾評価

- 座位or仰臥位で骨盤ロール  
  - 前傾・後傾可否、腰のみ運動・呼吸停止の有無

#### 5. 動作評価

- 歩行
- しゃがみ
- ヒップヒンジ
- 片脚立ち
- 体幹回旋
- 投球・スイング・キック
- 切り返し動作  
  単体可動域正常でも、実動作での代償確認

---

### 13. 介入優先順位

#### 優先1. 痛み・神経症状評価

- 医療評価が最優先

#### 優先2. 呼吸と骨盤前後傾の改善

- ペルビックティルトによる小可動域切り替え  
  - 仰向け・膝立ち
  - 息吐いて後傾→吸って中間
  - 腹筋や殿筋を固めすぎない
  - 5〜10回反復

#### 優先3. 胸椎伸展・回旋向上

- 四つ這い胸椎回旋  
  - お尻やや後方  
  - 片手頭後ろ  
  - 肘天井への胸開き  
  - 呼吸止めず左右5回

#### 優先4. 股関節回旋改善

- 90/90ポジション  
  - 前後脚90度  
  - 背筋不要な過伸展避け  
  - 内外旋を感じる  
  - 痛み/詰まりに応じ角度調整  
  - 左右30-60秒

#### 優先5. 抗回旋・運動統合

- 回旋されすぎを抑える  
  - デッドバグ
  - バードドッグ
  - パロフプレス
  - ヒップヒンジ
  - ハーフニーリング回旋
  - スプリットスタンス回旋  
  - 目的は「腰安定・股関節と胸椎の運動協調」

---

### 14. 姿勢タイプ別アプローチ

#### 反り腰傾向

- 腰椎伸展・肋骨前上がり・股関節進展の腰代償・腹圧求めて腹筋固定
- アプローチ
  - 息を吐いて肋骨下げ
  - 骨盤後傾練習
  - 殿筋主導の股関節伸展
  - 胸椎伸展強調

#### 猫背・後傾傾向

- 胸椎丸まり固定・骨盤後傾座位・股関節伸展困難・頭部前方化
- アプローチ
  - 胸椎伸展・回旋
  - 骨盤前傾練習
  - 股関節ヒンジ
  - 高さ・座位環境調整

#### スウェイバック傾向

- 骨盤前方移動・股関節前面緊張・骨盤後傾気味・胸椎後弯・頭部前突・ハム/腹部緊張
- アプローチ
  - 足裏支持感覚
  - 股関節後方ヒップヒンジ
  - 骨盤前後傾切替練習
  - 胸郭の骨盤上復帰

---

### 15. 実践ミニマムセット

#### デスクワーク

1. 30分以内ごとの姿勢変更
2. 骨盤前後5回運動
3. 胸椎左右小回旋
4. 1時間に1回立つ
5. 画面目線の高さ調整

#### 運動前

1. 痛み・しびれ確認
2. 骨盤前後傾確認
3. 股関節内外旋確認
4. 胸椎回旋確認
5. 軽抗回旋種目実施

#### 回旋系スポーツ前

1. 90/90で股関節回旋確認
2. 四つ這い胸椎回旋
3. ハーフニーリング回旋
4. パロフプレス
5. 低強度の競技動作開始

---

### 16. よくある誤解の整理

- 腰が硬い＝腰をもっとねじるべき  
  → 腰椎は回旋の主動部でなく、胸椎・股関節回旋を優先
- 骨盤前傾＝悪  
  → 前傾自体は善悪でなく、前傾固定や戻れないのが問題
- 骨盤後傾＝悪  
  → 後傾自体は善悪でなく、後傾固定や前傾不可が問題
- 画像所見変性＝痛み原因  
  → 画像のみで判断せず症状・動作・負荷歴・生活背景とセットで解釈
- 良い姿勢＝理想カーブ固定  
  → 形固定でなく、呼吸・骨盤前後傾・胸椎回旋・股関節回旋の切替を重視

---

### 17. 統合的判断プロセス

1. 赤信号の有無確認
2. 腰椎の過剰回旋有無
3. 胸椎・股関節回旋の可動確認
4. 骨盤前傾・後傾の切替能力
5. 脊柱カーブの固定有無
6. 画像＋症状＋動作＋負荷歴の総合評価
7. 呼吸・骨盤前後傾・胸椎回旋・股関節回旋・抗回旋から小規模介入

---

### 最終まとめ

- 腰回旋の守り方は「腰固定」「腰過剰回旋」ではなく、股関節・胸椎主動、腰椎は必要最小限で安定・微調整
- 全身連動と適切な分担・切り替えの習得が腰を守る鍵
