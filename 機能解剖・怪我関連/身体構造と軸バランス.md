# 身体構造と軸_バランス

［確立］前回の「荷重軸・安定軸・力の通り道」に、**支持基底面（Base of Support：接地している身体部分で囲まれた面）**を加えると、かなり実用的になります。

整理の中心はこれです。

```text
支持基底面：力を受ける土台
重心：身体全体の重さの中心
重心線：重心から床へ下ろした線
COP：足裏の圧の中心
床反力：地面から返ってくる力
荷重軸：その力が体内を通る線
安定軸：崩れないように支える構造
推進軸：次の動きへ力を出す方向
```

つまり、動作を見る順番は、

```text
① どこで地面に触れているか
↓
② 重心が支持基底面のどこにあるか
↓
③ 足裏の圧中心がどこへ動いているか
↓
④ 床反力がどの方向へ返っているか
↓
⑤ その力が足・膝・股関節・体幹へどう通っているか
↓
⑥ 次の支持基底面をどれだけ速く作れるか
```

この流れで見ると、
**バランス・安定・力発揮・痛みの偏り・パフォーマンス低下**を同じ地図で説明できます。


【伝え方】
体の安定は、足を広げれば終わりではない。
足をどこにつくか、その上に重心がどこにあるか、地面から返ってきた力をどこへ流すかで決まる。
強い動きは「止まれる」「崩れない」「すぐ動き出せる」の3つがそろっている状態。

---

# 1. 全体フレーム：BOSを入れた人体の力学地図

## 1-1. 5つの基本概念

【専門層】

| 用語          | 意味               | 現場での見方           |
| ----------- | ---------------- | ---------------- |
| 支持基底面 / BOS | 接地している身体部分で囲まれた面 | 足幅、手幅、片脚支持、四つ這い  |
| 重心 / COM    | 身体全体の重さの中心       | 骨盤〜下腹部付近にイメージ    |
| 重心線 / LOG   | 重心から床へ垂直に下ろした線   | BOS内にあるか、外へ出ているか |
| COP         | 足裏にかかる圧の中心       | 踵寄り、つま先寄り、内側、外側  |
| 床反力 / GRF   | 地面から身体へ返る力       | どの方向へ押し返されるか     |

※
- BOS: Base of Support（支持基底面）  
- COM: Center of Mass（重心）  
- LOG: Line of Gravity（重心線）  
- COP: Center of Pressure（圧中心）  
- GRF: Ground Reaction Force（床反力）

［確立］静止時の安定は、主に以下で高まる。

1. 支持基底面が広い
2. 重心が低い
3. 重心線が支持基底面内にある
4. 摩擦が十分
5. 感覚入力が正確
6. 筋出力と反射制御が間に合う

ただし、スポーツでは「倒れないこと」だけが正解ではありません。
動き出すには、あえて重心を支持基底面の端または外へ移動させ、次の足で新しい支持基底面を作る必要があります。

# 2. 重要な修正点

## 2-1. 「支持基底面が広い ＝ 常に良い」ではない

【専門層】

［確立］支持基底面が広いほど静的安定性は上がりやすい。
ただし、広すぎると以下が起きます。

* 重心移動が遅くなる
* 股関節・足関節の反応が鈍る
* 一歩目が出にくくなる
* 横方向には強いが、前後方向の切り替えが遅くなる
* 筋出力の方向と床反力がズレる

つまり、競技動作では
**安定しすぎる姿勢は、動き出しにくい姿勢**でもあります。

【伝え方】

足を広げると倒れにくくなる。
でも広げすぎると、次の一歩が遅くなる。
スポーツでは「安定」と「動きやすさ」のちょうどいい幅が必要。

---

## 2-1-1. パワーポジション目安

パワーポジションは、すぐに動き出せる中間姿勢です。  
主な特徴と目安は以下の通りです。

### 基本姿勢チェックリスト

- **足幅**：肩幅〜肩幅よりやや広め
- **つま先**：正面〜やや外向き
- **股関節**：軽く引く
- **膝**：軽く曲げる
- **重心**：母趾球〜足裏中央
- **体幹**：少し前傾
- **背骨**：丸めすぎず反りすぎず
- **目線**：前
- **腕**：すぐ使える位置

> スポーツで言えば、止まる・跳ぶ・走り出す・切り返す前の準備姿勢です。


### 大体の角度目安

| 部位       | 目安                | 意図                                  |
| ---------- | ------------------- | ------------------------------------- |
| 足幅       | 肩幅〜1.5倍未満     | 横にも前にも動ける幅                  |
| 足首       | すねが少し前傾      | 地面を押しやすい                      |
| 膝         | 20〜45°くらい屈曲   | 低すぎず反応しやすい                  |
| 股関節     | 30〜60°くらい屈曲   | お尻・ハムを使える                    |
| 体幹前傾   | 10〜30°くらい       | 前後左右に動きやすい                  |
| 重心       | 足裏中央〜やや前    | すぐ加速できる                        |

> ※ 角度は競技・体格・場面で調整


### 良いパワーポジションの条件

1. **股関節が使える**
    - 膝だけ曲げず、股関節を軽く折る。
    - **NG:** 膝だけ前に出る
    - **OK:** お尻を少し後ろに引き、股関節に乗る
    - 股関節が使えると、臀部・ハムストリングス・内転筋が働きやすくなる（ジャンプ・スプリント・切り返しの土台）。

2. **足裏全体で地面を感じる**
    - 基本イメージは「母趾球・小趾球・かかと」の3点で支える。
    - 静止・構え：足裏中央  
      加速直前：やや前  
      ジャンプ：足裏全体  
      切り返し：外足（外側）で支える
    - ※ 競技によって重心はベタ足〜やや前寄りで変動

3. **膝とつま先の向きが一致する**
    - 膝が内側に入ると力が逃げやすく、膝ストレス増。
    - **OK:** 膝とつま先が同じ方向  
      **NG:** つま先は外、膝は内
    - 特に女子選手・成長期選手の着地・切り返しで要確認

4. **腰を反らせて固めない**
    - 「胸を張る」の意識過剰は腰反りにつながる。
    - **OK:** 肋骨と骨盤が向かい合う  
      **NG:** 肋骨が開く、腰が反る
    - 腹圧を軽く保ち、腰ではなく股関節と体幹で支える。


### シーン別パワーポジション目安

| 場面                 | 姿勢の特徴                                             |
| ------------------ | -------------------------------------------------- |
| バレーのレシーブ       | やや低め、股関節・膝ともに曲げる                         |
| バスケのディフェンス    | 横移動しやすく、足幅やや広め                              |
| サッカーの守備対応     | 高すぎず低すぎず、片脚に寄りすぎない                        |
| スプリントの構え        | 重心やや前、股関節から前傾                                |
| ジャンプ準備           | 足裏全体、股関節に乗る                                    |
| 切り返し前             | 外足に乗れる幅、膝とつま先を一致                           |


### 現場で使えるチェックポイント

#### OK感覚

- その場から前後左右にすぐ動ける
- 軽くジャンプできる
- 足裏で地面を押せる
- 太もも前だけでなく、お尻にも少し張りを感じる
- 腰が詰まらない
- 膝が内側に落ちない

#### NG感覚

- 太もも前だけがきつい
- かかとが浮きすぎる
- 腰が反る
- 背中が丸まりすぎる
- 膝が内側に入る
- 低すぎて一歩目が遅い
- 高すぎて地面を押せない


### 合言葉・指導フレーズ

- 「足裏で床をつかむ」
- 「お尻を少し後ろ」
- 「膝とつま先を同じ向き」
- 「腰じゃなく股関節に乗る」
- 「低くなりすぎない」

特に便利なのは、「その姿勢から前にも横にもジャンプできるか？」
できなければ、「低すぎ・膝だけ曲げ・重心ズレ」などを疑う

## 2-2. 「重心が中央 ＝ 常に良い」でもない

【専門層】

［確立］静止して耐えるなら、重心線はBOS中央付近が有利。
しかし、加速・切り返し・ジャンプでは、重心線をBOSの端へ寄せる、または一瞬外へ出すことで動き出しが生まれる。

| 状況      | 重心線の理想           |
| ------- | ---------------- |
| 静止・待機   | BOS中央〜やや前方       |
| コンタクト耐性 | 押される方向に応じて中央〜反対側 |
| 加速      | 進行方向へ重心を傾ける      |
| 切り返し    | 支持脚の外側〜次の方向へ移動   |
| 着地      | BOS内へ素早く回収       |
| フェイント   | BOSの端を使って相手をずらす  |

※「動き出したい方向の逆に重心を外す」という表現はやや粗い。
正確には、**床を押す方向と、身体が進む方向は反対になる**ということ。
身体そのものの重心は、最終的には進みたい方向へ移る。

【伝え方】
止まるときは重心を真ん中に置く。
動き出すときは、真ん中から少し外す。
強い選手は、崩れそうで崩れないギリギリを使って動いている。

# 3. BOS × 荷重軸 × 力発揮の統合モデル

## 3-1. 力発揮しやすい構造とは

【専門層】

［確立］力発揮しやすい姿勢は、単に「安定している姿勢」ではありません。
以下がそろっている状態です。

| 要素     | 意味             | 例             |
| ------ | -------------- | ------------- |
| 適切なBOS | 足幅・手幅が目的に合う    | パワーポジション      |
| COM位置  | 重心が次の動きに適した位置  | やや前方、低め       |
| COP操作  | 足裏の圧を動かせる      | 踵→母趾球、外側→内側   |
| GRF方向  | 床反力を使える        | 地面を押して返りを受ける  |
| 関節配列   | 力が逃げない位置関係     | 足・膝・股関節の整列    |
| 筋剛性    | 必要な瞬間だけ硬くできる能力 | 着地、切り返し、スプリント |
| 感覚入力   | 足裏・関節位置の把握     | バランス、反射制御     |

### 力発揮の基本式イメージ

```text
接地面の作り方
×
重心位置
×
床反力の方向
×
関節の並び
×
筋出力タイミング
=
安定した力発揮
```

【伝え方】

強く動くには、筋肉だけでは足りない。
足をどこにつくか、体重をどこに置くか、地面をどの方向に押すかが大事。
筋力は、いい土台の上で初めて使いやすくなる。

# 4. パワーポジションの再整理

## 4-1. 定義

【専門層】

［有力］パワーポジションとは、
**外部刺激に耐えながら、前後左右へ素早く動き出せる準備姿勢**。

目的は「ただ低く構えること」ではなく、
**BOS・COM・COP・GRF・股関節・体幹を最適化すること**。

---

## 4-2. 作り方

| 要素  | 目安           | 理由            |
| --- | ------------ | ------------- |
| 足幅  | 肩幅〜肩幅よりやや広め  | 左右安定と一歩目のバランス |
| 足圧  | 踵・母趾球・小趾球の三点 | 足部トライポッド      |
| 重心  | BOS中央〜やや前方   | 後ろ重心を避け、即動ける  |
| 股関節 | 軽度屈曲         | 重心を下げ、臀部を使う   |
| 膝   | つま先方向へ軽く曲げる  | 膝外反を避ける       |
| 足関節 | 軽度背屈         | 床反力を受けやすい     |
| 体幹  | 肋骨と骨盤を重ねる    | 腰椎反りすぎ防止      |
| 視線  | 水平           | 頸部・体幹の過緊張防止   |

## 4-3. 典型的なエラー

| エラー      | 起こる問題         | 修正           |
| -------- | ------------- | ------------ |
| 足幅が狭すぎる  | 横に弱い          | 肩幅程度へ        |
| 足幅が広すぎる  | 一歩目が遅い        | 少し狭める        |
| 踵重心      | 後方へ崩れる        | 足圧を中央〜前方へ    |
| つま先重心すぎ  | 大腿四頭筋過多、膝前面負荷 | 踵も残す         |
| 膝が内側へ入る  | 膝外反ストレス       | 足部三点＋股関節外旋制御 |
| 腰を反る     | 腰椎伸展ストレス      | 肋骨を下げる       |
| 背中を丸めすぎる | 股関節出力低下       | 胸郭を保つ        |

【伝え方】

パワーポジションは、低くしゃがむことではない。
足裏三点で床をつかみ、股関節に少し乗って、どの方向にも動ける姿勢。
止まれるけど、すぐ動けるのが理想。

# 5. 部位別：BOSと荷重軸の関係

## 5-1. 足部：BOSの入口

【専門層】

［確立］足部は、BOSの中で最も重要な接地センサー。
踵骨・第1中足骨頭・第5中足骨頭による**足部トライポッド（三点支持）**が、BOSの質を作る。

| 接地点 | 役割        |
| --- | --------- |
| 踵   | 後方支持、初期接地 |
| 母趾球 | 推進、蹴り出し   |
| 小趾球 | 外側安定、切り返し |
| 足趾  | 微調整、前方制御  |

### 現場で見ること

* 足裏三点に圧があるか
* 土踏まずが潰れすぎていないか
* 小指側へ逃げすぎていないか
* 母趾球で押せるか
* 片脚立ちで足趾が過剰に握っていないか

【伝え方】

足裏は、かかと・親指の付け根・小指の付け根の三点で支える。
この三点が使えると、膝や股関節も安定しやすい。
足裏が崩れると、上の関節もズレやすい。

---

## 5-2. 足関節：BOSと重心移動の調整役

【専門層】

［確立］足関節は、BOS内で重心線を調整する主要関節。
背屈（つま先を上げる方向の動き）が不足すると、重心を前へ運びにくくなり、膝・股関節・腰で代償しやすい。

| 動作    | 足関節の役割            |
| ----- | ----------------- |
| スクワット | 膝を前方へ許容し、重心を中央に保つ |
| 着地    | 衝撃吸収              |
| 切り返し  | 外側支持と再加速          |
| 歩行    | 踵接地から前足部への移行      |
| スプリント | 短時間接地と反発          |

【伝え方】

足首が動かないと、重心をうまく前に運べない。
その分、膝や腰で無理をしやすい。
足首は、体重を前後に動かすハンドルのようなもの。

---

## 5-3. 膝：BOSと股関節の間の中継点

【専門層】

［有力］膝の安定は、膝そのものだけでなく、BOSと股関節の関係で決まる。
特に片脚支持ではBOSが狭くなるため、股関節外転筋・足部感覚・体幹制御の影響が大きい。

### ニーインの整理

ニーイン（膝が内側へ入る動き）は、単なる膝のエラーではありません。

| 原因候補     | 何が起きるか            |
| -------- | ----------------- |
| 足部回内過多   | 土踏まずが潰れ、脛骨が内旋しやすい |
| 股関節外転筋低下 | 骨盤が落ち、膝が内側へ入りやすい  |
| BOSが狭い   | 片脚支持で安定余裕が少ない     |
| 体幹側屈     | 重心線が支持脚から外れる      |
| 恐怖・不安    | 安定限界が狭くなる         |

【伝え方】

膝が内に入るのは、膝だけの問題ではない。
足裏が潰れる、股関節で支えられない、体が横に倒れることで起きる。
だから膝を見るときは、足と股関節もセットで見る。

---

## 5-4. 股関節・骨盤：BOSの上で重心を運ぶ装置

【専門層】

［確立］股関節と骨盤は、BOSの上でCOMを制御する中心。
片脚支持ではBOSが足一つ分になるため、骨盤・股関節の制御が低下すると、重心線が支持基底面外へ逃げやすい。

| 機能     | 役割               |
| ------ | ---------------- |
| 股関節外転  | 骨盤が横に落ちるのを防ぐ     |
| 股関節伸展  | 地面を後方へ押し、前方推進を作る |
| 股関節内外旋 | 方向転換、回旋動作        |
| 骨盤制御   | COMの位置調整         |
| 体幹連動   | 上半身のブレを抑える       |

【伝え方】

片脚で立つと、足の上に骨盤を乗せる必要がある。
股関節が弱いと、骨盤が落ちて膝や腰が頑張らされる。
走る・跳ぶ・切り返す動きでは、股関節が重心を運ぶ。

---

## 5-5. 体幹・胸郭：BOSの上で上半身を積む構造

【専門層】

［確立］体幹は、BOSとCOMの関係を調整する上部構造。
肋骨と骨盤が大きくズレると、重心線がBOSから外れやすくなり、腰椎・股関節・膝へ代償が出る。

### 見るべきこと

* 肋骨が開きすぎていないか
* 骨盤が前傾/後傾しすぎていないか
* 胸椎が回旋できるか
* 腰椎で反っていないか
* 呼吸で腹圧が作れるか

【伝え方】

体幹は、足の上に上半身をうまく乗せるための部分。
肋骨と骨盤がズレると、腰や膝に余計な負担がかかる。
お腹の力は、体を固めるためではなく、力を通すために使う。

# 6. 静的安定と動的安定

## 6-1. 静的安定

【専門層】

静的安定は、止まった状態で姿勢を保つ能力。

| 評価       | 見ること         |
| -------- | ------------ |
| 両脚立ち     | 足裏三点、左右荷重差   |
| 片脚立ち     | 骨盤落下、足趾過緊張   |
| パワーポジション | COMがBOS内にあるか |
| 押圧テスト    | 外乱への耐性       |
| 閉眼立位     | 感覚入力への依存     |

### 問題になりやすい例

* 高齢者のワイドベース歩行
* 怪我後の片脚不安定
* 足裏感覚低下
* 足趾の過剰な握り
* 骨盤の横揺れ

【伝え方】

止まって安定できるかを見るのが静的安定。
まずは、足裏にちゃんと乗れているか、片脚で骨盤が落ちないかを確認する。

---

## 6-2. 動的安定

【専門層】

動的安定は、動きながら重心を制御し、必要なタイミングで次のBOSを作る能力。

| 評価        | 見ること         |
| --------- | ------------ |
| ステップダウン   | 膝・骨盤・足部の制御   |
| サイドホップ    | 横方向のBOS再構築   |
| ランジ       | 前後方向の重心移動    |
| スプリットステップ | 着地後の再加速      |
| カッティング    | BOSからの脱出と再構築 |

［確立］歩行・走行は、重心が現在のBOSから外れ、次の足で新しいBOSを作る連続運動。
そのため、スポーツでは「崩れない能力」だけでなく、**崩れを利用して次の姿勢を作る能力**が重要。

【伝え方】

スポーツの安定は、止まってフラつかないことだけではない。
動きながら崩れて、次の足で立て直す力が必要。
切り返しがうまい選手は、次の足場を作るのが速い。

---

# 7. 競技別・場面別のBOS活用

## 7-1. コンタクトスポーツ

対象。

* ラグビー
* 柔道
* 相撲
* サッカーの競り合い
* バスケのポストプレー

【専門層】

戦略は、**方向特異的なBOS拡大とCOM低下**。

| 要素            | 狙い          |
| ------------- | ----------- |
| 足幅を広げる        | 外乱に耐える      |
| 重心を低くする       | 倒されにくくする    |
| 押される方向へBOSを作る | 力を受ける       |
| 股関節で支える       | 膝・腰への集中を避ける |
| 体幹剛性を上げる      | 力を逃がさない     |

注意点。
常に重心をBOS中央に置くのではなく、相手の力の方向に対して、**倒されない範囲で重心線を調整する**こと。

【伝え方】

ぶつかる競技では、足を広げて低くなるだけでは足りない。
相手が押してくる方向に、足場を作る。
地面を押し返せる姿勢が強い姿勢。

---

## 7-2. スプリント・加速

【専門層】

戦略は、**BOSからCOMを前方へ移し、床反力を推進へ変えること**。

| 局面   | BOSの使い方            |
| ---- | ------------------ |
| スタート | 前方へ重心を傾ける          |
| 加速   | 接地点を体の後方寄りに置く      |
| 最高速度 | 接地時間を短くする          |
| 減速   | BOSを前方へ作り、ブレーキをかける |

### 見ること

* 接地が体の前すぎないか
* 骨盤が後ろに残っていないか
* 足部が潰れていないか
* 母趾球で押せるか
* 体幹が反っていないか

【伝え方】

速く動き出すには、体を前に運ぶ必要がある。
足は地面を後ろに押し、地面から前に進む力をもらう。
足だけ速く動かしても、重心が残ると加速しにくい。

---

## 7-3. 切り返し・アジリティ

【専門層】

戦略は、**減速用BOSを作り、次の方向へBOSを作り直すこと**。

| 局面   | 重要ポイント         |
| ---- | -------------- |
| 減速   | 足を進行方向前方〜外側へ置く |
| 支持   | 股関節でCOMを受ける    |
| 切り返し | 外側足部・腓骨側安定     |
| 再加速  | 母趾球へ圧を移す       |
| 体幹   | 次方向へ先行させる      |

### よくあるエラー

* 足だけ外へ出て、骨盤が残る
* 膝が内側に入る
* 上半身が起きすぎる
* 接地時間が長い
* 足部が潰れる

【伝え方】

切り返しは、足を出すだけではない。
まず止まる足場を作り、股関節で体重を受けて、次の方向へ押し返す。
足場を作り直すのが速い選手ほど、切り返しが速い。

---

## 7-4. 球技ディフェンス

【専門層】

戦略は、**横方向BOSの確保と、左右への素早い再構築**。

| 要素        | 意味              |
| --------- | --------------- |
| 横に広めのBOS  | 左右の揺さぶりへ対応      |
| 低めのCOM    | 反応距離を短くする       |
| 足圧の移動     | 左右への準備          |
| 股関節主導     | 膝だけで止まらない       |
| スプリットステップ | 一瞬BOSを広げ、次動作へ移行 |

【伝え方】

ディフェンスは、広く構えて終わりではない。
相手が動いた方向に、すぐ新しい足場を作る。
横に強く、でも固まりすぎない姿勢が必要。

---

## 7-5. ジャンプ・着地

【専門層】

戦略は、**BOS内へCOMを回収し、足・膝・股関節で衝撃を分散すること**。

| 局面    | 重要ポイント     |
| ----- | ---------- |
| 離地前   | 足裏三点、股関節伸展 |
| 空中    | COM制御      |
| 接地    | 足部〜股関節で吸収  |
| 安定    | 膝外反を防ぐ     |
| 再ジャンプ | 母趾球と股関節で出力 |

### 着地で見ること

* 足幅が狭すぎないか
* 膝が内側へ入らないか
* 音が大きすぎないか
* 体幹が前後左右に揺れないか
* 次の動きへ移れるか

【伝え方】

着地は、膝だけで受けるものではない。
足、膝、股関節、お腹で分けて受ける。
良い着地は、静かで、次の動きにすぐ移れる。

# 9. 最終統合図

```text
支持基底面 / BOS
= 地面との接点・土台
        ↓
重心 / COM
= 身体の重さの中心
        ↓
重心線 / LOG
= 重心がBOS内か外か
        ↓
COP
= 足裏の圧の中心
        ↓

= 地面から返ってくる力
        ↓

= 足・膝・股関節・体幹へ通る力
        ↓
安定軸
= 崩れないように支える構造
        ↓
推進軸
= 次の動きへ出力する方向
        ↓
動作の質
= 安定 / 力発揮 / 疲れにくさ / 痛み予防
```

# 最終まとめ

【専門層】

［確立］支持基底面は、バランスの出発点。
ただし、安定性はBOSの広さだけでは決まらない。
**BOS、COM、LOG、COP、床反力、関節配列、筋出力、感覚入力、反射制御**が統合されて決まる。

スポーツや日常動作で重要なのは、

1. **安定できること**
2. **力を出せること**
3. **すぐ次の支持基底面を作れること**

この3つ。

前回の「荷重軸・安定軸・力の通り道」にBOSを加えると、最終的にはこう整理できる。

```text
良い動き
=
足場を作る
+
重心を運ぶ
+
床反力を使う
+
関節で力を分散する
+
次の足場へ移る
```

【伝え方】

体の安定は、足を広げれば完成ではない。
足裏で地面を感じて、重心をちょうどいい場所に置き、地面から返ってくる力を足・股関節・体幹へ流す。
強い動きは、止まれるだけでなく、すぐ次の一歩へ移れる動き。
