# 結論

「機能性向上」や「日常生活を楽に」をトレーニングへ結びつける場合、現場では単なる「筋力アップ」ではなく、

* どの動作で困るのか
* どの方向へ力を出せないのか
* どの局面で不安定なのか
* どの速度で制御できないのか

まで分解することが重要。

特にパーソナルトレーナー視点では、

> 「筋肉を鍛える」
> ではなく、
> 「生活タスクを遂行できる身体能力を作る」

へ翻訳できるかが実用性の差になりやすい。

---

# パーソナルトレーナー視点の整理

## 現場で本当に見るべき5要素

| 要素   | 内容           | 例           |
| ---- | ------------ | ----------- |
| 力発揮  | 力を出せるか       | 立ち上がれない     |
| 力の制御 | ブレずに止められるか   | 階段下降で膝が崩れる  |
| 力の伝達 | 体幹〜四肢へ連動できるか | 荷物を持つと腰だけ痛い |
| 左右支持 | 片脚で支えられるか    | 歩行でふらつく     |
| 速度対応 | 素早く反応できるか    | つまずきで踏み直せない |

# 「生活動作」から逆算した分類

# 1. 立つ・座る・しゃがむ能力

## 日常での意味

* 椅子から立つ
* トイレ
* 車の乗り降り
* 床から立つ
* 階段開始

## 本質

単なる脚力ではなく、

* 重心移動
* 股関節伸展（股関節を伸ばす）
* 膝伸展（膝を伸ばす）
* 足圧制御

の統合。

## よくある代償

| 代償          | 背景                 |
| ----------- | ------------------ |
| 膝だけで立つ      | 股関節伸展不足            |
| 腰を反らせる      | 体幹固定不足             |
| 前ももばかり張る    | ヒンジ機能低下            |
| 手を使わないと立てない | 下肢筋力不足 or 前方重心移動不足 |

## 優先トレーニング

### 自重

* 椅子スクワット
* ボックススクワット
* テンポスクワット（ゆっくり）
* Sit to Stand
* TRXスクワット

### 負荷あり

* ゴブレットスクワット
* レッグプレス
* フロントスクワット
* ステップアップ

## 現場でのキュー（声かけ）

* 「床を押す」
* 「お尻で立つ」
* 「膝だけで頑張らない」
* 「頭を前に運んでから立つ」

---

# 2. 持ち上げる・拾う能力

## 日常での意味

* 洗濯物
* 子どもの抱っこ
* 買い物袋
* 床の荷物
* 掃除


# 本質

重要なのは「腰を曲げる」ではなく、**ヒップヒンジ**（股関節中心に折りたたむ能力）。

ここが弱いと、

* 腰椎屈曲（腰丸め）
* 膝だけ過剰使用
* 頸部緊張

へ代償しやすい。

## 優先トレーニング

### 自重

* ヒップヒンジ練習
* 壁タッチヒンジ
* グルートブリッジ
* 四つ這いロッキング

### 負荷あり

* RDL
* ケトルベルデッドリフト
* Trap Bar Deadlift
* Cable Pull Through

## 特に重要

### 「持ち上げる」より「下ろす」

日常では、

* 荷物を置く
* 子どもを下ろす
* 座る

など、減速制御（エキセントリック）が非常に多い。

---

# 3. 歩行・片脚支持能力

## 日常での意味

* 歩行
* 階段
* 段差
* 靴を履く
* 電車内
* 方向転換


# 本質

歩行は「片脚立ちの連続」。
特に重要なのは、**前額面制御**（横ブレ制御）


## 重要筋

| 筋     | 役割      |
| ----- | ------- |
| 中殿筋   | 骨盤安定    |
| 内転筋群  | 支持脚制御   |
| 足部内在筋 | 足圧安定    |
| 腹斜筋   | 体幹横ブレ抑制 |


## よくある問題

| 現象       | 背景        |
| -------- | --------- |
| 膝内側へ入る   | 股関節外転制御不足 |
| 骨盤が落ちる   | 中殿筋機能低下   |
| 足が不安定    | 足部戦略低下    |
| 歩行で疲れやすい | 片脚支持耐久不足  |


## 優先トレーニング

### 自重

* 片脚立ち
* タンデム歩行
* ステップダウン
* ラテラルステップ
* スプリットスクワット

### 負荷あり

* Bulgarian Split Squat
* Step Up
* Lateral Lunge
* Suitcase Carry

---

# 4. 押す能力

## 日常での意味

* ドア
* 台車
* 床から起きる
* 転倒時の支持
* 手をついて起きる

# 本質

胸より、**肩甲帯安定**の影響が大きい。

特に、

* 前鋸筋（肩甲骨を肋骨へ固定）
* 下部僧帽筋
* 体幹固定

が重要。


## 「腕立てできる＝実用的」とは限らない理由

高回数Push Upができても、

* 肩甲骨固定
* 体幹連動
* 荷重位制御

が弱いケースあり。


## 優先トレーニング

### 自重

* Wall Push Up
* Incline Push Up
* Bear Position Hold
* Quadruped Push

### 負荷あり

* Landmine Press
* Dumbbell Press
* Cable Press

---

# 5. 引く能力

## 日常での意味

* ドアを引く
* 手すり
* 荷物を寄せる
* 姿勢保持

# 本質

「引く」は腕ではなく、**肩甲骨制御**が主体。

## 特に重要

| 動き    | 必要     |
| ----- | ------ |
| 肩甲骨下制 | 首すくみ防止 |
| 肩甲骨内転 | 背部安定   |
| 胸郭安定  | 腰代償防止  |

## 自重ベース

* Table Row
* TRX Row
* Band Row
* Scapular Retraction

---

# 6. 運ぶ能力（キャリー）

## 実は最も生活転移しやすい

理由：

* 歩行
* 握力
* 体幹
* 呼吸
* 片脚支持
* 抗側屈（横倒れ防止）

を同時使用。


# キャリー系の価値

## ファーマーズキャリー

両側安定
→ 荷物運搬。


## スーツケースキャリー

片側負荷。
→ 横ブレ制御。
→ 歩行安定。


## フロントキャリー

抱っこに近い。
→ 体幹前面負荷。


# 自重で近づける方法

* バッグ保持歩行
* 水タンク運搬
* 子ども抱っこ歩行
* Bear Hug Carry

---

# 7. 回旋・抗回旋能力

## 日常で重要な理由

生活は「真っ直ぐ」だけではない。

* 振り返る
* 掃除
* 車操作
* 荷物を横へ置く

など、回旋制御が多い。

# 本質

重要なのは、**「捻る力」より、「捻られすぎない制御」**

## 優先トレーニング

### 自重

* Bear Crawl
* Dead Bug
* Bird Dog
* Side Plank

### 負荷あり

* Pallof Press
* Cable Chop
* Offset Carry

---

# 8. 「速く動ける」は高齢者ほど重要

## 筋力だけでは不十分

転倒回避では、
* 速く踏み直す
* 瞬間的に支える
* 一瞬で減速
が必要。

# パワー低下の影響

高齢では、

* 最大筋力より、
* Rate of Force Development（力を素早く立ち上げる能力）

低下が大きい報告あり。


# 現場向け実装

## 高齢者でも可能

### 自重

* 素早いSit to Stand
* 軽いステップ反応
* 小ジャンプ
* テンポ変更

### 負荷あり

* 軽負荷高速スクワット
* MB Throw
* KB Swing（適応条件あり）

# パーソナルトレーナーとしての実践フレーム

## 評価→処方

## ① 何が困るか

例：

* 階段
* 抱っこ
* 腰
* 疲れやすい


## ② どの能力不足か

| 問題   | 能力       |
| ---- | -------- |
| 階段   | 片脚支持＋減速  |
| 腰不安  | ヒンジ＋体幹   |
| 抱っこ  | キャリー＋抗伸展 |
| 転倒不安 | パワー＋片脚反応 |


## ③ 生活へ翻訳

例：

* 「中殿筋」ではなく
  →「片脚でブレない力」

* 「腹横筋」ではなく
  →「荷物で腰が潰れない力」


# 実際に現場で強い説明

## 悪い例

* 「大臀筋を鍛えます」
* 「体幹です」


## 良い例

* 「階段が楽になる」
* 「抱っこで腰が潰れにくくなる」
* 「つまずいても踏ん張りやすい」
* 「洗濯物を持っても疲れにくい」

# 科学的背景

## 特異性（Specificity）

Specificity Principle
動作・速度・姿勢・負荷方向が近いほど転移しやすい。


## SAID原則

SAID Principle
身体は与えられた要求に適応。

つまり、
* 座位ばかり
  → 座位適応

* 片脚課題
  → 片脚安定適応
  
になりやすい。
