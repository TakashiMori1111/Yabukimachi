# 肩甲骨・広背筋と肩こり ── 機能解剖学からの整理

※一般的健康情報としての整理。個別の医療判断・治療指示ではない。

## 結論

- **肩こりの正体** ＝ 僧帽筋上部と肩甲挙筋が「働きすぎて休めていない」状態。筋力不足ではなく、**神経が「ずっと力を入れろ」と命令し続けている**ことが本質
- **凝りは「結果」であって「原因」ではない** ＝ 凝っている場所を揉んでも戻りやすいのは、原因が別の場所にあるから
- **原因の上流** ＝ 胸郭（肋骨＋背骨）の可動性低下 → 肩甲骨が動けない → 僧帽筋上部が代償
- **広背筋の過緊張** ＝ 巻き肩・肩甲骨の固定・体幹回旋制限を助長し、肩こりの上流をさらに悪化させる構造
- **解決の方向** ＝ 「凝っている場所を緩める」のではなく「**凝らなくて済む条件を作る**」

# 【A】肩こりのメカニズム

## 1. 凝っている場所の正体

### 「首と肩の間のあの凝り」＝ほぼこの2つの筋

| 筋肉 | 位置 | 役割 | なぜ凝るか |
|------|------|------|-----------|
| **僧帽筋上部** | 首〜肩の表層 | 肩をすくめる・頭を支える | 頭の重さを支え続けて疲労する |
| **肩甲挙筋** | 肩甲骨〜首の深層寄り | 肩甲骨を引き上げる | 肩甲骨の動きが不足すると、代わりに過剰に働く |

- この2つが**常にONになっている状態** ＝ いわゆる「肩こり」

### ポイント

- 凝りの正体は「筋肉が硬くなった」ではなく、**「働きすぎて休めていない」**状態
- 筋肉そのものの問題というより、**神経が「ずっと力を入れろ」と命令し続けている**ことが本質
  - → 筋肉のコリ＝筋の問題ではなく、脳からの指令の問題

---

## 2. なぜ「働きすぎ」が起きるのか（5つの原因）

### 原因①：頭が重い（構造上の宿命）

- 人間の頭は約4〜6kg（ボウリング球1個分）
- しかも重心が**前寄り**にある
- 頭が少しでも前に出ると、後ろから引っ張って支える必要がある
- その「引っ張り役」が僧帽筋上部と肩甲挙筋

> イメージ：釣り竿の先に重りがついた状態。竿を立てていれば楽だが、前に傾けた途端、手元にものすごい負担がかかる。これが「頭が前に出た姿勢」で首〜肩に起きていること。

### 原因②：目を使うと頭が固定される

- 人間は情報の大部分を目から得ている
- スマホ・パソコン・子どもの様子を見る ── すべて「目線の固定」
- 目線が固定されると頭も固定される
- 頭が固定されると、首〜肩の筋肉は**「動かす役」から「固める役」に切り替わる**
- 固める役として最も動員されやすいのが僧帽筋上部

### 原因③：呼吸が浅くなる

- 忙しい・緊張・集中 → 呼吸が浅く速くなる
- 本来、呼吸は**肋骨を広げて行う**もの（横隔膜＋肋間筋）
- 浅い呼吸では肋骨が動かず、代わりに**首・肩の筋肉が呼吸の補助に動員される**
- 僧帽筋上部と斜角筋（首の横の筋肉）が「呼吸するたびに働く」状態 → 1日に約2万回分の余計な仕事
- さらに、広背筋が付着する**胸郭後下部も動かなくなる** → 広背筋も呼吸補助筋として過緊張に加担

> 「肩で息をしている」という表現は、比喩ではなく実際に起きている現象。

### 原因④：肩甲骨を動かす場面が日常にほぼない

- 日常動作の大半は「肘から先」で完結する（タイピング、スマホ、料理、抱っこ等）
- **肩甲骨はほとんど動いていない**
- 脳は使わない動きを忘れていく
- 結果：**肩甲骨＝固定物、腕＝操作部位**、という神経パターンが定着する
- 肩甲骨が動かなければ、動きの帳尻を合わせるのは僧帽筋上部

### 原因⑤：体は「安定」を最優先する

- 神経系の原則：**不安定より安定、動かすより固める**
- 疲れているとき、慣れない動きのとき、緊張しているとき
  - → 体は胸郭と肩甲骨を**固める方向を選ぶ**
- 固める役 ＝ 僧帽筋上部・肩甲挙筋

---

## 3. 揉んでも戻る理由

### 凝りは「結果」であって「原因」ではない

凝っている場所（僧帽筋上部）を揉む・針を打つ → 一時的に筋緊張は下がる

しかし、以下が変わっていなければ**数時間〜翌日には元に戻る**：

- 頭の位置（前に出た姿勢）
- 視線の使い方
- 呼吸パターン
- 肩甲骨の使われ方
- 胸郭（背骨＋肋骨）の動き

> 例えるなら：雑草を刈っても、根を取らないとまた生える。凝りを揉むのは「刈った雑草がまた伸びるのを待つ」ようなもの。

# 【B】肩甲骨・胸郭・広背筋の構造

## 4. 肩甲骨と胸郭の関係

### 肩甲骨は「浮いている骨」

- 肩甲骨は、骨同士のつながりが**鎖骨との1か所しかない**
- 残りはすべて**筋肉で胸郭（肋骨）の上に吊られている**
- つまり、**胸郭の状態が肩甲骨の動きを決める**

### 胸郭が動かないと肩甲骨も動けない

腕を上げるとき肩甲骨に必要な動き：
- **上方回旋**（上向きに回る）
- **後傾**（上端が後ろに倒れる）
- **外旋**（外向きに回る）

これらはすべて、**胸郭（背骨が反る＋肋骨が広がる）が前提条件**

胸郭が固まっている（猫背・肋骨が開かない）状態では：
- 肩甲骨の回旋・後傾が**物理的に出せない**
- 動けない分を**僧帽筋上部が「上に引き上げる」戦略で代償する**

> イメージ：テーブルの上に置いた皿（肩甲骨）を滑らせたいのに、テーブル自体が傾いて固まっている（胸郭が硬い）。皿は滑れないので、上から持ち上げるしかない（僧帽筋上部が頑張る）。

---

## 5. 広背筋の解剖学的構造と機能

> 広背筋（こうはいきん）は背中の大きな筋肉で、体幹と腕をつなぐ"橋渡し"の役割を持つ。ここが硬くなると、肩甲骨の動き・体幹の回旋・呼吸すべてに制限がかかる。

### 5-1. 解剖学的構造

- **起始** ＝ 胸腰筋膜（背骨〜骨盤後面を覆う強い膜）、下位胸椎・腰椎棘突起、腸骨稜
  - → 背中から骨盤まで広く付着している
- **停止** ＝ 上腕骨小結節稜（腕の骨の前内側）
- **特徴** ＝ 胸腰筋膜を介して体幹・骨盤と強固に連結

### 5-2. 機能（作用）

| 作用 | 動きのイメージ |
|------|--------------|
| 肩関節伸展 | 腕を後ろに引く |
| 肩関節内転 | 腕を体側に引き寄せる |
| 肩関節内旋 | 腕を内側にひねる |
| 体幹―上肢の力伝達 | 体幹と腕の橋渡し |

---

## 6. 巻き肩と広背筋過緊張の併発

### 6-1. それぞれの肩甲骨への牽引方向

| 状態 | 肩甲骨への影響 |
|------|--------------|
| **巻き肩** | 肩甲骨：前傾（前に倒れる）＋外転（外に開く）。上腕骨：内旋位 |
| **広背筋過緊張** | 上腕骨を内旋・伸展方向へ牽引。肩甲骨を下制方向へ牽引 |

### 6-2. 併発時の肩甲骨位置

- **前傾＋外転＋下制** ＝ 肩甲骨が肋骨前面に貼りつくような固定状態
  - → 肩甲骨の後傾・上方回旋が出にくくなる

### 6-3. 構造的な問題点

- **上肢挙上時** ＝ 肩甲上腕リズム（肩と腕の協調運動：腕2°に対して肩甲骨1°の割合で連動する仕組み）が破綻
- **結果**：
  - 肩前面の詰まり感
  - 首・僧帽筋上部の過活動（いつもの"代償者"がまた働く）
  - 腕を上げる・胸を開くといった動作がスムーズにできない

---

## 7. 一般生活で広背筋が張りやすい要因

### ① 長時間座位・猫背姿勢

- 背中が丸まる姿勢 → 肩が前方・内側に巻き込まれる配置
- 広背筋が**短縮位で固定**される
- → 伸ばされないまま活動低下＋血流低下

### ② 片側優位動作の反復

- マウス操作、片手荷物、片側抱っこ
- → 左右差のある広背筋緊張が生じる
- → 体幹回旋時のブレーキ役として常時緊張

### ③ 体幹回旋・側屈の欠如

- 日常動作が「前後」中心で、ひねる・横に倒す動きが極端に少ない
- → 広背筋の**斜走線維（ななめの筋線維）**が使われず、滑走性（筋が動く余地）が低下

### ④ 呼吸様式の影響

- 浅い胸式呼吸 → 肋骨の動きが小さい
- → 広背筋が付着する胸郭後下部が動かない → 呼吸補助筋として過緊張
- ※原因③（セクション2）と同じメカニズムが広背筋側にも波及

# 【C】回旋制限と連鎖

## 8. 広背筋過緊張が回旋に与える影響

### 8-1. 体幹回旋の「制動解除」

- 広背筋は体幹回旋に対して**ブレーキ側**として働く場面が多い
- **過緊張時** ＝ 胸郭と骨盤の相対回旋が制限される
- **緊張低下時** ＝ 胸郭─骨盤の分離運動が可能になる
  - → 身体をひねる動きが軽くなる

### 8-2. 胸腰筋膜の滑走性改善

- 広背筋は胸腰筋膜を強く牽引している
- **硬化時** ＝ 筋膜の層間滑走（膜と膜の間の滑り）が低下
- **緩和時** ＝ 体幹回旋、歩行時の腕振り、投げる・振り向くといった動作が少ない力で成立

### 8-3. 肩甲帯と体幹の協調回復

- **過緊張時** ＝ 肩甲骨が下制・内転方向に固定 → 上肢挙上や回旋で代償動作が増加
- **緩和時** ＝ 肩甲骨の可動余地が拡大 → **体幹主導 → 上肢追従**の運動連鎖が成立

### 8-4. 日常動作レベルでの影響

| 状態 | 動作への影響 |
|------|-------------|
| 広背筋が硬い | 体がひねりにくい、振り向きづらい、腕が重い |
| 広背筋が緩む | 歩行が軽い、振り向きが速い、腕が自然に出る |

---

## 9. 回旋動作を制限するその他の構造

> 広背筋だけが原因ではない。以下の構造も回旋制限に関与する。

### 9-1. 腹斜筋群（外腹斜筋・内腹斜筋）

- **役割** ＝ 体幹回旋の主動作筋
- **過緊張時** ＝ 回旋の「出力筋」が同時に「制動筋」化
- **弊害** ＝ ひねり動作の初動が重い、腰椎単独回旋の増加

### 9-2. 脊柱起立筋群（背骨沿いの筋群）

- **役割** ＝ 体幹伸展・姿勢保持
- **過緊張時** ＝ 胸椎の回旋が低下
- **弊害** ＝ 回旋が腰椎（本来回旋が少ない部位）に集中 → 腰部ストレス増大

### 9-3. 腸腰筋（股関節前面の深層筋）

- **役割** ＝ 股関節屈曲・体幹安定
- **短縮・硬化時** ＝ 骨盤前傾固定
- **弊害** ＝ 骨盤と胸郭の相対回旋が減少 → 体幹回旋が「上半身のみ」になる

### 9-4. 胸郭自体（肋骨・肋軟骨）

- **役割** ＝ 回旋と呼吸の土台
- **可動性低下** ＝ 肋骨が動かない
- **弊害** ＝ 呼吸と回旋が分離、動作が硬く疲れやすい
  - → 胸郭の問題はセクション4の肩甲骨制限にも直結する

# 【D】対策

## 10. 解決の方向性

### 根本方針

**「凝っている場所を緩める」のではなく、「凝らなくて済む条件を作る」**

| やること | 狙い |
|---------|------|
| **胸郭を動かす**（背骨を反る・肋骨を広げる） | 肩甲骨が動ける「土台」を作る |
| **肩甲骨の支持筋を再教育する**（下部僧帽筋・前鋸筋・広背筋） | 僧帽筋上部の「代わり」を育てる |
| **呼吸を整える**（肋骨を使った呼吸） | 僧帽筋上部・広背筋を呼吸補助から解放する |
| **広背筋の緊張を下げる** | 肩甲骨の可動余地を広げ、回旋制限を解除する |

### 経験則：運動介入で改善しやすいパターン

> 以下は個人的な経験に基づく観察であり、すべての人に当てはまるとは限らない。

- 針やマッサージで一時的に楽になるが**すぐ戻る** → 胸郭・肩甲骨の運動介入で改善しやすかった
- 胸郭回旋（ワイパー系のストレッチ）で**即時的に凝りが軽くなる** → 胸郭の可動性制限が主因である可能性が高い
- ロウ（引く動作）やラットプルダウンを**肩をすくめずに**行えるようになると、日常の凝りが減りやすかった
- 自重だけでも効果は出る。**高負荷より「正しい場所が働いている感覚」が大事**

## 11. 器具なしエクササイズ（優先度順）

**すべて共通ルール：「首と肩の間が力まない」ことが最優先。力んだ時点で逆効果。**

### ① 四つ這いロウ（肩甲骨の支持再学習）

- 四つ這いから片手を軽く浮かせ、肘を後ろに引く意識
- 実際に大きく動かなくてもOK
- 合言葉：**「首は長く、肩はすくめない」**
- 左右 各5〜8回 × 2〜3セット（1回3〜5秒キープ）

### ② 壁プル（ラットプル動作の神経学習）

- 壁に向かって立ち、両手を壁につけて「上から引き下ろす」意識
- 壁は動かないので、実際には力を入れるだけ（等尺性収縮）
- 合言葉：**「肘で引く、肩で引かない」**
- 5秒 × 5〜8回

### ③ うつ伏せYホールド（下部僧帽筋の活性化）

- うつ伏せで親指を天井に向け、Y字に腕を軽く浮かせる
- 高く上げない。首が反らない
- 5〜10秒 × 5回

### ④ 壁スライド（肩甲骨を肋骨に沿わせる制御）

- 壁に前腕をつけ、肩をすくめずに上へスライド
- 合言葉：**「肩甲骨が肋骨に吸い付く感じ」**
- 6〜10回 × 2セット

### ⑤ 胸郭回旋＋呼吸（仕上げ）

- 仰向けまたは四つ這いで体をひねり、ひねった側の背中に息を入れる
- 肋骨の広がりを感じられればOK
- 左右 各5呼吸

### よくある失敗

| 失敗パターン | なぜダメか |
|------------|-----------|
| 「効かせよう」として力む | 僧帽筋上部がまた出てしまう |
| 肩を下げすぎる | 別の代償が出る |
| 胸を張りすぎる | 腰が反って胸郭が動いていない |
| 呼吸を止める | 呼吸補助筋がまた動員される |

## 12. 悪循環と好循環のモデル

### 悪循環（多くの人が陥っているパターン）

```
頭が前に出る → 視線固定 → 呼吸浅化 → 胸郭が動かなくなる
→ 広背筋も短縮・過緊張 → 肩甲骨が動けなくなる
→ 僧帽筋上部が常時ON → 凝り形成
→ 凝りを揉む → 一時的に楽 → 条件が変わっていないので再燃
```

### 好循環（介入の方向性）

```
胸郭を動かす → 肋骨が広がる → 呼吸が変わる
→ 広背筋の緊張低下 → 肩甲骨が動ける → 支持筋が働く
→ 僧帽筋上部が休める → 凝りの「上流」が改善 → 再燃しにくい
```

# 【E】補足

## 13. ママさん・産後に特に起きやすい理由

育児期は、セクション2の5つの原因が**同時に揃いやすい**。

| 原因 | 育児場面 |
|------|---------|
| 頭が前に出る | 授乳、おむつ替え、添い寝 |
| 視線の固定 | 子どもの見守り、スマホ（片手抱っこ中） |
| 浅い呼吸 | 睡眠不足、常時の緊張・気疲れ |
| 肩甲骨が動かない | 抱っこ紐、前かがみ姿勢の持続 |
| 体が「固める」を選ぶ | 疲労、慣れない動作の連続 |

### 産後特有の要因

- **腹筋群（腹横筋・骨盤底筋）の機能低下** → 体幹の安定性が下がる → 代わりに肩まわりで安定を取ろうとする
- **抱っこ紐による肩甲帯の固定** → 肩甲骨が長時間「動けない」環境に置かれる
- **片側抱っこの反復** → 広背筋の左右差・体幹回旋制限が助長される

## 14. 受診すべきサイン

以下がある場合は、セルフケアの前に医療機関（整形外科）の受診を優先：
