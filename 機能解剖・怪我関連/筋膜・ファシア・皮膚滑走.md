# 筋膜・ファシア・皮膚滑走とテンセグリティ

## 結論

ファシアは、筋肉を包む膜だけではなく、腱・腱膜・靭帯・関節包・皮下組織などを含む結合組織ネットワークとして扱うと現場で理解しやすい。

ただし、ファシアを「全身を一瞬で整える魔法のネットワーク」として扱うのは不適切。競技・ケアで優先すべきなのは、次の整理。

1. **筋膜・ファシアは実在する重要組織**
2. **パフォーマンスでは腱・腱膜・筋腱移行部が特に重要**
3. **痛みや硬さは、ファシア単独ではなく負荷量・神経系・筋力・可動域・睡眠・心理要因も含めて判断**
4. **テンセグリティは説明モデルであり、介入効果の単独根拠ではない**
5. **皮膚滑走は実務上の評価視点として有用だが、科学的根拠は発展途上**

この資料は一般情報であり、個別の診断・治療助言ではない。急性外傷、強い痛み、しびれ、腫れ、内出血、荷重困難、夜間痛、発熱などがある場合は医療機関評価を優先。

# 2. まず定義

## ファシアとは

**根拠レベル：高**

Fascia Research Societyの整理では、ファシアには狭義と広義がある。

### 狭義のファシア

筋肉や内臓などを包み、分け、つなぐシート状または鞘状の結合組織。

### 広義のファシアルシステム

身体中に広がる、コラーゲンを含む結合組織の連続体。腱、靭帯、腱膜、関節包、浅筋膜、深筋膜、神経血管鞘、皮下組織なども含めて考える。

簡単に言えば、ファシアは次のような組織。

> 筋肉・腱・靭帯・関節・皮下組織をつなぎ、力を伝え、組織同士の滑りを助ける結合組織ネットワーク

## 筋膜・腱・腱膜の違い

| 名称 | 簡単な説明 | 例 |
|---|---|---|
| 筋膜 | 筋肉や筋群を包む膜 | 大腿筋膜、胸腰筋膜、筋外膜、筋周膜、筋内膜 |
| 腱 | 筋肉と骨をつなぐロープ状の結合組織 | アキレス腱、膝蓋腱 |
| 腱膜 | 薄く広がった腱 | 足底腱膜、腹直筋鞘、外腹斜筋腱膜 |
| 靭帯 | 骨と骨をつなぎ関節を安定させる組織 | 前十字靭帯、足関節外側靭帯 |
| 関節包 | 関節を包む袋状の組織 | 肩関節包、股関節包 |

現場で「筋膜」と一括りにされるものの中には、実際には腱・腱膜・靭帯・関節包が含まれることが多い。競技やケガを考える場合は、この区別が重要。

# 3. 筋膜・ファシアで確実に言えること

## 3-1. ファシアは力を伝える

**根拠レベル：中**

筋肉の力は、腱を通じて骨へ伝わるだけではない。周囲の筋膜・腱膜・結合組織にも一部伝わる。これを筋膜性力伝達という。

ヒト生体内の研究をまとめたスコーピングレビューでは、筋膜性力伝達の存在は慎重に支持されている。ただし、研究の質や方法にはばらつきがあり、どの程度パフォーマンスや痛みに影響するかはまだ限定的な理解。

### 現場での言い換え

筋肉は1本ずつ完全に独立したモーターではない。周囲の膜や腱膜を介して、隣の組織にも少し力を伝える。

## 3-2. ファシアは滑走する

**根拠レベル：中**

筋肉同士、皮膚と筋肉、筋膜層同士は、動作中に少しずつずれる。この滑りを滑走性という。

滑走性が低下すると、動きにくさ・張り感・痛みと関連する可能性がある。ただし、次の断定は不可。

- 滑りが悪いから必ず痛い
- 手技で癒着が剥がれる
- 筋膜リリースで根本原因が消える

### 現場での言い換え

競技動作では、筋肉が縮むだけでなく、組織同士がスムーズにずれる必要がある。

## 3-3. ファシアには痛みを感じる神経がある

**根拠レベル：中**

胸腰筋膜などのファシアには、痛み信号を拾う神経が存在する。腰痛研究では、胸腰筋膜が痛み発生に関与しうる組織として注目されている。

ただし、胸腰筋膜が腰痛の唯一原因と断定する根拠はない。腰痛は、組織負荷、神経感作、睡眠、ストレス、心理社会的要因、運動不足などが重なることが多い。

## 3-4. ファシアはセンサーとしても働く

**根拠レベル：中**

皮膚、浅筋膜、深筋膜、筋肉、腱には、圧・伸張・位置情報を拾うセンサーがある。専門用語では機械受容器、固有感覚受容器という。

- 機械受容器：圧、触覚、伸びなどを感じるセンサー
- 固有感覚受容器：関節や筋肉の位置・動きを感じるセンサー

この情報は姿勢制御や動作調整に関係する。ただし「筋膜だけが脳へ運動情報を送っている」と考えるのは過剰。

# 4. テンセグリティの正しい扱い

## テンセグリティとは

**根拠レベル：中**

テンセグリティとは、圧縮材と張力材のバランスで全体の形を保つ構造モデル。

身体に当てはめる場合は、次のように理解すると使いやすい。

| 役割 | 身体での例 |
|---|---|
| 圧縮材 | 骨 |
| 張力材 | 筋肉、腱、靭帯、筋膜、腱膜 |

## 使える理解

身体を「骨が積み木のように重なっているだけの構造」と見るより、筋肉・腱・靭帯・筋膜の張り具合で全体の形と動きが決まる、と考えると姿勢や連動を説明しやすい。

### 実務で使える例

- 足部の剛性が、膝・股関節・骨盤の動きに影響する可能性
- 胸郭の硬さが、肩や腰の動きに影響する可能性
- 股関節制御が、膝外側痛や鼠径部痛に影響する可能性

## 使いすぎ注意

**根拠レベル：不明**

次の主張は断定不可。

- 全身は完全なテンセグリティ構造である
- 1か所を緩めると全身が整う
- 筋膜ラインを押せばパフォーマンスが上がる
- 痛みの原因はすべてファシア

テンセグリティは評価の視点としては有用。しかし、介入は通常の運動科学に落とす必要がある。

- 筋力
- 可動域
- 腱負荷耐性
- 動作制御
- 疲労管理
- 練習量管理

# 5. 競技現場で重要なファシア関連部位

## 5-1. 足底腱膜・アキレス腱・下腿三頭筋

**重要度：高**  
**根拠レベル：高から中**

### 関連競技

- 陸上短距離
- 長距離
- サッカー
- バスケットボール
- バレーボール
- テニス
- ラグビー
- バレエ
- ジャンプ競技

### 役割

足底腱膜とアキレス腱は、走る・跳ぶ動作でバネのように働く。足部アーチ、足関節の剛性、下腿三頭筋の出力が合わさり、接地時間・反発・加速に関係する。

### 多い障害

- アキレス腱障害
- 足底腱膜症
- 下腿筋損傷
- シンスプリント様症状との併存

### 見るべき項目

- 片脚カーフレイズの回数・痛み・左右差
- ホッピング時の接地音、接地時間、反発感
- 足関節背屈可動域
- 足部アーチの過剰低下または過剰固定
- 急なシューズ・スパイク・サーフェス変更
- スプリント量・ジャンプ量・坂道走の急増

## 5-2. ハムストリングス腱膜・筋腱移行部

**重要度：高**

### 多い障害

## 5-4. 膝蓋腱・大腿四頭筋腱膜

**重要度：高**  
**根拠レベル：高から中**

- サッカー
- 陸上跳躍
- 重量挙げ

### 役割

### 見るべき項目

- ジャンプ総量
- 着地音
- 膝前面痛の出る角度
- スクワット・ランジ時の膝前方移動耐性
- 大腿四頭筋の筋力
- 腱負荷後24時間の痛み反応

膝がつま先より前に出ること自体は悪ではない。問題は、その負荷に膝蓋腱と大腿四頭筋が耐えられるかどうか。

# 6. 皮膚の滑走性

## 皮膚は動く組織

**根拠レベル：低から中**

皮膚は身体の表面を覆うだけでなく、皮下組織・浅筋膜を介して、下の筋肉や深筋膜と相対的にずれる。

この「ずれ」を皮膚滑走性と呼ぶ。専門的には、皮膚と浅層組織の外的可動性として扱うと安全。

## 層の整理

| 層 | 役割 |
|---|---|
| 表皮 | 外界との境界。運動への直接関与は小さい |
| 真皮 | コラーゲン・エラスチンを含む支持層 |
| 皮下組織 | 皮膚と深部組織のずれを作る層 |
| 浅筋膜 | 皮下脂肪・結合組織・神経血管などを含む浅層のファシア |
| 深筋膜 | 筋群を包み、区画を作る強い膜 |

## 皮膚滑走性で言えること

**根拠レベル：低から中**

言えること。

- 皮膚・皮下組織・浅筋膜は、動作中にずれる
- 手術後瘢痕、外傷後、炎症後では滑走性が低下することがある
- 皮膚表面の変形から、下層の軟部組織の硬さを推測する研究は出てきている
- 軟部組織モビライゼーションで可動域や痛みが変化する報告はある

まだ断定できないこと。

- 皮膚滑走が悪いから必ず痛い
- 皮膚をつまめば全身の運動連鎖が整う
- 皮膚介入で筋出力が安定して上がる
- 皮膚滑走のセルフチェックだけで原因部位を特定できる

## 皮膚滑走性が低下しやすい条件

**根拠レベル：低**

以下は実務上の観察を含む整理。

- 同じ姿勢の持続
- 運動不足
- 捻挫・打撲・手術後
- 瘢痕周囲
- 反復負荷による局所の張り
- ストレスや緊張による防御的な力み

ただし、ストレスや緊張が皮膚・筋膜のトーンをどの程度直接変えるかは個人差が大きい。

## セルフチェック

**根拠レベル：低**

### やり方

1. 座位または立位で行う
2. 太もも、お腹、背中、肩まわりなどの皮膚を軽くつまむ
3. 上下・左右・斜めに軽くずらす
4. 痛み、左右差、方向差を確認する

### 見るポイント

| 項目 | 意味 |
|---|---|
| 動く方向に偏りがある | 特定方向の滑走が少ない可能性 |
| 左右差がある | 姿勢・動作・過去の外傷の影響の可能性 |
| 痛み・引きつれ感がある | 瘢痕、炎症後変化、過敏性の可能性 |

つまめないこと自体が異常とは限らない。筋肉量、皮下脂肪量、部位、年齢、性別で大きく変わる。

## セルフケア

**根拠レベル：低**

### 基本ルール

- 痛気持ちいい以下
- ゴリゴリ押さない
- 皮膚を軽くつまむ、またはずらす
- 呼吸を止めない
- 赤みが強く残るほど行わない
- 痛みが増える場合は中止

### 方法

| 方法 | やり方 |
|---|---|
| つまむ | 皮膚を軽く持ち上げ、左右に小さくゆする |
| ずらす | 手のひらで皮膚を上下・左右にゆっくり動かす |
| 呼吸と併用 | 息を吐きながら力を抜き、軽くずらす |

### 位置づけ

皮膚滑走ケアは、トレーニングやリハビリの主役ではなく、下準備として扱う方が安全。

- 運動前：感覚入力を増やし、動き始めを作る目的
- 運動後：張り感の調整、リラックス目的
- 瘢痕周囲：医療職の指導下で行うのが安全

# 7. よくある誤解

## 誤解1：筋膜リリースで癒着が剥がれる

**評価：断定不可**

フォームローリングや手技で可動域が一時的に改善することはある。ただし、それが物理的に癒着を剥がしたためとは断定できない。

効果機序は、神経系、痛みの感受性、筋緊張、血流、組織温度、主観的リラックスなどが複合している可能性。

## 誤解2：筋膜ライン通りに鍛えればパフォーマンスが上がる

**評価：断定不可**

筋膜連結や筋膜性力伝達は部分的に支持される。しかし、いわゆる筋膜ラインをそのままトレーニング設計の主軸にする根拠は限定的。

現場では、次の条件を優先。

- 関節角度
- 速度
- 力の方向
- 接地条件
- 片脚支持
- 疲労条件
- 反応判断
- 競技特異性

## 誤解3：ストレッチすれば筋膜だけが伸びる

**評価：単純化しすぎ**

ストレッチで可動域は改善しうる。ただし、変化するのは筋膜だけではない。

- 筋肉
- 腱
- 神経系
- 痛み許容
- 結合組織特性
- 伸張への慣れ

これらが複合している。

## 誤解4：痛い場所の筋膜が原因

**評価：不明**

痛い場所にファシアが関与することはある。しかし、痛みは多因子。

- 組織損傷
- 負荷量
- 神経感作
- 睡眠不足
- 心理的ストレス
- 栄養状態
- 競技ストレス
- 過去の外傷

痛い場所だけを押す、伸ばす、剥がすという対応は不十分になりやすい。

---

# 8. トレーナーが見るべき評価項目

## 8-1. 組織負荷の管理

**根拠レベル：高から中**

最初に見るべき項目。

- 直近7日間の走行距離
- スプリント本数
- ジャンプ回数
- 切り返し回数
- 投球数
- 高強度セッション数
- サーフェス変更
- シューズ・スパイク変更
- 睡眠不足
- 疲労蓄積

「硬いから痛い」より、「準備できていない負荷が入った」ケースが多い。

## 8-2. 腱・腱膜の負荷耐性

**根拠レベル：中**

優先テスト。

- 片脚カーフレイズ
- ホッピング
- 片脚スクワット
- Copenhagen adduction
- Nordic hamstring
- 片脚RDL
- Drop jump
- 反復ジャンプ

見る項目。

- 痛み
- 左右差
- 回数
- 動作の質
- 24時間後の反応
- 疲労時のフォーム崩れ

## 8-3. 滑走性より先に、出力と制御

**根拠レベル：中**

「筋膜が硬い」だけで評価を終えない。まず以下を確認。

- 筋力不足
- 腱負荷耐性不足
- 可動域不足
- 動作制御不良
- 疲労耐性不足
- 練習量過多
- 睡眠不足

## 8-4. 全身連動

**根拠レベル：中**

下肢スポーツでは、足部から骨盤までを一連で見る。

> 足部 → 足関節 → 膝 → 股関節 → 骨盤

投球・スイング競技では、下肢から上肢までを見る。

> 足部 → 股関節 → 骨盤 → 胸郭 → 肩甲骨 → 肩 → 肘

# 9. 介入の優先順位

## 優先1：負荷量の調整

**根拠レベル：高から中**

痛みのある腱・腱膜・筋腱移行部では、最初に練習量と高強度負荷を調整する。

完全休止が常に最適とは限らない。症状を悪化させない範囲で、段階的負荷を保つケースが多い。ただし急性外傷や断裂疑いでは医療評価が前提。

## 優先2：アイソメトリック

**根拠レベル：中**

関節を動かさず力を入れる方法。痛みがある時期の負荷導入として使いやすい場合がある。

例。

- 壁押しカーフ
- スプリットスクワット保持
- Copenhagen hold
- ハムストリングブリッジ保持
- 等尺性膝伸展

## 優先3：重いスロー筋トレ

**根拠レベル：中**

ゆっくりした高負荷筋トレ。腱・腱膜・筋腱移行部の耐性作りに使われる。

例。

- カーフレイズ
- スクワット
- RDL
- レッグプレス
- ヒップスラスト
- Copenhagen adduction

## 優先4：弾性系トレーニング

**根拠レベル：中**

腱・腱膜のバネ機能を戻す段階。

例。

- ポゴジャンプ
- ホップ
- バウンディング
- スキップ
- メディシンボールスロー
- 短距離加速走

## 優先5：競技特異的負荷

**根拠レベル：中**

実際の競技に近い条件へ戻す段階。

例。

- サッカー：キック、切り返し、スプリント
- バスケットボール：ジャンプ、着地、減速
- 野球：投球、回旋、加速減速
- バレエ：ポワント、ターンアウト、ジャンプ着地
- 陸上：最大速度走、スタート、接地剛性

---

# 10. フォームローリング・ストレッチ・皮膚ケアの位置づけ

## フォームローリング

**根拠レベル：中**

使える場面。

- ウォームアップ前の可動域確保
- 運動後の主観的疲労感軽減
- 一時的な張り感の調整
- セルフケアのルーティン化

限界。

- 筋膜癒着を剥がすと断定不可
- パフォーマンス向上の主役ではない
- 痛みのある腱や骨突起を強く圧迫しない

## 静的ストレッチ

**根拠レベル：中**

使える場面。

- 可動域改善
- クールダウン
- 技術的に必要な可動域確保
- バレエ、体操、格闘技など可動域要求が大きい競技

注意点。

- 最大筋力・瞬発力直前に長時間行う場合は、パフォーマンス低下の可能性を考慮
- 短時間・適量・その後に動的ウォームアップを入れる場合、影響は条件依存

## 皮膚滑走ケア

**根拠レベル：低**

使える可能性がある場面。

- 動き始めの違和感
- 浅い層の引きつれ感
- 瘢痕周囲のつっぱり
- 強く押すケアが合わない人
- トレーニング前の感覚入力

限界。

- 科学的根拠はまだ発展途上
- 効果は即時的・主観的変化にとどまることが多い
- 原因診断には使えない
- 痛みが強い場合は医療評価が優先

# 12. 赤信号

以下は一般情報として医療機関評価を優先。

- 急な鋭い痛み
- 「ブチッ」「弾けた」感覚
- 明らかな腫れ
- 内出血
- 片脚荷重困難
- 踵上げ不能
- しびれ
- 感覚低下
- 夜間痛
- 安静時痛の増悪
- 発熱や全身症状
- 外傷後に悪化傾向

# 13. 最終整理

ファシアを現場で使うなら、抽象的に「全身がつながっている」と言うだけでは不十分。

重要なのは、次の順番。

1. **定義を分ける**
   - 筋膜、腱、腱膜、靭帯、関節包、皮下組織を区別。

2. **根拠レベルを分ける**
   - 系統的レビューで言えること、臨床推論、経験則を混ぜない。

3. **競技動作に落とす**
   - 走る、跳ぶ、切り返す、投げる、蹴る、着地する。

4. **介入は通常の運動科学へ戻す**
   - 筋力、腱耐性、可動域、動作制御、疲労管理、負荷管理。

5. **手技・ローラー・ストレッチ・皮膚ケアは補助**
   - 主役は段階的な負荷適応。

一言で言えば、

> 「筋膜を整える」より、「結合組織が耐えられる負荷を段階的に作る」方が、スポーツ医学的には重要度が高い。
