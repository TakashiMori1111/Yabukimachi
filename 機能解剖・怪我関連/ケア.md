# 運動後ケアと傷害予防の科学的根拠

## 1. 筋肉痛（DOMS）のメカニズム

### 1-1. 遅発性筋肉痛（DOMS）の定義

**遅発性筋肉痛（Delayed Onset Muscle Soreness: DOMS）**

- 運動後8〜24時間で発生し、24〜72時間でピークに達する筋痛
- 主に不慣れな運動や高強度エキセントリック収縮（伸張性収縮）後に発生
- 数日〜1週間程度で自然軽快

### 1-2. 発生メカニズム（現在の定説）

**筋線維の微細損傷モデル**

1. エキセントリック収縮時の機械的ストレス → 筋線維（特にZ帯）の微細損傷
2. 損傷部位への炎症性細胞（好中球、マクロファージ）の浸潤
3. 炎症メディエーター（プロスタグランジン、ブラジキニンなど）の放出
4. 侵害受容器の感作 → 痛覚の増強
5. 筋線維の修復・再生プロセス（サテライト細胞の活性化）

**補足：乳酸は原因ではない**

- 従来「乳酸＝疲労物質」とされたが、現在は否定
- 乳酸は運動中に産生されるが、数時間以内に代謝され、DOMSの発生時期（24〜72時間後）とは時間的に一致しない
- 乳酸はむしろエネルギー源として再利用される（Cori回路）

### 1-3. 不明な点・個人差

- DOMSの程度や持続期間に大きな個人差がある理由は完全には解明されていない
- 遺伝的要因、トレーニング歴、筋線維タイプ組成、炎症反応の個人差などが関与すると推測

## 2. マッサージ・ほぐしの効果と限界

### 2-1. マッサージの期待される効果

**血流改善による代謝促進**

- 機械的刺激による血管拡張 → 酸素・栄養供給の増加
- 代謝産物（疲労関連物質）の除去促進
- 筋修復プロセスの円滑化

**筋膜・軟部組織の柔軟性改善**

- 筋膜の癒着や硬結の軽減
- 筋・筋膜の滑走性改善 → 可動域の向上

**神経生理学的効果（ゲートコントロール理論）**

- 触圧覚刺激が脊髄レベルで痛覚伝達を抑制
- 固有受容器への刺激 → 筋緊張の抑制（Ia抑制）
- 副交感神経活動の亢進 → リラクゼーション効果

### 2-2. エビデンスの現状と限界

**肯定的な報告**

- 主観的な痛み軽減効果は複数の研究で報告
- 短期的な可動域改善効果

**否定的・限定的な報告**

- パフォーマンス回復への効果は限定的（メタアナリシス, 2020年代）
- プラセボ効果との区別が困難
- 筋損傷の組織学的修復速度への影響は不明確

### 2-3. 注意点・禁忌

**炎症期（受傷後48〜72時間）の強圧マッサージ**

- 炎症反応の増悪リスク
- 出血・浮腫の悪化可能性
- 修復プロセスの遅延リスク

**鑑別診断の重要性**

- DOMS vs. 筋挫傷（肉離れ）の鑑別が必須
- Grade II以上の筋損傷（部分断裂）では強圧禁忌
- 疑わしい場合は医療機関受診を優先

### 2-4. 実践的推奨

**推奨されるマッサージ**

- 軽圧〜中圧（痛みを伴わない範囲）
- 短時間（1部位あたり1〜2分）
- 運動後24時間以降、または軽度のDOMSに対して
- リラクゼーション目的として位置づける

**避けるべきマッサージ**

- 運動直後〜48時間以内の強圧マッサージ
- 痛みを伴う強い圧迫
- 急性損傷（腫脹・熱感・著明な圧痛）への施術

## 3. 筋膜リリースの科学的評価

### 3-1. 筋膜（ファシア）の解剖学的理解

**筋膜の定義**

- 筋肉・臓器を包む結合組織のネットワーク
- 浅筋膜、深筋膜、筋外膜、筋周膜、筋内膜などの階層構造
- コラーゲン線維、エラスチン線維、基質（ヒアルロン酸など）で構成

**筋膜の機能**

- 力の伝達（筋膜連鎖、myofascial chain）
- 組織間の滑走性確保
- 固有受容感覚の伝達（機械受容器が豊富に分布）

### 3-2. 筋膜リリースの理論的根拠

**期待される効果**

- 筋膜の滑走性改善 → 可動域拡大
- 筋膜の粘弾性変化 → 組織の柔軟性向上
- 固有受容器への刺激 → 神経筋制御の改善
- 筋膜内の侵害受容器の脱感作 → 痛み軽減

### 3-3. エビデンスレベルと限界

**現状のエビデンス**

- 短期的な可動域改善効果：複数のRCTで報告（効果量は小〜中程度）
- 主観的な痛み・こわばり感の軽減：報告あり
- 長期的効果、傷害予防効果：エビデンス不十分

**科学的議論の焦点**

- 「筋膜のみを選択的にリリースできるか」は疑問視されている
- 実際には筋・腱・神経・血管を含む複合的な組織への刺激
- 効果の多くは神経生理学的機序（ゲートコントロール、筋緊張の抑制）による可能性
- プラセボ効果の寄与も無視できない

**注意点**

- 過度な圧迫 → 組織損傷、内出血、痛みの増悪リスク
- 「筋膜リリース＝万能」という過信は危険
- 他の介入（運動療法、姿勢改善）との併用が前提

### 3-4. 臨床的位置づけ

筋膜リリースは「補助的介入」として位置づけるべき

**適応**

- 主訴：慢性的なこわばり、軽度の痛み、可動域制限
- 適用条件：急性炎症がない、組織損傷がない

**実施方法**

- 軽圧〜中圧、痛みを伴わない範囲
- 短時間（30秒〜2分/部位）
- フォームローラー、マッサージボール、徒手など

**併用が必須**

- 運動療法、姿勢指導、生活習慣改善

## 4. 入浴（温熱療法）の生理学的効果

### 4-1. 温熱刺激の生理学的作用

**循環動態への影響**

- 皮膚・筋血流量の増加（血管拡張）
- 組織への酸素・栄養供給の促進
- 代謝産物の除去促進

**軟部組織への影響**

- 筋温上昇 → 筋・腱の粘弾性低下（柔軟性向上）
- コラーゲン線維の伸展性向上
- 関節可動域の改善

**神経系への影響**

- 副交感神経活動の亢進 → リラクゼーション
- 筋紡錘の感度低下 → 筋緊張の軽減
- エンドルフィン分泌 → 鎮痛効果

### 4-2. 運動習慣の有無による効果の差

**運動習慣がない人への効果が大きい理由**

- 日常的な筋活動が少ない → 血流・代謝が慢性的に低下
- 入浴が「受動的な循環改善手段」として機能
- 筋・結合組織のこわばり予防、可動域維持に寄与

**運動習慣がある人**

- 運動自体が循環・代謝を促進するため、入浴の追加効果は相対的に小さい
- ただし、リカバリー手段としては有効（心理的効果も含む）

### 4-3. 実施上の推奨条件

**温度・時間**

- 湯温：38〜40℃（ぬるめ〜適温）
- 時間：10〜15分程度
- 高温・長時間は心血管系への負担増、脱水リスク

**タイミング**

- 運動後のクールダウン期（激しい運動直後は避ける、30分以上経過後が望ましい）
- 就寝1〜2時間前（深部体温の低下 → 睡眠の質向上）
- 日常的な習慣として

**禁忌・注意**

- 急性炎症期（受傷後48時間以内）は温熱を避ける（RICE/PEACE原則）
- 心血管疾患、高血圧のある場合は医師と相談

## 5. ストレッチの効果とタイミング

### 5-1. ストレッチの分類と生理学的効果

**静的ストレッチ（Static Stretching）**

- 筋腱複合体の伸張 → 粘弾性変化、可動域改善
- 筋紡錘の脱感作 → 筋緊張の低下
- 効果持続時間：数分〜数時間（一過性）

**動的ストレッチ（Dynamic Stretching）**

- 関節可動域内での反復運動
- 筋温上昇、神経筋活性化
- パフォーマンス準備に適する

**PNFストレッチ（Proprioceptive Neuromuscular Facilitation）**

- 収縮-弛緩サイクルを利用
- 相反抑制、自原抑制の神経メカニズム活用
- 可動域改善効果が大きい（静的ストレッチより効果量大）

### 5-2. タイミング別の推奨と根拠

| タイミング | 推奨種類 | 目的・根拠 | 注意点 |
|-----------|---------|-----------|--------|
| **運動前** | 動的ストレッチ | 神経筋活性化、関節可動域確保、筋温上昇 → パフォーマンス向上、傷害予防 | 静的ストレッチの長時間実施（60秒以上/筋群）は筋力・パワー発揮を一時的に低下させる可能性（メタアナリシス） |
| **運動後** | 静的ストレッチ | 筋緊張の軽減、クールダウン、柔軟性の維持・改善 | DOMS予防効果は限定的（複数のメタアナリシスで効果量小） |
| **日常生活** | 静的・動的併用 | 慢性的なこわばり予防、姿勢改善、血流促進 | 継続性が重要（週3回以上、4週間以上で効果） |

### 5-3. ストレッチの限界とエビデンス

**傷害予防効果**

- 「ストレッチ単独での傷害予防効果は限定的」（Cochrane Review, 2014; 更新2020）
- 柔軟性向上と傷害リスク低下の因果関係は不明確
- ウォームアップ全体の一部としては有効

**DOMS軽減効果**

- 効果量は小さい（メタアナリシス）
- 主観的な「気持ちよさ」はあるが、組織修復速度への影響は不明

**過度なストレッチのリスク**

- 筋腱複合体の微細損傷
- 関節不安定性の増大（過度な靭帯弛緩）
- 痛みを伴うストレッチは炎症を増悪させる可能性

**推奨実施方法**

- 1筋群あたり30〜60秒、2〜4セット
- 痛みを伴わない範囲（軽い張り感まで）
- 呼吸を止めない（リラックスした状態）
- 週3回以上、4週間以上の継続で効果

## 6. 肉離れ（筋挫傷）の発生機序と予防

### 6-1. 筋挫傷の分類（British Athletics Muscle Injury Classification）

**Grade 0**：筋硬結、圧痛あり、機能障害なし

**Grade I**：軽度の筋線維損傷、軽度の機能障害、復帰まで1〜2週間

**Grade II**：部分断裂、明確な機能障害、復帰まで3〜6週間

**Grade III**：完全断裂、高度な機能障害、復帰まで3ヶ月以上

### 6-2. 発生メカニズム（バイオメカニクス的理解）

**高リスク動作**

- 高速エキセントリック収縮（急激な減速、着地、方向転換）
- 筋が伸張位で最大収縮を強いられる状況
- ハムストリングス、大腿四頭筋、腓腹筋に好発

**組織レベルの脆弱性**

- 筋腱移行部（Myotendinous Junction: MTJ）に応力集中
- 筋線維タイプの移行部（Type I/II境界）
- 既存の瘢痕組織部位（再発リスク高）

### 6-3. リスク因子（エビデンスベース）

**内的因子**

- **既往歴**（最大のリスク因子、オッズ比2〜6倍）
- **加齢**（筋腱の粘弾性低下、サテライト細胞の減少）
- **筋力不足・筋力不均衡**（H/Q比の低下など）
- **柔軟性の極端な低下または過剰**
- **疲労蓄積**

**外的因子**

- ウォームアップ不足
- 不適切なトレーニング負荷（急激な負荷増加）
- 環境要因（低温、硬い地面）

### 6-4. 「硬さ」と筋損傷の関係（誤解の修正）

**誤解**

- 「筋肉が硬い → だから切れる」

**正確な理解**

- 「適応していない組織に、準備なく高負荷を加える → 組織耐性を超える → 損傷」

**補足：柔軟性と傷害リスクの関係**

- 柔軟性が極端に低い場合：リスク増（可動域制限による代償動作）
- 柔軟性が極端に高い場合：リスク増（関節安定性低下、筋による制御範囲の拡大）
- **適度な柔軟性＋筋力＋神経筋制御**が重要

## 7. 傷害予防のための実践的戦略

### 7-1. ウォームアップの科学的根拠

**目的**

- 筋温上昇（2〜3℃の上昇で筋収縮速度・力発揮が向上）
- 神経筋活性化（運動単位の動員パターン最適化）
- 関節可動域の確保
- 心血管系の準備（心拍数・心拍出量の段階的上昇）

**推奨プロトコル（計15〜30分）**

1. **一般的ウォームアップ（5〜10分）**
   - 軽い有酸素運動（ジョギング、サイクリングなど）
   - 目標：心拍数を安静時の60〜70%程度まで上昇、軽く汗ばむ程度

2. **動的ストレッチ（5〜10分）**
   - 関節可動域全体を使った動的動作
   - レッグスイング、股関節回旋、ランジウォーク、アームサークルなど

3. **競技特異的ウォームアップ（5〜10分）**
   - 段階的負荷増加（50% → 70% → 90%の強度）
   - 競技動作の模擬（草野球ならキャッチボール、軽いスイング、短距離ダッシュなど）

### 7-2. 日常的なコンディショニング（週2〜3回以上）

**筋力トレーニング**

- **下肢筋力**：スクワット、ランジ、ヒップスラスト、カーフレイズ
- **体幹安定性**：プランク、サイドプランク、デッドバグ、呼吸と連動した腹圧トレーニング
- **エキセントリックトレーニング**：ノルディックハムストリング、スローテンポスクワット（3〜5秒で下降）

**可動域トレーニング**

- 股関節の多方向可動域確保（屈曲・伸展・内外転・回旋）
- 胸椎回旋可動性の維持
- 足関節の背屈可動域確保（特にふくらはぎの柔軟性）

**神経筋制御トレーニング**

- 片脚立位バランス（目を開けて/閉じて、各30秒）
- 不安定面でのトレーニング（バランスパッド、バランスボードなど）
- プライオメトリクス（段階的導入：両脚ジャンプ → 片脚ジャンプ → 方向転換）

### 7-3. 水分補給の重要性

**脱水と筋損傷リスクの関係**

- 体重の2%以上の脱水 → 筋力・持久力の低下（10〜20%）
- 血液粘性の増加 → 筋血流量の低下
- 電解質バランスの乱れ → 筋痙攣リスク増

**推奨**

- 運動前：体重1kgあたり5〜7ml（2時間前）
- 運動中：15〜20分ごとに150〜250ml
- 運動後：失った体重の150%を補給（発汗による損失を考慮）

## 8. 神経筋制御と防御性収縮

### 8-1. 神経筋制御の基礎

**運動制御の階層構造**

- **脊髄レベル**：反射、筋緊張の基礎制御
- **脳幹レベル**：姿勢制御、バランス
- **大脳皮質レベル**：随意運動、運動学習

**固有受容感覚の役割**

- **筋紡錘**：筋長・筋長変化速度の検出
- **ゴルジ腱器官**：筋張力の検出
- **関節受容器、皮膚受容器**：位置覚、運動覚

### 8-2. 防御性収縮（Protective Muscle Guarding）

**定義**

- 痛みや不安定性に対する神経系の防御反応
- 筋緊張の亢進、運動範囲の制限
- 「神経のブレーキ過剰」状態

**発生メカニズム**

- 侵害受容器の活性化 → α運動ニューロンの興奮性亢進
- 痛み経験による運動パターンの変容（motor engram の変化）
- 予測的な筋活動パターンの変化（feedforward control の異常）

**慢性化のリスク**

- 不使用による筋萎縮・筋力低下
- 運動パターンの固定化 → 代償動作の定着
- 痛みの慢性化（中枢性感作）

**臨床的意義**

- 「筋力が弱い」のではなく「使っていい範囲を神経が忘れている」状態
- 「安全に使われた回数」が神経系の信頼度を決定

### 8-3. 腱・結合組織の適応

**メカノトランスダクション（機械刺激の生物学的変換）**

- 腱への機械的負荷 → 腱細胞（tenocyte）の活性化
- コラーゲン合成の促進、組織リモデリング
- 適応には4〜12週間の継続的負荷が必要

**不使用による退行**

- コラーゲン線維の配列乱れ
- 腱の断面積減少、剛性低下
- 応力耐性の低下（2〜3週間の不使用で顕著）

### 8-4. 神経筋制御の改善戦略

**原則：段階的負荷増加（Graded Exposure）**

**① 低負荷・高頻度の運動**

- 痛みを伴わない範囲（可動域の60〜70%）
- 神経系への「安全」シグナルの蓄積
- 例：小さなヒップヒンジ、ゆっくりした膝曲げ伸ばし、その場歩行

**② アイソメトリック収縮（等尺性収縮）**

- 関節角度を固定した筋収縮
- 腱への負荷、痛み抑制効果（中枢性疼痛抑制機構の活性化）
- 10〜20秒 × 3〜5セット、痛みが増悪しない範囲
- 例：軽い中腰保持、ヒップヒンジ途中で保持、片脚立ち

**③ エキセントリックトレーニング（段階的導入）**

- 伸張性収縮への適応促進
- 筋腱複合体の構造的適応（腱の剛性向上、筋線維の直列サルコメア増加）
- 低負荷から開始、4〜12週間かけて漸増
- 例：ゆっくりしゃがむ（3〜5秒）、ゆっくり脚を下ろす、階段をゆっくり降りる

**④ 運動学習・パターン修正**

- 正しい動作パターンの反復
- フィードバック（視覚、触覚、言語）の活用
- 段階的な速度・負荷の増加

## 9. 統合的ケアプロトコル

### 9-1. ケアの優先順位

**第1優先：循環改善（"流す"）**

- 軽い有酸素運動（ウォーキング、サイクリング）
- リズミカルな関節運動
- 入浴、温熱療法

**第2優先：神経筋制御の再教育（"動かす"）**

- 低負荷での関節運動
- アイソメトリック収縮
- 呼吸と連動した動作

**第3優先：柔軟性の調整（"伸ばす"）**

- 動的ストレッチ（運動前）
- 静的ストレッチ（運動後、日常）
- 筋膜リリース（補助的）

**第4優先：負荷への適応（"使う"）**

- 段階的な筋力トレーニング
- エキセントリックトレーニング
- 競技特異的トレーニング

### 9-2. 実施順序（セッション内）

**運動前**

1. **温める**：軽い有酸素運動（5〜10分）
2. **動かす**：動的ストレッチ、関節可動域運動（5〜10分）
3. **準備する**：競技特異的ウォームアップ、段階的負荷増加（5〜10分）
4. **使う**：本運動

**運動後**

5. **戻す**：クールダウン、軽い有酸素運動（5〜10分）
6. **整える**：静的ストレッチ、筋膜リリース（5〜10分）
7. **回復促進**：入浴、栄養補給

### 9-3. NGパターンと神経生理学的背景

| NGパターン | 神経生理学的問題 | 結果 |
|-----------|----------------|------|
| 硬い → 強く伸ばす | 侵害受容器の活性化 → 防御性収縮の増強 | 筋緊張の悪化、損傷リスク増 |
| 痛い → もっとほぐす | 組織損傷の悪化、炎症の増悪 | 回復遅延、慢性痛化 |
| 動かない → 休み続ける | 不使用性萎縮、神経筋制御の低下 | 再発リスク増、機能低下 |
| 準備なしで全力 | 組織の準備不足、神経筋活性化不足 | 急性損傷リスク増 |

## 10. 特殊集団への応用

### 10-1. 運動習慣のない人（週1回未満の運動）

**リスク要因**

- 筋力・筋持久力の低下
- 神経筋制御の低下（運動単位の動員効率低下）
- 腱・結合組織の適応不足
- 心肺機能の低下

**推奨戦略**

- **日常的な軽い運動習慣の確立**（週2〜3回、10〜15分）
  - ウォーキング、階段昇降、軽いスクワット
- **運動前のウォームアップを長めに**（15〜20分）
- **段階的負荷増加の徹底**（50% → 70% → 90%、各段階5分以上）
- **運動後の入浴・軽いストレッチ**

### 10-2. 週末アスリート（草野球、運動会など）

**リスク要因**

- 平日の運動不足と週末の高負荷のギャップ
- 競技中の興奮による自己モニタリング低下（痛みや疲労の感覚鈍化）
- 加齢による組織の回復力低下

**推奨戦略**

**平日の最低限の運動習慣（週2回、各15分）**

- 下肢筋力維持（スクワット、ランジ）
- 股関節可動域維持（ダイナミックストレッチ）
- 体幹トレーニング（プランク、呼吸エクササイズ）

**競技当日のウォームアップ徹底**

- 試合1時間前から段階的に開始
- 軽いジョギング → 動的ストレッチ → 競技動作の模擬（キャッチボール、軽いスイング）
- 段階的負荷増加を厳守

**試合後のクールダウン**

- 軽いジョギング、ウォーキング（5〜10分）
- 静的ストレッチ（主要筋群、各30秒）
- 入浴（38〜40℃、10〜15分）

### 10-3. 加齢に伴う変化と対策

**加齢による組織変化**

- **筋量減少**（サルコペニア：30歳以降、年0.5〜1%減少）
- **筋線維タイプの変化**（Type II線維の優先的萎縮）
- **腱の粘弾性低下**、コラーゲン架橋の増加
- **神経筋接合部の変性**
- **サテライト細胞の減少** → 修復能力の低下

**対策の重点**

- 筋力トレーニングの継続（週2回以上、中〜高強度）
- タンパク質摂取の確保（体重1kgあたり1.2〜1.6g/日、高齢者では1.6〜2.0g/日）
- エキセントリックトレーニングの導入（腱の適応促進）
- ウォームアップ時間の延長（若年者の1.5〜2倍）

## 11. 再発予防のリハビリテーション原則

### 11-1. 組織修復の時間経過

**急性期（受傷後0〜3日）**

- **炎症期**：好中球・マクロファージの浸潤、出血・浮腫
- **対応**：PEACE & LOVE原則（保護、安静、挙上、圧迫、負荷管理）
- **禁忌**：温熱、マッサージ、ストレッチ

**亜急性期（受傷後3日〜3週間）**

- **増殖期**：線維芽細胞の活性化、コラーゲン合成、肉芽組織形成
- **対応**：段階的な可動域運動、低負荷アイソメトリック
- **目標**：痛みのない可動域の回復、筋萎縮の予防

**慢性期（受傷後3週間以降）**

- **リモデリング期**：コラーゲン線維の配列、瘢痕組織の成熟・強化
- **対応**：エキセントリックトレーニング、競技復帰プログラム
- **目標**：組織の機械的強度の回復、再発予防

### 11-2. 段階的負荷増加の原則（Phase別プログラム）

**Phase 1：痛みのない可動域運動**

- 目標：可動域の回復、循環促進
- 方法：自動運動、他動運動、痛みを伴わない範囲
- 期間：受傷後3〜7日程度

**Phase 2：アイソメトリック収縮**

- 目標：筋力の維持、痛み抑制
- 方法：複数角度でのアイソメトリック（10〜20秒 × 3〜5セット）
- 期間：受傷後1〜2週間

**Phase 3：コンセントリック・エキセントリック収縮**

- 目標：筋力回復、腱の適応
- 方法：低負荷から開始、2週間ごとに10〜20%負荷増加
- 期間：受傷後2〜6週間

**Phase 4：機能的トレーニング**

- 目標：競技動作への適応
- 方法：ジャンプ、方向転換、スプリントの段階的導入
- 期間：受傷後4〜8週間

**Phase 5：競技復帰**

- **基準**：患側/健側の筋力比90%以上、痛みなし、競技動作の完遂
- **方法**：段階的な競技参加（練習 → 部分参加 → 完全復帰）
- **期間**：Grade Iで2〜3週間、Grade IIで6〜12週間

### 11-3. 再発予防の長期戦略

**最大のリスク因子は「既往歴」**

- 再発率：初回損傷後12〜30%（研究により幅あり）
- 再発までの期間：多くは2ヶ月以内
- 再発時の損傷は初回より重症化しやすい

**予防戦略**

- エキセントリックトレーニングの継続（週2回以上、長期的に）
- ウォームアップの習慣化
- 疲労管理（過負荷の回避、十分な休息）
- 柔軟性と筋力のバランス維持
- 競技復帰後も段階的負荷増加を継続

## 12. 実践的推奨事項のまとめ

### 12-1. 運動前の準備（必須）

1. 軽い有酸素運動で体温・筋温を上げる（5〜10分）
2. 動的ストレッチで関節可動域を確保（5〜10分）
3. 段階的に負荷を上げる（50% → 70% → 90%、各5分）
4. 水分補給を忘れない

### 12-2. 運動後のリカバリー

1. クールダウン（軽い有酸素運動、5〜10分）
2. 静的ストレッチ（主要筋群、各30秒）
3. 入浴（38〜40℃、10〜15分、運動後30分以上経過後）
4. 水分・栄養補給（タンパク質20〜30g、糖質1〜1.5g/kg体重）

### 12-3. 日常的な習慣（週2〜3回以上）

1. 筋力トレーニング（下肢・体幹、各10〜15分）
2. 可動域トレーニング（股関節・胸椎、各5〜10分）
3. 軽い有酸素運動（ウォーキングなど、20〜30分）

### 12-4. 避けるべき行動

- 準備運動なしでの全力運動
- 炎症期（受傷後48時間以内）の強圧マッサージ・温熱
- 痛みを伴うストレッチの継続
- 長期間の完全安静（慢性期）
- 急激なトレーニング負荷の増加（週10%以上の負荷増は高リスク）

## 14. 重要な概念の整理

### 14-1. 傷害予防の本質

**誤解**

- 「柔らかくすれば怪我しない」

**正確な理解**

- 「準備された組織を、適切な負荷で使う」

### 14-2. ケアの本質

**誤解**

- 「ケア＝ストレッチ・マッサージ」

**正確な理解**

- 「ケア＝循環改善 + 神経筋制御の最適化 + 組織適応の促進」

### 14-3. 硬さと傷害の関係

**誤解**

- 「硬い筋肉は切れやすい」

**正確な理解**

- 「適応していない組織に、準備なく高負荷を加えると損傷する」

**補足**

- 「硬さが固定された体を、準備なく最大負荷で使うから切れる」
- 柔軟性は「適度」が重要（極端な低下も過剰も高リスク）

### 14-4. 神経反応の重要性

**誤解**

- 「筋力があれば大丈夫」

**正確な理解**

- 「筋力 + 神経筋制御 + 組織適応の三位一体が必要」

## 16. クイックリファレンス

### 16-1. 「たまに運動する人」への安全策

1. **運動前**：15〜20分のウォームアップ（軽い有酸素 → 動的ストレッチ → 段階的負荷増加）
2. **運動中**：水分補給（15〜20分ごと）、段階的に強度を上げる
3. **運動後**：クールダウン → 静的ストレッチ → 入浴
4. **日常**：週2回の軽い運動習慣（スクワット、ウォーキング、ストレッチ）

### 16-2. 筋肉痛が出たときの対応

**軽度のDOMS（日常生活に支障なし）**

- 軽い有酸素運動（アクティブリカバリー）
- 入浴（38〜40℃、10〜15分）
- 軽圧マッサージ、筋膜リリース
- 静的ストレッチ（痛みを伴わない範囲）

**中等度以上のDOMS（動作時痛あり）**

- 強圧マッサージは避ける
- 無理な運動は避け、軽い動き程度に留める
- 必要に応じて鎮痛薬（NSAIDs）、ただし長期使用は避ける
- 3〜5日で軽快しない場合は医療機関受診

**鑑別が必要な症状（肉離れの可能性）**

- 運動中の「ブチッ」という断裂音・感覚
- 著明な腫脹、皮下出血（内出血）
- 荷重困難、歩行困難
- 圧痛が限局している

→ これらがあれば、DOMS ではなく筋挫傷の可能性。医療機関受診を推奨。

### 16-3. ケアの実施順序（再掲）

**優先順位**

1. **流す**（循環改善）
2. **動かす**（神経筋制御）
3. **伸ばす**（柔軟性調整）
4. **使う**（負荷適応）

**NGパターン**

- 伸ばす → 流す（順序が逆）
- 強く伸ばす → もっとほぐす（過剰刺激）
- 休む → 休み続ける（不使用の悪循環）
