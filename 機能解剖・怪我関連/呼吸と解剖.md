# 呼吸の機能解剖と実践ガイド

> **位置づけ**：一般的な健康・運動指導用資料
> **非対象**：診断、治療、疾患管理、個別医療判断
> **基本方針**：呼吸法を万能視せず、目的別の呼吸戦略として整理
> **重要注意**：胸痛、強い息苦しさ、動悸、失神、神経症状、睡眠時無呼吸の疑い、安静時息切れがある場合は、呼吸法ではなく医療機関での確認が前提

# 1. 呼吸の構造的枠組み

呼吸は、
**- 横隔膜（主な吸気筋）**
**- 胸郭（肋骨のかご）**
**- 腹壁（腹部の壁）**
**- 骨盤底筋群（骨盤の底の筋群）**
が協調して起こる運動。

## 1-1. 胸郭

### 構成

* 胸椎（背中側の背骨）
* 肋骨（左右の骨）
* 胸骨（前胸部の骨）
* 肋椎関節（肋骨と背骨の関節）
* 胸肋関節（肋骨と胸骨の関節）

### 機能

* 心臓・肺の保護
* 呼吸時の胸腔容積変化
* 肩甲帯（肩まわり）と脊柱（背骨）の土台
* 体幹回旋・側屈への関与

---

## 1-2. 胸腔と腹腔

| 区画 | 位置     | 主な内容物  | 呼吸時の特徴     |
| -- | ------ | ------ | ---------- |
| 胸腔 | 横隔膜より上 | 肺、心臓   | 吸気で容積増加    |
| 腹腔 | 横隔膜より下 | 消化器系など | 横隔膜下降で内圧変化 |

### 修正ポイント

「胸腔と腹腔を横隔膜が隔てる」という説明は妥当。
ただし、呼吸時には両者が完全に別々に動くのではなく、**横隔膜を介して圧と形が相互に変化する**と説明する方が正確。

---

## 1-3. 横隔膜

### 基本整理

| 項目    | 内容                      |
| ----- | ----------------------- |
| 構造    | ドーム状の横紋筋（意識的にも自律的にも働く筋） |
| 位置    | 胸腔と腹腔の境界                |
| 主機能   | 吸気、腹腔内圧形成、姿勢制御への関与      |
| 神経支配  | 横隔神経 C3〜C5              |
| 吸気時   | 収縮して下降し、胸腔容積を増やす        |
| 安静呼気時 | 弛緩して上方へ戻る               |
| 努力呼気時 | 腹筋群の活動により、横隔膜が押し上げられやすい |

横隔神経はC3〜C5由来で、横隔膜の主要な運動支配を担います。([NCBI][1])

### 注意すべき表現

| 避けたい表現          | 理由           | 推奨表現             |
| --------------- | ------------ | ---------------- |
| 横隔膜を下げたまま吐く     | 呼気の生理と矛盾しやすい | 腹部の張りを保ちながら静かに吐く |
| 横隔膜呼吸ができれば姿勢が治る | 効果の断定が強すぎる   | 姿勢制御に関与する可能性がある  |
| 横隔膜が弱い          | 評価なしでは断定不可   | 横隔膜の動きが観察上乏しい    |

# 2. 呼吸生理の基本

呼吸の本質は、**肺胞（肺の末端構造）で酸素と二酸化炭素を交換すること**。
呼吸指導では、単に「深く吸う」よりも、**呼吸数・換気量・呼気時間・息苦しさの有無**を確認する方が重要。

---

## 2-1. 安静時呼吸の目安

| 項目       |    一般的目安 | 注意                   |
| -------- | -------: | -------------------- |
| 成人安静時呼吸数 | 12〜20回/分 | 体格、年齢、疾患、緊張で変動       |
| 1回換気量    |   約500mL | 安静時成人の概算             |
| 解剖学的死腔   |   約150mL | 気道に残り、ガス交換に直接使われない空気 |
| 肺胞換気量    | 約350mL/回 | 500mL − 150mL の概算    |

成人の安静時呼吸数は一般に12〜20回/分とされ、二酸化炭素濃度の上昇は呼吸を深く速くする主要な刺激。([Merck Manuals][2])

---

## 2-2. 呼吸を促す主な要因

### 主要因

* 血中二酸化炭素増加
* pH低下（酸性方向への変化）
* 低酸素
* 運動負荷
* 情動ストレス
* 疼痛
* 発熱
* 疾患

### 指導上の注意

「息が吸えない」という訴えは、酸素不足だけでなく、**二酸化炭素、過換気、不安、胸郭可動性、姿勢、疾患**の影響を受けます。

そのため、呼吸法だけで判断しない設計が必要。

# 3. 肋骨・胸郭の運動学

胸郭は、前後・左右・上下・背側に広がる**三次元構造**として扱うべき。

---

## 3-1. 肋骨運動の整理

| 部位       | 主な運動イメージ  | 主な容積変化   |
| -------- | --------- | -------- |
| 上位肋骨     | ポンプハンドル運動 | 前後径の拡大   |
| 中〜下位肋骨   | バケツハンドル運動 | 横径の拡大    |
| 第11〜12肋骨 | キャリパー運動   | 側方・後方の開閉 |

「上位肋骨＝第1〜6」「下位肋骨＝第7〜10」と固定しすぎるより、**部位ごとに優位な動きがある**程度の表現が安全。
**個体差、姿勢、胸椎可動性、呼吸努力の強さで変化**。

---

## 3-2. 吸気と呼気の胸郭変化

| 呼吸相  | 胸郭   | 肋骨            | 腹部           |
| ---- | ---- | ------------- | ------------ |
| 吸気   | 拡張   | 挙上・外方拡張       | 前方・側方・後方へ広がる |
| 安静呼気 | 縮小   | 弾性反跳で下降・内方へ戻る | 自然に戻る        |
| 努力呼気 | 強く縮小 | 内肋間筋・腹筋群が関与   | 腹筋群が収縮       |

### 現場キュー

* 「胸の上だけではなく、脇腹と背中にも空気を入れる」
* 「吸うと肋骨が横と後ろに広がる」
* 「吐くと肋骨がやわらかく戻る」
* 「肩を上げて吸わない」

# 4. 呼吸パターンの分類

## 結論

腹式呼吸、胸式呼吸、IAP呼吸法は、**良い・悪いではなく、目的が違う呼吸戦略**

---

## 4-1. 腹部優位呼吸

### 定義

吸気時に腹部の拡張が目立つ呼吸様式。横隔膜下降と腹壁の変位を感じやすい方法。

### 向く目的

* リラックス
* 呼吸の自己認識
* 補助呼吸筋の過活動抑制
* 睡眠前のクールダウン
* 初期導入

### 注意

腹部だけを前に押し出す呼吸は不十分。
理想は、腹部前面だけでなく、**側腹部・背部・下部肋骨にも広がりが出ること**。

---

## 4-2. 胸郭優位呼吸

### 定義

胸郭の拡張が目立つ呼吸様式。

### 正常に使われる場面

* 運動時
* 発声
* 歌唱
* 高強度運動
* 胸郭拡張エクササイズ
* ピラティスやヨガの一部

### 問題になりやすい状態

胸式呼吸そのものではなく、以下の状態が問題。

* 安静時でも肩が上がる
* 胸鎖乳突筋（首前の筋）や斜角筋（首横の筋）が過活動
* 呼吸数が多い
* 下部肋骨が動かない
* 背中側に空気が入らない
* 息苦しさや不安感を伴う

### 推奨表現

> 胸郭優位呼吸は正常な呼吸戦略の一つ。
> ただし、安静時に上胸部と首肩の補助呼吸筋へ過度に依存する場合は、呼吸効率や筋緊張の面で調整対象となる。

---

## 4-3. IAP呼吸・ブレーシング系呼吸

### 定義

IAP（腹腔内圧）を適度に高め、体幹の剛性（ぐらつきにくさ）を作る呼吸戦略。

### 重要な修正

「吸っても吐いてもお腹を膨らませる」ではなく、以下の表現が適切。

> 吸気で腹部・側腹部・背部に広がりを作り、呼気中も腹部を完全に抜かず、必要な範囲で内圧を保つ。

Hodges & Gandeviaの研究では、横隔膜と腹筋群の共活動が持続的なIAP上昇に関係し、IAPは四肢運動時の反力に応じて増加。([PubMed][3])

### 向く目的

* スクワット
* デッドリフト
* ヒンジ動作
* プレス動作
* 方向転換
* 体幹保持
* 片脚支持
* 外乱に耐える動作

### 注意

IAPは高ければ高いほど良いわけではなく。目的は**課題に必要な分だけ圧を作ること**

---

# 5. 自律神経と呼吸

自律神経への介入では、「腹式か胸式か」よりも、下記が重要
**呼吸数を落とすこと**
**呼気をやや長くすること**
**過換気を避けること**

---

## 5-1. 根拠のある言い方

ゆっくりした随意呼吸は、HRV（心拍変動：自律神経活動の指標）に影響し、副交感神経活動や自律神経調整に関与する可能性がある。
ただし、効果は呼吸頻度、介入期間、対象者、評価指標により異なる。
2022年のシステマティックレビュー・メタ解析でも、随意的なスロー breathing とHRVの関係が検討されている。([PubMed][4])

---

## 5-2. 実践設定

| 目的        |   吸気 |    呼気 | 実施時間 | 注意          |
| --------- | ---: | ----: | ---: | ----------- |
| 初心者のリラックス | 3〜4秒 |  4〜6秒 | 1〜3分 | 苦しければ短縮     |
| 睡眠前       | 3〜5秒 | 6〜10秒 | 2〜5分 | 20秒呼気は上級者向け |
| 緊張緩和      |   3秒 |  5〜6秒 | 1〜2分 | 大きく吸いすぎない   |
| 過換気傾向     |  小さめ | 長すぎない |  短時間 | 息止めを避ける     |

---

## 5-3. 20秒呼気の扱い

### 評価

20秒かけて吐く呼吸は、一般向けの初期設定としては長すぎる可能性がある。

### 修正案

* 初心者：4〜6秒呼気
* 慣れた人：6〜10秒呼気
* 上級者：10〜20秒呼気
* 息苦しさ、めまい、しびれ、不安増大があれば中止

# 6. 姿勢と呼吸

姿勢は呼吸に影響するが、「姿勢を正せば呼吸が治る」「呼吸を整えれば姿勢が治る」という単純な関係ではない。
現場では、**胸郭・骨盤・頭頸部の位置を調整して、呼吸しやすい条件を作る**ことが重要。

---

## 6-1. 呼吸を妨げやすい姿勢

| 所見    | 起こりやすい呼吸変化         | 指導方針            |
| ----- | ------------------ | --------------- |
| 頭部前方位 | 首の補助呼吸筋が働きやすい      | 顎を軽く引き、後頭部を高く保つ |
| 過度な猫背 | 胸郭前面・背側の拡張低下       | 胸椎伸展・回旋を出す      |
| リブフレア | 呼気時に肋骨が閉じにくい       | 肋骨を骨盤上へ戻す       |
| 反り腰   | 腹部前面が伸張され、腹圧制御が不安定 | 骨盤と肋骨の位置調整      |
| 肩すくめ  | 補助呼吸筋依存            | 肩を耳から遠ざける       |

---

## 6-2. ZOAの扱い

### 定義

ZOA（Zone of Apposition）は、横隔膜の筋線維が胸郭内面に接している領域。

### 指導上の意味

* 下位肋骨が過度に開きっぱなしでない
* 胸郭と骨盤が大きくズレていない
* 呼気で肋骨が自然に戻る
* 吸気で腹部・側腹部・背部に広がりがある

### 注意

「ZOAが整えば横隔膜が初めて機能する」という表現は強すぎる。
正しくは、**ZOAは横隔膜の力学的効率に関係する重要な概念**

# 7. 評価プロセス

呼吸評価は、主観的な「できている/できていない」ではなく、**観察・触診・症状・課題動作**を組み合わせて行うべき。

---

## 7-1. 基本評価

| 項目    | 見ること           | 注意サイン           |
| ----- | -------------- | --------------- |
| 呼吸数   | 1分間の回数         | 安静時20回/分超、極端な遅さ |
| 呼吸の深さ | 浅いか、過剰に大きいか    | 過換気、息苦しさ        |
| 肩の動き  | 吸気で肩が上がるか      | 補助呼吸筋依存         |
| 下部肋骨  | 横・後ろへ広がるか      | 一方向のみの拡張        |
| 腹部    | 前だけでなく側腹部も広がるか | 腹部前方だけ突出        |
| 呼気    | 肋骨が戻るか         | リブフレア残存         |
| 頸部    | 斜角筋・胸鎖乳突筋の緊張   | 首こり、顎上がり        |
| 症状    | めまい、しびれ、胸痛     | 中止・医療確認         |

---

## 7-2. 評価時の言い方

| 避ける表現      | 推奨表現                     |
| ---------- | ------------------------ |
| 横隔膜が使えていない | 横隔膜由来と思われる下部胸郭・腹部の動きが少ない |
| 呼吸が悪い      | 今の課題に対して効率が落ちている可能性      |
| 胸式だからダメ    | 上胸部と首肩に偏っている             |
| 腹圧が弱い      | 動作中に体幹の張りを保ちにくい          |

# 8. 介入プロセス

呼吸介入は、**ポジション設定 → 感覚入力 → 呼吸制御 → 動作統合**の順に進めると安全で再現性が高くなる。

---

## Phase 1：ポジション設定

### 目的

* 過緊張の抑制
* 呼吸しやすい胸郭・骨盤配置
* 補助呼吸筋依存の軽減

### 推奨肢位

| 肢位         | 向く対象       | 目的         |
| ---------- | ---------- | ---------- |
| 仰臥位・膝立て    | 初心者、緊張が強い人 | 脱力と腹部呼吸感覚  |
| 90/90ポジション | 反り腰、リブフレア  | 肋骨と骨盤の配置調整 |
| 側臥位        | 左右差、背側拡張   | 片側胸郭の感覚入力  |
| 四つ這い       | 背側呼吸、体幹連動  | 胸椎・肋骨可動性   |
| 座位         | 日常動作への接続   | 実用性        |

◆ 90/90ポジションの作り方
1. 仰向けに寝て、両足を壁（または椅子）に乗せる
2. 股関節と膝関節がそれぞれ「90度」になるように調整
3. 骨盤を軽く後傾（床に押し付けるイメージ）させ、腰と床の隙間を埋める。
   1. 吐くのに伴い、開いた肋骨が下がり、お腹が薄くなっていくのを手で感じる。
   2. 息を完全に吐ききった状態で2〜3秒キープし、再び吸うことを繰り返す。

※ 背骨の柔軟性： 反り腰で背中や腰の筋肉が硬くなっていると、肋骨を正しく閉めることができない。背骨の屈曲ワークを取り入れるとより効果的。
※ 日常の意識： 立っている時も、この「肋骨と骨盤が向かい合っている感覚」を意識し、みぞおちが前に突き出ないよう姿勢を保つ

---

## Phase 2：感覚入力

### 方法

* 手を下部肋骨に置く
* 手を側腹部に置く
* 背中側にタオルや床の接触を感じる
* 軽い重りを腹部に置く
* 鏡で肩の挙上を確認する

### 注意

重りを使う場合は軽量から開始。
息苦しさ、腹部圧迫感、不安感が出る場合は中止。

---

## Phase 3：呼吸制御

### リラックス目的

* 鼻から3〜4秒吸う
* 口または鼻から4〜6秒吐く
* 1〜3分
* 7〜8割以下の呼吸量

### 胸郭可動性目的

* 吸気で肋骨の横・背側拡張
* 呼気で肋骨が自然に閉じる
* 背骨の屈曲・伸展・回旋と組み合わせる

### IAP目的

* 吸気で腹部・側腹部・背部に広がり
* 呼気中も腹部を完全に抜かない
* 息を止めず、必要に応じて短い呼気を使う

---

## Phase 4：動作統合

| 種目    | 呼吸の狙い                |
| ----- | -------------------- |
| デッドバグ | 腹圧を保ちながら四肢を動かす       |
| スクワット | 下降前に圧を作り、挙上中に過剰脱力しない |
| ヒンジ   | 腰椎過伸展を避け、腹部・背部の張りを保つ |
| プレス   | 息こらえ過多を避け、体幹剛性を保つ    |
| ランジ   | 片脚支持で呼吸が止まらないようにする   |

---

# 9. 実践メニュー

## 9-1. リラックス用：スロー呼吸

### 目的

* 緊張低下
* 呼吸数低下
* 入眠前の切り替え
* 自律神経調整の補助

### 手順

1. 椅子または仰向け
2. 首・肩の力を抜く
3. 鼻から3〜4秒吸う
4. 口または鼻から4〜6秒吐く
5. 1〜3分実施
6. 慣れたら呼気を6〜10秒へ延長

### 中止基準

* めまい
* 手足のしびれ
* 強い息苦しさ
* 不安感の増大
* 胸部不快感

---

## 9-2. 胸郭可動性用：背中に吸う呼吸

### 目的

* 背側胸郭の拡張
* 肋骨可動性の改善
* 肩すくめ呼吸の抑制

### 手順

1. 四つ這いまたはチャイルドポーズ
2. 背中を軽く丸める
3. 肩をすくめない
4. 鼻から吸い、背中側が広がる感覚を探す
5. 長く吐き、肋骨が戻る感覚を確認
6. 5〜8呼吸

### キュー

* 「背中に空気を入れる」
* 「肩ではなく、肋骨を広げる」
* 「吐くと背中と肋骨がやわらかく戻る」

---

## 9-3. IAP用：360°ブレーシング呼吸

### 目的

* 腹腔内圧形成
* 体幹安定
* トレーニング前の準備

### 手順

1. 座位または立位
2. 両手を下部肋骨〜側腹部へ置く
3. 鼻から吸い、腹部・側腹部・背部に軽い広がりを作る
4. 口から短く静かに吐きながら、胴体の張りを残す
5. 3〜5呼吸
6. スクワットやヒンジへ接続

### NG

* 息を止め続ける
* 肩が上がる
* 腰を反る
* 腹部を固めすぎる
* 顔や首に力が入る

---

## 9-4. デスクワーク後の3分リセット

### 手順

1. 椅子に座る
2. 足裏を床へ置く
3. 肋骨を骨盤の上に置く
4. 肩を耳から遠ざける
5. 3秒吸う
6. 5〜6秒吐く
7. 5呼吸
8. 胸椎の丸める・反らす動きを5回
9. 左右の胸椎回旋を各3回

---

## 9-5. トレーニング前の呼吸準備

### 手順

1. 360°ブレーシング呼吸を3〜5呼吸
2. デッドバグを左右3〜5回
3. ヒンジ動作を5回
4. スクワット動作を5回
5. メイン種目へ移行

### 狙い

* 呼吸の停止癖を減らす
* 腹圧形成を確認する
* 胸郭と骨盤の位置を整える
* 出力前の過緊張を抑える

# 11. 安全管理

呼吸法は低リスクに見えますが、過換気、息こらえ、過剰な腹圧、循環器負担を起こす場合があります。

---

## 11-1. 中止・調整基準

以下が出た場合は中止または短縮。

* めまい
* 胸痛
* 強い息苦しさ
* 動悸
* 手足のしびれ
* 頭痛悪化
* 吐き気
* 不安感の増大
* 視界の異常
* 失神感

---

## 11-2. 注意が必要な対象

| 対象         | 注意点           |
| ---------- | ------------- |
| 高血圧        | 強い息こらえ・怒責を避ける |
| 心疾患リスク     | 高腹圧・バルサルバに注意  |
| 妊娠中・産後     | 過剰腹圧・骨盤底負荷に注意 |
| 尿失禁・骨盤臓器脱  | 強いブレーシングに注意   |
| ヘルニア       | 腹圧上昇に注意       |
| パニック・過換気傾向 | 深く吸いすぎない      |
| 呼吸器疾患      | 医療者の確認が前提     |
| 睡眠時無呼吸疑い   | 呼吸法で代替しない     |

---

## 11-3. 怒責・バルサルバの扱い

### 定義

怒責・バルサルバ法は、息を止めて強く力むことで腹腔内圧を高める方法。

### 利点

* 高重量挙上で体幹剛性を作りやすい
* 最大筋力発揮で使われることがある

### 注意

* 血圧上昇
* めまい
* 循環器負担
* 顔面・頸部の過緊張
* 対象者によって不適切

# 15. 最終結論

この再構成では、呼吸を「正しい呼吸法」ではなく、**目的別に選ぶ運動・生理学的ツール**として整理しています。

最も重要な変更点は以下。

* 腹式呼吸を絶対視しない
* 胸式呼吸を悪者にしない
* IAPを高ければ良いものとして扱わない
* 横隔膜を「下げたまま吐く」と言わない
* 自律神経への効果を断定しない
* 呼吸法の中止基準と注意対象を明確にする
* 指導キューを目的別に分ける

この形であれば、一般向け資料、パーソナルトレーニング現場、スポーツ現場の基礎教材として、かなり安全性と専門性の高い構成になります。
