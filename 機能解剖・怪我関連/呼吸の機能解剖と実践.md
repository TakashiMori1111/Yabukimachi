# 呼吸の機能解剖と実践ガイド

> **対象**：一般的健康増進・機能改善（医療行為・診断の代替ではない）
> **目的**：呼吸機能の最適化、自律神経調整、姿勢改善、体幹安定性の獲得

---

## 1. 呼吸の構造的枠組み

### 1-1. 胸郭（Thoracic Cage）

- **構成**：胸椎（後）・肋骨（側）・胸骨（前）による籠（かご）状構造
- **機能**：心臓と肺の保護、呼吸運動の支点

### 1-2. 胸腔と腹腔

| 区画 | 位置 | 内容物 |
|:-----|:-----|:-------|
| **胸腔** | 横隔膜より上 | 肺・心臓 |
| **腹腔** | 横隔膜より下 | 消化器系（筋性の壁＝腹筋群で囲まれる） |

- **境界**：**横隔膜**が両者を隔てる

### 1-3. 横隔膜（Diaphragm）― 呼吸の主役

| 項目 | 内容 |
|:-----|:-----|
| **構造** | ドーム状の横紋筋（自分の意思でも動かせる筋肉）。中心腱と筋部で構成 |
| **付着部** | 胸骨剣状突起、下位肋骨（第7〜12）、腰椎（L1〜L3） |
| **神経支配** | 横隔神経（C3〜C5）|
| **吸気時** | 収縮 → ドームが**下降・平坦化** → 胸腔容積↑・腹腔内圧↑ |
| **呼気時** | 弛緩 → ドームが**上昇・復元** → 胸腔容積↓（受動的） |

```
安静時          吸気時
   ∩              ―
  / \     →     /   \
 胸腔           胸腔拡大
  腹腔           腹腔圧迫
```

> **補足**：1日に約2万〜2万5千回の呼吸のほとんどを横隔膜が担っている。現代人は胸上部だけを使う浅い呼吸（胸式呼吸）になりがちだが、意識的な横隔膜呼吸を取り入れることでリラックス効果が得られる

## 2. 肋骨の運動学（Kinematics）

### 2-1. 3つの動きパターン

| 肋骨の部位 | 動きの名称 | 働き | イメージ |
|:-----------|:-----------|:-----|:---------|
| **上位肋骨（第1〜6）** | ポンプハンドル運動 | 前後径の拡大 | 胸骨が前上方に持ち上がる（井戸のポンプのような動き） |
| **下位肋骨（第7〜10）** | バケツハンドル運動 | 横径の拡大 | 肋骨が左右に広がる（バケツの持ち手が上がる動き） |
| **浮遊肋（第11〜12）** | キャリパー運動 | 開閉運動 | 横隔膜の付着部として重要 |

```
ポンプハンドル運動（上位）      バケツハンドル運動（下位）
      ↑ 胸骨が上がる                   広がる
     ／                            ←  ∩  →
    ／                                │
   ○ 肋椎関節を軸に回転          肋椎関節を軸に
```

### 2-2. 用語の定義

| 用語 | 意味 | タイミング |
|:-----|:-----|:-----------|
| **外旋** | 肋骨弓が外へ開く方向（胸郭が横に広がる） | 吸気時 |
| **内旋** | 肋骨弓が内へ閉じる方向（胸郭がしぼむ） | 呼気時 |

## 3. 吸気と呼気のメカニズム

### 3-1. 吸気（息を吸う）

| 要素 | 動き |
|:-----|:-----|
| **横隔膜** | 収縮 → ドーム下降 → 胸腔容積↑ |
| **肋骨** | 外旋・挙上 → 胸郭が前後・左右に拡大 |
| **腹部** | 臓器が押し下げられ膨らむ |
| **肺** | 胸腔内の陰圧が高まり、空気が流入 |

- **関与する筋肉**：横隔膜（主）、外肋間筋、斜角筋、胸鎖乳突筋（補助）

### 3-2. 呼気（息を吐く）

| 要素 | 安静呼気 | 努力呼気（強く吐く） |
|:-----|:---------|:---------------------|
| **横隔膜** | 弛緩 → ドーム上昇（受動的） | 弛緩＋腹筋で押し上げ |
| **肋骨** | 下降・内旋（弾性反跳で受動的） | 下降・内旋の強調（能動的） |
| **腹部** | 元に戻る | 腹筋群の収縮で積極的に凹む |

- **安静呼気のポイント**：横隔膜の弛緩と肺・胸郭の弾性反跳（ゴムが元に戻る力）で起こるため、ほぼ筋力を使わない
- **努力呼気の筋肉**：内肋間筋（肋骨を引き下げる）、腹直筋・腹斜筋・腹横筋（腹腔内圧を高め横隔膜を押し上げる）

### 3-3. 比較まとめ

| 項目 | 吸気 | 呼気（安静時） | 呼気（努力時） |
|:-----|:-----|:---------------|:---------------|
| **横隔膜** | 収縮 → 下降 | 弛緩 → 上昇 | 弛緩＋腹筋で押し上げ |
| **肋骨** | 挙上・拡大 | 下降・縮小（受動的） | 下降・縮小（能動的） |
| **外肋間筋** | 収縮 | 弛緩 | 弛緩 |
| **内肋間筋** | 弛緩 | 弛緩 | 収縮 |
| **腹筋群** | 弛緩 | 弛緩 | 収縮 |
| **胸腔容積** | 増大 | 減少 | 減少（より強力） |

## 4. 呼吸パターンの分類

### 4-1. 腹式呼吸（横隔膜優位）

| 項目 | 内容 |
|:-----|:-----|
| **メカニズム** | 横隔膜の上下運動が主導。吸気時に腹壁が拡張 |
| **神経系** | 副交感神経優位（リラックス・安静時） |
| **メリット** | 換気効率の向上（肺底部への送気）、腹腔内圧（IAP）上昇による体幹安定（腰椎保護）、内臓への物理的マッサージ効果（腸の蠕動運動促進） |

### 4-2. 胸式呼吸（胸郭・補助筋優位）

| 項目 | 内容 |
|:-----|:-----|
| **メカニズム** | 外肋間筋・呼吸補助筋による胸郭の挙上・拡張 |
| **神経系** | 交感神経優位（活動時・緊張時・緊急時） |
| **常態化のリスク** | 換気効率の低下（死腔換気＝ガス交換に使われない無駄な空気が増加）、首・肩周りの筋緊張亢進（いわゆる肩こり）、ガス交換効率の低下 |

### 4-3. 努力呼吸と補助呼吸筋

- **関与筋**：胸鎖乳突筋、斜角筋群、僧帽筋上部、大胸筋
- **臨床的意義**：安静時にこれらの筋が活動している場合、呼吸機能不全や過度なストレス状態を示唆

## 5. 姿勢と呼吸の相互関係

### 5-1. 不良姿勢による悪循環

```
猫背・頭部前方位（FHP）
  ↓
胸郭の可動性制限（つぶれた状態）→ 横隔膜の下降制限
  ↓
頸部・肩甲帯の補助筋を過剰動員（「肩で息をする」状態）
  ↓
慢性的な首・肩こり / 浅い呼吸 / 交感神経過活動 / 不安感の増大
```

### 5-2. ZOA（Zone of Apposition）― 理想的なアライメント

- **定義**：横隔膜と腹筋群が対向するエリア
- **条件**：肋骨が浮き上がらず（リブフレアなし）、骨盤に対してほぼ平行な状態
- **意義**：このポジションで初めて横隔膜が「呼吸筋」兼「姿勢安定筋」として機能する

> **リブフレア**：肋骨の下端が外側に浮き上がった状態。横隔膜の機能低下と腹筋群の弱化を示すサイン

## 6. 評価と介入プロセス

### 6-1. 評価ポイント

#### 視診

| チェック項目 | 正常 | 異常サイン |
|:-------------|:-----|:-----------|
| **主動部位** | お腹・下部肋骨が動く | 胸・肩が先に動く |
| **肋骨の動き** | 吸気で外旋（広がる）、呼気で内旋（閉じる） | 動きが乏しい・左右差がある |
| **呼吸数** | 12〜16回/分 | 20回/分以上（頻呼吸） |
| **顎の向き** | 自然な位置 | 上がっている・苦しそう → 枕等で調整 |

#### 触診

- 斜角筋・胸鎖乳突筋の緊張度（硬ければ補助筋の過活動）
- 下部肋骨の拡張感（左右均等に広がるか）

### 6-2. 段階的介入プロセス

「できない」原因は能力欠如ではなく、**条件設定の不備**が大半。

#### Phase 1：姿勢条件の最適化（Positioning）

- **目的**：脱力と横隔膜の可動域確保
- **設定**：
  - 仰臥位（仰向け）
  - 股関節・膝関節屈曲（膝を立てる or 台に乗せる）
- **膝を曲げる理由**：骨盤前傾（反り腰）を抑制し、腹壁の緊張を緩和 → 横隔膜が動きやすくなる
- **顎が上がる人**：枕やタオルで頭部を支え、頸部の緊張を取り除く

#### Phase 2：感覚入力の強化（Sensory Input）

- **目的**：固有受容感覚（体の位置や動きを感じ取る感覚）へのフィードバック
- **手法**：
  - **触覚**：自分の手を腹部（みぞおち・下腹部）と下部肋骨に置く
  - **抵抗**：腹部に重り（サンドバッグ等）やタオルを置き、吸気で押し返す感覚を掴む

#### Phase 3：動作学習と統合（Motor Learning）

- **目的**：意識的なコントロールと日常生活への汎化
- **手法**：
  - **スローブレス**：吸気4秒・呼気6秒などのリズム統制
  - **IAPの意識**：吸気で高めた腹腔内圧を維持しつつ四肢を動かす（デッドバグ等への展開）

## 7. 指導キューイング（一般向け言語化の技術）

専門用語を避け、イメージ（比喩）と体性感覚に訴える表現を選択。

### 7-1. 横隔膜の動き ― 注射器のイメージ

| 場面 | キュー |
|:-----|:-------|
| **吸気** | 「注射器のピストンが下がるように、お腹の奥の膜が下がって空気が入ってくる」 |
| **呼気** | 「力が抜けて、膜がフワッと上に戻っていく」 |

### 7-2. 腹部・肋骨の動き ― 風船のイメージ

| 場面 | キュー |
|:-----|:-------|
| **吸気** | 「肩は上げず、お腹の中に風船があって、前・横・背中に膨らむイメージ」 |
| **呼気** | 「風船の空気が抜けて、おへそが背骨にくっつく」 |

### 7-3. NG動作への修正キュー

| NG動作 | 修正キュー |
|:-------|:-----------|
| 肩が上がる | 「肩を耳から遠ざける」「首を長く保つ」 |
| 腰が反る | 「背中を床に溶け込ませる」「腰の隙間を埋める」 |

### 7-4. 呼吸指導の基本原則

- **とにかくゆっくり吐き切る**：副交感神経の活性化が主目的 → 「細く長く〜」の声かけ
- **吸気は自然に任せる**：吐き切れば自然と吸える。吸う方を頑張らせない

## 8. 自律訓練法（Autogenic Training）

呼吸法と併用することで、心身の弛緩効果を最大化する技法。

### 8-1. 概要

- **考案者**：ドイツの精神科医シュルツ（J.H. Schultz）
- **機序**：自己暗示による受動的注意集中 → 交感神経活動を抑制 → 副交感神経優位へ

### 8-2. 基本公式（簡易版）

| 順序 | 公式 | 意味 |
|:-----|:-----|:-----|
| 0 | 背景公式 | 「気持ちがとても落ち着いている」 |
| 1 | 重感 | 「手足が重たい」（筋弛緩の感覚） |
| 2 | 温感 | 「手足が温かい」（末梢血管拡張の感覚） |
| 3 | 呼吸調整 | 「呼吸が楽になっている」 |

### 8-3. 実施上の注意

- **受動的注意**：無理に感じようとせず、「ただそうである」と待つ姿勢
- **消去動作**：終了時は必ず伸びや屈伸を行い、覚醒状態へ戻す（ふらつき防止）
