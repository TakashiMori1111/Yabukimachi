## 1. 背景・会社の状況

### 打ち合わせの経緯

- 前年度（2026年3月）に矢吹町の企業向けヘルスケアを1回実施済み（参加者約5名）
- 1回のみでは姿勢改善の実感が難しいとの声があり、**3〜4ヶ月かけて複数回実施**したい意向
- 前回参加者と同じメンバーでの継続を想定

### 会社・勤務の特徴

- 小規模企業。矢吹に移転してきたばかりで、**福利厚生・健康プログラムは未整備**（ラジオ体操等もなし）
- 作業形態：
  - **男性の約8割は立ち仕事**
  - 仕上げ・顕微鏡作業は**座り仕事が中心**（集中作業のため首・肩の負担あり）
  - 指導時は立ち、実作業は座りが多い
- 納品期限：**毎日16時まで**に持ち込みが必要な案件あり。平日は生産優先のため**就業時間内の実施は厳しい**
- 残業は増加傾向。就業時間内対応は**17:30まで**可能だが、納品・打ち合わせで17時前後までかかることも

### 健康への取り組み状況

- 健康診断以外の定期的な取り組みなし
- 社長が健康意識を高めたい意向。**姿勢が客観的にわかる測定**への関心が高い（前回、本人も姿勢の歪みを実感）
- 前回、腰痛で休んでいた女性社員はリハビリ後回復。自宅でできる運動への関心あり

---

## 2. 実施方針（双方の合意方向）

### 全体の進め方

1. **複数回連続プラン**（3〜4ヶ月・6回前後を目安）
2. **初回・最終回**に測定（InBody・AI姿勢分析）。中間回は体操のみ
3. 姿勢変化の把握には、**同じ服装・同じ場所・同じ環境**での測定が望ましい
4. 体組成の数値変化は生活習慣の影響も大きいため、**姿勢・体感・意識の変化**も重視

### 人数・グループ編成

- 前回同様、**全員一斉（約5名）**での実施を希望
- 社長意向：班に分けると片方の工程が滞るため、**1回で全員参加**が望ましい（社長も参加）

### 日程・時間帯

| 項目 | 内容 |
|---|---|
| 曜日 | **土曜日**（第1・第3週）。第2・第4週は休み |
| 開始時期 | 早ければ早い方がよい（8月お盆前後まで調整可能） |
| 頻度 | **2週に1回**または**月1回**。最初は様子見で1ヶ月分から開始し、継続可否を判断 |
| 時間帯 | 午前・午後どちらでも可。**時間が決まっていれば**対応しやすい |
| 集合 | **9時集合**想定（朝礼後9時現場スタートのため）。通勤を会場直行でも可 |
| 所要時間 | 測定込み最大1時間以内。測定なしは内容により短縮可。**50分じっくり**希望の声あり |
| 平日 | 原則厳しい。終了17:30までなら相談の余地あり |

### 実施場所

- 前回：工場内（狭く、社長も参加で手狭だった）
- 今年度候補：
  - **kokottoのマルチルーム**
  - **文化センター**等
- 人数が少ないため移動可能。ゆったり50分確保したい意向
- 使用料は会社負担も検討していたが、**町事業のため使用料は不要**

## 3. 現場の事情・実施上の論点

### 労働時間との兼ね合い

- 就業時間を削るデメリットは認識済み
- 社長は実施に前向き。従業員も「やってもいいかな」程度の反応
- 大企業のようなラジオ体操等の慣習がないため、**新しい取り組みとして始めやすい**タイミング

### 個別の体調・不調への対応

- 前回、腰痛の女性社員は疲労骨折から回復。無理のない範囲での参加を希望
- リハビリは15〜20分程度で自宅運動を渡されるのみ。トレーナー指導への期待あり

### 継続・セルフケア

- 前回教えた自宅運動で改善が見られた事例あり
- YouTube等の動画だけでは続かない。トレーナーが来て教える形式への評価が高い
- 継続にはセルフケア習慣の定着が重要。矢吹スポーツクラブ等の運動教室案内も検討

### 広報・SNS

- 町HP・SNSへの掲載：**OK**（顔出しなし・企業名は掲載可）
- 採用・福利厚生のアピールにも活用したい意向（社長）
