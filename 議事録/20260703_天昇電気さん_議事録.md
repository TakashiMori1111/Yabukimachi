## 要旨

経営懇話会での体験を経て趣旨理解済み。
実施に前向きではあるが、実際に価値があるかどうか、社内などからの色々な見え方があるかもしれなくて、始めること・継続することへのハードルがある。
主な懸案事項は**実施時間**で、「なるべく短く、月1の1時間は厳しい」という意向。協議の結果、**複数回連続プラン**で進めることで一旦は合意。
初回は測定込み約50〜60分、2回目以降は測定なしで15〜20分程度の体操中心とする。
日程は武藤工場長よりメールにて候補日を連絡いただく（7月中を目標）。

---

## 議事内容

### 1. 天昇電気さんの現状

- 経営懇話会でプログラムを体感済みのため、趣旨は把握している
- 健康への取り組み：ラジオ体操（毎日継続）、年1回程度のウェブセミナー視聴、健康診断、ストレスチェック
- ラジオ体操は継続しているが、**意味を理解して取り組めているかが課題**（社長自身も認識している）
- 従業員数：40名 / 工場常駐：約30名
- 実施できる場合の想定人数：**10〜15名**（生産ライン離れられない人・夜勤シフトの方を除く）

### 2. プログラム内容の説明・質疑

- 資料4・5番（実施条件・プログラム構成）を中心に説明
- InBodyによる体組成測定・AI姿勢分析「シセイカルテ」の「見える化」に関心あり
- 「自分では上がっていると思っていても実際には上がっていない」など、**客観的なフィードバックへの期待**が大きい
- 立ち仕事従事者が多く、腰痛・膝痛・肩こりなど個別の悩みを持つ人がいる
- 測定数値の変化については、生活習慣も影響するため過度な期待は持たせず、**「今の状態の見える化」として位置づける**ことで共通認識

### 3. 実施形式・条件の確認

| 項目 | 内容 |
|---|---|
| プラン | 複数回連続プラン |
| 初回 | 測定（InBody・姿勢分析）＋体操：約50〜60分 |
| 2〜最終回前 | 体操のみ：15〜20分 |
| 最終回 | 測定＋アンケート（ビフォーアフター確認） |
| 参加人数 | 10〜15名（1グループ） |
| 実施場所 | 会議室・休憩スペース（机を移動してスペース確保可） |
| 服装 | 作業着のままで可 |
| 実施時間帯 | 就業時間内（会社として削る覚悟あり、頻度次第） |

- 月1回の1時間はしんどいという意向 → 頻度・時間はフレキシブルに調整
- 参考事例として**山形印刷さん**（昨年1回→今年度は安全委員会の時間を使って月1回実施中）を共有

### 4. PR・情報発信

- 町ホームページ・SNSへの掲載：**OK**（顔出しなし、後ろ姿・会社名は掲載可）

### 5. 地域展開の可能性（話題として）

- 工場長より「地域の方も巻き込んでやるのもいいのでは」という提案あり
- 会社のグラウンドや広いスペースも提供可能との意向
- ただし本社へのお伺いが必要なため、まずは社内での実施が現実的

### 6. 健康経営の文脈・背景共有

- 国の「健康経営」施策・健康増進法の観点からも正当性がある取り組み
- 腰痛・肩こりによる生産性低下は数百億単位の調査データあり
- 田村工業さん（前日訪問）では労災への懸念から参加検討中、という情報も共有
