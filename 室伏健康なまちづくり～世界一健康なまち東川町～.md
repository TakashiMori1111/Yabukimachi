# 「国民のライフパフォーマンスの向上に向けて スポーツを通じた健康なまちづくり～世界一健康なまち東川町～」

### 1. プロジェクトの背景と目的
この取り組みは、スポーツ庁が推進する「スポーツ・イン・ライフ推進プロジェクト」の一環であり、トップアスリートの知見を一般国民に還元する「2020レガシー」の創出を目指しています。

*   **研究テーマ**: 「運動機能低下に対する地域における効果的な運動療法のあり方に関する研究」。
*   **ライフパフォーマンスの定義**: 怪我や病気がない状態を目指すだけでなく、何があってもすぐに回復する「レジリエンス（回復力）」や、仕事の生産性が高い状態、前向きに生活できる能力を指します。
*   **コンセプト**: 「本物を身近に」「ハイパフォーマンスをライフパフォーマンスに」。

### 2. 運動の質的アプローチ（コンディショニングの概念）
運動は単なる「体力作り」ではなく、食事と同じように種類と目的を分ける必要があると強調されています。

*   **運動の4要素**:
    1.  **筋骨格系・神経系**: 筋肉、関節、モーターコントロール（体の使い方）。
    2.  **呼吸器・循環器**: 心肺機能、持久力。
    3.  **代謝系**: エネルギー消費、内臓機能。
    4.  **メンタル**: 精神的健康。
*   **スキルとフィジカルの違い**:
    *   **フィジカル**: 筋力、柔軟性、持久力。獲得には長年の積み重ねが必要。
    *   **スキル（モーターコントロール）**: 体の動かし方の技術。正しい考え方や工夫により、短期間で獲得が可能。

### 3. 東川町における実証研究の成果
早稲田大学の金岡教授らによって実施された、3ヶ月間の運動介入プログラムの結果です。

*   **参加者データ**: 76名（男性20名、女性56名）。平均年齢51歳（40代女性が最多）。
*   **継続率（乾燥率）**: **88%**（非常に高い数字）。
*   **主な数値的改善データ**:
    *   **労働生産性（プレゼンティズム）**: プログラム前 **78%** → 終了後 **85%** へ向上（統計学的に有意）。
    *   **腰痛の改善**: 腰痛ありと回答した人が22名から **15名** に減少。痛みの程度（10段階評価）は平均 **3.6** → **1.6** へ半減。
    *   **身体機能の向上**:
        *   **骨格可動域**: 自分で骨盤を動かす能力（可動域）が有意に拡大。
        *   **バランス能力**: 16点 → **20点** に向上。
        *   **セルフチェック（KOJIアウェアネス）**: 50点満点中、平均 **32.8点** → **38点** へ向上。
*   **特筆すべき点**: 3ヶ月で「柔軟性（筋肉の柔らかさ）」に変化はなかったが、「自分で体を動かす能力（モビリティ）」は明確に高まった。

### 4. 「東川町モデル」の展開と実績
株式会社R-bodyとの連携により、町全体でコンディショニングが浸透しています。

*   **施設利用の実績**:
    *   B&G海洋センターのトレーニング室利用者数が**北海道内で1位**、全国24位を記録。
    *   利用者数はR-body導入1年目で2倍、2年目で**3.5倍**に増加。
    *   セッション利用者の肩こり・膝痛等の改善率は**73%**、目標達成率は**84%**。
*   **教育への導入**: 町内の全3つの小学校（第1、第2、第3小学校）の義務教育課程に「コンディショニング授業」を導入。
*   **組織体制**: 役場内に日本初と思われる「**ライフパフォーマンス室**」を設置。
*   **地域展開**: 東川町のノウハウを近隣の幌加内町など他の自治体へも広める活動を開始。

### 5. スポーツ庁による最新の研究と知見
室伏広治長官より、健康増進と経済を結びつける新たな知見が示されました。

*   **身体機能の曲がり角**: 800名のデータ分析の結果、運動機能が急激に低下するポイントは **49.1歳** であることが判明。
*   **怪我の予測**: 箱根駅伝ランナーの調査では、スクリーニングスコアが**46点以下**の選手は、3週間以上の離脱を伴う怪我をする確率が高い。
*   **労働生産性とGDP**: 日本のGDPがドイツに抜かれた現状を受け、50代以降の元気な高齢者を増やすことが労働生産性の維持に不可欠である。
*   **内臓と運動機能の関係**: 内臓のコントロールが良い人は、外側の運動器の機能も良いという画期的な研究成果が近く発表される予定。

### 6. 今後の展望：町民が主役のまちづくり
東川町モデルの次なるステージは、町民が「教わる側」から「広める側」に回ることです。

*   **「町民トレーナー」構想**: プログラムで健康になった町民が、今度は隣の人や身近な人にコンディショニングを教える「コーチ」の役割を担うことで、健康コミュニティを強化する。
*   **日常のスポーツ化**: 雪かきなどの日常動作も、意識の持ち方一つでライフパフォーマンス向上のためのトレーニング（運動欲求の充足）へと変えていく。
*   **生涯アスリート**: 「二本足で歩き続ける」ために、3ヶ月で終わるのではなく、食事と同じように一生運動（コンディショニング）を続けるリテラシーを定着させる。

# 結論

この講演会の核は、単なる「運動しましょう」ではなく、次の4点。

1. **トップアスリートに使ってきたコンディショニングを、一般住民の健康づくりへ移すこと**
2. **3か月の介入で、腰痛・主観的仕事パフォーマンス・身体機能が有意に改善したこと**
3. **運動は“量”だけでなく、“目的”と“質”が重要であること**
4. **健康の先にある「ライフパフォーマンス向上」を、町ぐるみで文化として定着させること**

加えて、一般向けに強く効く伝え方としては、次の2軸が有効。

* **契機づけ（今すぐ始める理由）**
* **損失回避（やらないことで失うもの）**

つまり、この講演会は
**「痛くなってからでは遅い。動ける今のうちに、体を整える習慣を生活の中に埋め込む」**
というメッセージに整理できる内容。

* 雪かき、歩行、振り向き、立ち上がりもトレーニングに変えられる
* 上手に動くことは、しんどさの節約になる
* 1回頑張るより、やめない仕組みの方が強い

## 6-3. 周囲を巻き込む言い方

* 自分がやると、家族も変わる
* 教える側になると、自分も続く
* 健康づくりは個人競技ではなく、地域の団体戦

# 7. この講演会で使える「損失回避」メッセージ

一般向けには、利益訴求だけでなく「失うもの」を明確にした方が動機づけになりやすい。

## 7-1. 放置の損失

* 腰痛や肩こりを放置すると、動き方の崩れが固定化しやすい
* 動かしにくさを放置すると、できる動作が年単位で減りやすい
* 体を使わない期間が長いほど、戻るのに時間がかかる
* 小さな不調の見逃しが、大きな不自由の入口になりうる

## 7-2. 年齢の損失

* 50歳前後で機能低下が加速しやすい
* “まだ大丈夫”の先延ばしが、一番大きな取りこぼしになる
* 高齢になってからの改善は可能だが、維持コストは高くなりやすい

## 7-3. 生活の損失

* 後ろを振り向けない
* 子どもを抱えるのがつらい
* 雪かきで痛める
* 長く働けない
* 趣味をやめる
* 外出が減る

これらは全部、放置の延長線上にある可能性。

## 7-4. 仕事・社会面の損失

* 仕事の集中力低下
* だるさによるパフォーマンス低下
* 早期退職リスク
* 医療費・介護費の増大
* 地域全体の活力低下

### 一言化

**体の機能低下は、本人だけの問題で終わらない**
