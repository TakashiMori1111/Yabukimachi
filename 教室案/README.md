## 🎯 教室のコンセプト

### 目的
体の「歪み」や「不調」を根本から改善し、機能的でしなやかな体をつくる

### 特徴
1. **呼吸から始める**: IAPブレス（腹腔内圧呼吸法）で体幹を安定化
2. **股関節重視**: 動きの要となる股関節の機能を最大化
3. **全身の連動性**: 部分ではなく、体全体をしなやかに使う
4. **科学的根拠**: 運動解剖学に基づいた理論的アプローチ
5. **個別対応**: 一人ひとりの体の癖や課題に対応

### 差別化ポイント
- ヨガ・ピラティスとは異なる「機能改善」特化型
- セルフケア方法まで習得可能
- 少人数制で質の高い個別指導

## 💡 カリキュラムの理論的背景

### 基づく資料
当教室のカリキュラムは、以下の専門的なトレーニング理論に基づいています：

1. **IAPブレス（腹腔内圧呼吸法）**
   - 横隔膜・腹横筋・骨盤底筋・多裂筋の連動
   - 360°の腹圧で体幹を安定化

2. **コアトレーニング**
   - ドローイン（収縮）とブレーシング（膨張）
   - 固定ではなく「安定」を目指す
   - 寝返り動作での全身連動

3. **股関節機能改善**
   - インナーマッスル（中殿筋・深層外旋六筋・内転筋群）の活性化
   - 多方向のモビリティワーク
   - 股関節主導の動きの習得

4. **運動連鎖**
   - 体の中心（コア・股関節）から末端へのパワー伝達
   - 背骨の柔軟性と体幹の安定性の統合
   - 時間差（ねじれ）を生む滑らかな動き
