### パターン①｜カリキュラム4回コース（独自体操・若年〜中年層向け）

#### 1. 基本情報

| 項目 | 内容 |
|------|------|
| **目的** | 仕事・家事・子育てで疲れがちな世代に向けて、**呼吸・コア・股関節・全身連動**を整え、腰痛・肩こり・慢性疲労の予防につなげる。 |
| **対象** | 30〜50代の子育て世代・働き世代（運動初心者歓迎） |
| **期間・回数** | 週1回×全4回（各60分） |
| **会場** | 公民館・ヘルスステーション等の多目的室（ヨガマットが敷ける広さ） |
| **定員** | 10〜15名程度 |
| **特徴** | 1回ごとにテーマを変えながらも、**身体の「土台作り」**に特化した独自体操メソッド。自宅でも継続しやすい内容。 |

#### 2. コース全体像

- **第1回**：呼吸とコアの目覚め 〜体の土台を知る〜  
  - 正しい呼吸法（ドローイン／ブレーシング）と、仰向けでの足上げ・膝つきプランクなどで**体幹の感覚づくり**。  
- **第2回**：体幹の安定性を高める 〜ブレない体づくり〜  
  - プランクバリエーション、四つん這いバードドッグ、寝返りトレーニング入門で**「固定でなく安定」の体幹**を学ぶ。  
- **第3回**：股関節を整える 〜動きの要を目覚めさせる〜  
  - クラムシェル、内転筋トレーニング、股関節モビリティワークで**股関節主導の動き**を身につける。  
- **第4回**：全身の連動性を高める 〜しなやかな体へ〜  
  - 寝返り応用、90/90、立ち上がり動作などで、**日常動作やスポーツに活かせる全身連動**を統合。  

#### 3. 成果・評価イメージ

- **主観スコア（0〜10）**  
  - 立ちやすさ／座りやすさ  
  - 肩こり・腰の重さ  
  - 体の軽さ・疲れにくさ  
- **行動変化**  
  - 「週◯回ストレッチ・体操をするようになった」  
  - 「姿勢や呼吸を意識する場面が増えた」 など  

→ 若年〜中年層向けの**“身体メンテナンス系”入門コース**として、  
　単独事業でも、年間コースへの導入プログラムとしても提案可能。

#### 4. アンケート項目（案）

| 質問 | 回答方式 |
|------|----------|
| ① この4回コースで、**体の軽さ・動きやすさ**の変化を実感しましたか？ | 5段階（全くそう思わない〜非常にそう思う） |
| ② 日常生活で、**姿勢や呼吸を意識する場面**が増えましたか？ | 5段階 |
| ③ この体操は、**他の一般的なストレッチ・体操と比べて「質が違う」**と感じましたか？ | 5段階 |
| ④ 講師の説明は、**専門的だが分かりやすい**と感じましたか？ | 5段階 |
| ⑤ このコースを、**家族や同世代の友人にすすめたい**と思いますか？ | 5段階 |
| ⑥ 今後も、**同様のコースがあれば継続して受講したい**と思いますか？ | 5段階 |

| 項目 | 内容 |

| **対象** | 65〜80代の高齢者（フレイル・プレフレイル層） |

| **構成** | 前半30分：下半身リセット体操（足首・膝・股関節・重心制御）／後半55〜60分：リングフィット等のeスポーツ運動 |

#### 2. 各回テーマ案（例）

- **第1回**：オリエンテーション＆事前測定  
  - 事業説明、体調確認、TUG・椅子立ち上がり・主観スコア測定

- **第2回**：足首と重心を整える  
  - 足部・足首リリース、足指ストレッチ、アキレス腱ストレッチ

- **第3回**：膝の安定と安心して立てる感覚  
  - ハムストリング・内転筋ストレッチ、膝まわりの安定化  
  - 椅子立ち上がりに近いフィットスキルを中心に実施  
- **第4回**：股関節を目覚めさせる  
  - 股関節モビリティワーク、臀部まわりのリリース

- **第5回**：全身連動と歩行スムーズ化  
  - 寝返り動作、小さな方向転換動作などを取り入れ、歩行に近い負荷へ  
  - アドベンチャーモード短編で「歩く・踏み込む・回る」を経験  
- **第6回**：個別課題のフォロー＆チャレンジ  
  - 足が重い／膝が不安／すぐ疲れる…など、参加者の声に合わせた調整回  
  - 負荷を上げたい人向け・控えめにしたい人向けメニューを用意  
- **第7回**：総合実践＆“できること”の再確認  
  - 前半体操を通して全身を整えたうえで、

- **第8回**：振り返り＆事後測定・今後の提案  
  - 初回と同じ指標で再測定（TUG／椅子立ち上がり／主観スコア 等）  
  - アンケート記入（独自体操評価・eスポーツ評価・継続意向）
