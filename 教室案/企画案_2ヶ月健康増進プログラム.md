# 下半身強化×リングフィット健康増進プログラム
## 2ヶ月集中コース 企画案

---

## 📋 プログラム概要

### 基本情報
- **期間**: 2ヶ月間（全4回）
- **頻度**: 月2回
- **所要時間**: 1回90分
- **対象**: 50〜70代の高齢者（運動初心者可）
- **定員**: 10名程度
- **会場**: Switch投影設備を備えた多目的スペース

### プログラムの狙い
本プログラムは、**下半身の安定性・可動域向上**と**楽しみながら続けられる運動習慣の獲得**を目的とした、科学的根拠に基づく健康増進プログラムです。

**前半30分**で身体の土台となる下半身を丁寧に整え、**後半55分**でリングフィットアドベンチャーを活用した実践的な全身運動を行うことで、短期間（2ヶ月）でも測定可能な健康指標の改善を目指します。

## 🏃 前半パート：下半身特化プログラム（30分）

### プログラムの目的
- 転倒予防に直結する足首・膝・股関節の安定性向上
- 日常動作（歩行・立ち座り）の質的改善
- 毎回同じ内容を繰り返すことで、身体の変化を実感しやすくする

### 実施内容（習熟重視型：全4回とも同一内容）

#### 【Phase 1】足部・足首の基礎調整（10分）

##### 1. 足首周辺の筋膜リリース（各部位15秒×両足）
- ①親指と人差し指の間
- ②内くるぶしの下（内反調整）
- ③アキレス腱周り
- ④足裏全体
- ⑤前脛骨筋（すねの前側）
- ⑥ふくらはぎ内側（ヒラメ筋）
- ⑦ふくらはぎ全体

**指導ポイント**:
- 硬いフローリングに足をつけて実施
- 「痛気持ちいい」程度の強さで
- 呼吸を止めない

##### 2. 足指のストレッチ＆可動性向上（3分）
- 各指を引っ張りながら横回転
- 小指・真ん中3本・親指の順に丁寧に
- 足指を床にしっかりつける感覚の確認

**注意点**:
- 爪が外側を向いている人→小指を外旋させる意識
- 膝の位置で痛みを調整
- 上半身はしっかり立てておく

##### 3. アキレス腱ストレッチ（2分）
- ゆっくり母指球重心で体重移動
- 少し胸を張り、呼吸を対象箇所に入れ込む
- かかとを床から離さない

#### 【Phase 2】膝関節・股関節の可動域拡大（15分）

##### 4. ハムストリングストレッチ（4分）
```
姿勢：
- 腰幅、つま先真っ直ぐ、膝を軽く緩める
- 鼠蹊部を押して、股関節から屈曲
- かかと重心、胸張る、顎引く

ポイント：
- 外側に重心が寄らないよう、母指球で踏む
- 背中から真っ直ぐのラインを保ったまま呼吸
- お尻を後ろに引くイメージ（股関節屈曲）
- 起始部（お尻）から伸びているイメージ
```

**注意事項**:
- 硬い人はヨガブロック使用（腰痛予防）
- 首の後ろはリラックス
- 膝が内側に入らないよう注意

##### 5. 内転筋（内もも）ストレッチ（4分）
```
動的準備：
- 先に脚を動かす（円を描くように、付け根から）

ストレッチ：
- 肘を内側にして、股関関節から屈曲
- 骨盤が丸まらないよう、胸を張り、腰を反る
- 終了後、内側母指球側に重心が移っている感覚を確認

バリエーション：
- カエルストレッチ
- 椅子を使った内旋・外旋運動
- 座位で対角に捻る（片手後ろ、片手は膝）
```

##### 6. 臀部周りの筋膜リリース＆ストレッチ（5分）
```
筋膜リリース（ボールまたはローラー使用）：
- A地点: 殿溝外側の"えくぼ"部（大殿筋中央〜坐骨付近）
- B地点: 腸骨稜やや後方の上外側（中殿筋）
- 前後左右に揺らす・動かす

ストレッチ：
- 胸を張ったまま、お腹から近づいていく
- 右手で押しながら、右に捻る（対角線の意識）
```

##### 7. 腸腰筋の動的可動性確保（2分）
- 座位または立位で実施可
- 脚を上げる際は「縮めながらおろす」（ストンと落とさない）
- 膝が外側に出過ぎないよう、斜め後ろに持っていくイメージ

#### 【Phase 3】全体統合＆確認（5分）

##### 8. 立位での重心確認＆歩行確認
- 内側母指球への重心移動を確認
- その場での足踏み→重心の変化を実感
- 数歩歩いて、足の軽さ・安定感を体感

##### 9. クールダウン＆次パートへの準備
- 深呼吸3回
- 水分補給の声かけ

### 前半パートで期待される効果
- **即効性**: 足の軽さ、可動域の広がりをその場で実感
- **転倒予防**: 足首・膝の安定性向上
- **姿勢改善**: 重心位置の適正化
- **痛み軽減**: 膝痛・腰痛の軽減

## 🎮 後半パート：リングフィットアドベンチャー（55分）

### 実施形態
- **投影方式**: 大型スクリーンにSwitchを投影
- **参加形式**: 全員で同じ動きを実施（集団エクササイズ型）
- **指導体制**: インストラクター1名がゲーム進行＋動作サポート

### プログラム構成

#### 【Week 1-2】初回〜2回目：基礎習得期

| 時間 | 内容 | 詳細 |
|------|------|------|
| 0:00-0:10 | **導入＆リングコン体験** | ・リングコンの押し込み・引っ張り体験<br>・レッグバンドの装着確認<br>・基本姿勢のレクチャー |
| 0:10-0:35 | **カスタムモード：下半身強化セット** | ・スクワット系（ワイドスクワット等）<br>・ランニング動作<br>・バランス系フィットスキル<br>※ゲーム内「カスタム」で下半身メニュー選択 |
| 0:35-0:50 | **ミニゲームモード** | ・「ろくろ回し」（体幹回旋）<br>・「モグラたたき」（素早い動作）<br>・参加者同士で記録を共有して盛り上がる |
| 0:50-0:55 | **クールダウン＆振り返り** | ・ゲーム内のヨガ・ストレッチモード<br>・感想シェア<br>・次回予告 |

**指導ポイント（初回〜2回目）**:
- 無理をしない声かけ（「自分のペースでOK」）
- 正しいフォームの確認（膝が内側に入らない、腰を反らない等）
- 楽しさ優先（記録よりも動きの質）

#### 【Week 3-4】3回目〜4回目：習熟＆挑戦期

| 時間 | 内容 | 詳細 |
|------|------|------|
| 0:00-0:05 | **ウォームアップ** | 軽いランニング・リングコン体操 |
| 0:05-0:30 | **アドベンチャーモード（短編）** | ・ステージクリア型で達成感を味わう<br>・道中の敵との「フィットバトル」で多様な動きを体験<br>・前半パートで鍛えた下半身を実践 |
| 0:30-0:45 | **カスタムセット：全身バランス型** | ・下半身＋上半身＋体幹のバランスメニュー<br>・TUG・椅子立ち上がりに近い動作を含む |
| 0:45-0:50 | **ミニゲームチャレンジ** | ・初回記録との比較<br>・「成長の見える化」 |
| 0:50-0:55 | **クールダウン＆まとめ** | ・静的ストレッチ<br>・2ヶ月の振り返り（最終回のみ） |

**指導ポイント（3〜4回目）**:
- 初回との違いを言語化（「足が軽くなった」「しゃがみやすい」等）
- 少しずつ負荷を上げる提案（回数・時間）
- 自宅での継続を促す声かけ

### リングフィット後半パートで期待される効果
- **筋力向上**: 特に大腿四頭筋・臀筋群（椅子立ち上がりテストに直結）
- **心肺機能**: 有酸素運動による持久力向上
- **協調性・バランス**: TUG（歩行＋方向転換）能力の向上
- **精神面**: ゲーム性による楽しさ→主観的幸福度・QOL向上

## 📊 測定する健康指標

### 測定タイミング
- **初回（1回目開始前）**: ベースライン測定
- **最終回（4回目終了後）**: 効果測定

### 選定指標と測定方法

#### 【優先度A】必須指標（即効性＋低負担）

##### 1. TUG（Timed Up & Go）テスト ★★★★★
```
測定方法：
1. 椅子に座った状態からスタート
2. 立ち上がり、3m先のコーンまで歩く
3. コーンを回って椅子に戻り、座るまでの時間を計測

所要時間：1人あたり約1分

期待される変化：
- 2ヶ月で平均0.5〜1.5秒の短縮
- 特に高齢者で改善が出やすい
```

**本プログラムとの関連性**:
- 前半：股関節・足首の可動域向上→方向転換のスムーズさ
- 後半：リングフィットの歩行動作→歩行速度・バランス改善

##### 2. 椅子立ち上がりテスト（5回） ★★★★★
```
測定方法：
1. 椅子に座り、腕を胸の前で組む
2. 立ち上がって座る動作を5回繰り返す
3. かかった時間を計測

所要時間：1人あたり約1分

期待される変化：
- 2ヶ月で平均1〜3秒の短縮
- 大腿四頭筋の筋力向上を直接反映
```

**本プログラムとの関連性**:
- 前半：ハムストリング・臀筋のストレッチ→立ち上がり動作の改善
- 後半：リングフィットのスクワット系運動→脚力の直接的強化

##### 3. 主観的幸福度（1〜10点） ★★★★★
```
測定方法：
「最近、どのくらい幸せだと感じますか？」
1点（全く幸せでない）〜10点（非常に幸せ）で回答

所要時間：1人あたり約30秒

期待される変化：
- 平均1〜2点の向上
- 世代交流・ゲーム要素で大幅改善の可能性
```

**本プログラムとの関連性**:
- 集団参加による社会的交流
- リングフィットの楽しさ→運動への肯定的イメージ
- 身体変化の実感→自己効力感の向上

#### 【優先度B】推奨指標（補完的測定）

##### 4. 4m歩行速度 ★★★★☆
```
測定方法：
4m区間を普通の速さで歩き、時間を計測

所要時間：1人あたり約1分

期待される変化：
- 0.1〜0.2m/秒の向上
- 「健康寿命の指標」として報告価値が高い
```

##### 5. SPPB（Short Physical Performance Battery）★★★★☆
```
測定内容：
- バランステスト（閉脚・セミタンデム・タンデム立位）
- 4m歩行
- 椅子立ち上がり5回
→合計0〜12点で評価

所要時間：1人あたり約10分

期待される変化：
- 平均1〜2点の向上
- 世界標準の指標として説得力◎
```

**推奨の組み合わせ**:
- **予算・時間に余裕がある場合**: 5指標すべて測定
- **最小限で実施する場合**: TUG + 椅子立ち上がり + 主観的幸福度の3つ

## ⚠️ 実施上の注意点

### 安全管理
- **事前確認**: 参加者の既往歴・現在の痛みを問診票で把握
- **禁忌事項**: 
  - 膝・腰に強い痛みがある場合→医師の許可確認
  - 手術後3ヶ月未満→参加を控える
- **当日の体調確認**: 毎回開始前に声かけ
- **中断の判断**: 痛み・めまい・息切れがあれば即座に休憩

### 前半パート（下半身特化）
- **個人差への対応**: 
  - 硬い人→ヨガブロック、クッション等を活用
  - 痛みがある人→範囲を狭める、片足ずつ実施
- **フォーム確認**: 
  - 膝が内側に入らない
  - 腰を過度に反らない
  - 呼吸を止めない

### 後半パート（リングフィット）
- **負荷調整**: 
  - ゲーム内設定で負荷を個別調整
  - 「自分のペースでOK」を繰り返し伝える
- **転倒予防**: 
  - 十分なスペース確保
  - 滑りにくいマット敷設
  - バランス系種目は椅子や壁の近くで実施
- **水分補給**: こまめな声かけ

### 測定時の注意
- **プライバシー配慮**: 数値は個人情報として慎重に扱う
- **測定環境**: 毎回同じ条件で実施（床の材質、測定者等）
- **モチベーション**: 結果の良し悪しでなく「変化」に注目する声かけ
