# 運動教室カリキュラム：年間コース（月2回）

**コンセプト**: 体の歪みを根本から改善し、しなやかで機能的な体をつくる  
**対象**: 継続的に体を整えたい方、パフォーマンス向上を目指す方  
**時間**: 各回60分  
**頻度**: 月2回（第1・第3週など）×12ヶ月 = 年間24回  
**必要なもの**: ヨガマット、フォームローラー（推奨）、ボール・クッション

---

## 年間カリキュラムの全体構成

### 第1期（1-3ヶ月 / 第1-6回）：基礎構築期
**テーマ**: 呼吸とコアの徹底習得  
体の土台となる呼吸法とコアの安定性を確立します。

### 第2期（4-6ヶ月 / 第7-12回）：機能改善期
**テーマ**: 股関節の機能を最大化  
股関節の可動域拡大とインナーマッスルの強化を行います。

### 第3期（7-9ヶ月 / 第13-18回）：統合発展期
**テーマ**: 全身の連動性を高める  
コアと股関節を統合し、実践的な動きにつなげます。

### 第4期（10-12ヶ月 / 第19-24回）：応用実践期
**テーマ**: 個別課題への対応と応用  
各自の目標に応じた高度なトレーニングを実施します。
